## Version16Dot16.MajorNumber property

Gets or sets major version number. Value has sense only in hexademical notation, for example version 0.5 for 'maxp' in actual font files is 4 bytes: {0, 0, 80, 0}, what has hexademical representation 0x00005000. After reading this version from font file, Major and minor numbers for object [`Version16Dot16`](../../version16dot16) will be 0 and 20480 respectively. And these values in hexademical form are 0x0000 and 0x5000.
