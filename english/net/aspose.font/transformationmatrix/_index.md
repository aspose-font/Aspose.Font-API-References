## TransformationMatrix class

Represents 3x3 transformation matrix &#x7C; A B 0 &#x7C; &#x7C; C D 0 &#x7C; &#x7C; TX TY 1 &#x7C;. Transforms coordinates in the following way: x1 = A*x + C*y + TX; y1 = B*x + D*y + TY.
