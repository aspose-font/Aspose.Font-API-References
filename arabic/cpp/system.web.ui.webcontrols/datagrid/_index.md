## DataGrid class


فئة تجريبية تجعل ربط الشيفرة المترجمة بمراجع [DataGrid](./) ممكنًا، لكنها غير قابلة للتنفيذ. لا تحتوي على أي أعضاء مُنفذة بشكل صحيح. يجب إنشاء كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم أبدًا بإنشاء نسخة من هذا النوع على المكدس أو باستخدام عامل new، لأن ذلك سيتسبب في أخطاء وقت التشغيل أو أخطاء التأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class DataGrid : public System::Object
```
