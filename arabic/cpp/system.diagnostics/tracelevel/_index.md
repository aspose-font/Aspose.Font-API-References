## TraceLevel enum


يحدد الرسائل التي يجب إخراجها لفئات [System.Diagnostics.Debug](../debug/)، [System.Diagnostics.Trace](../trace/) و System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### القيم

| الاسم | القيمة | الوصف |
| --- | --- | --- |
| Off | 0 | لا تُخرج أي رسائل تتبع أو تصحيح. |
| Error | 1 | يُخرج رسائل معالجة الأخطاء. |
| Warning | 2 | يُخرج رسائل التحذير ومعالجة الأخطاء. |
| Info | 3 | يُخرج رسائل معلوماتية، وتحذيرات، ومعالجة الأخطاء. |
| Verbose | 4 | يُخرج جميع رسائل التصحيح والتتبع. |
