## KeyInfoName class


يمثل العنصر الفرعي 'KeyName' في XMLDSIG أو عنصر '[KeyInfo](../keyinfo/)' في تشفير XML. يجب تخصيص كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم بإنشاء نسخة من هذا النوع على المكدس أو باستخدام عامل new، حيث سيؤدي ذلك إلى أخطاء وقت التشغيل و/أو أعطال في التأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class KeyInfoName : public System::Security::Cryptography::Xml::KeyInfoClause
```
