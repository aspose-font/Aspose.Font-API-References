## HttpContext class


فئة تجريبية تجعل من الممكن ربط الشيفرة المترجمة بمراجع [HttpContext](./)، ولكن دون تنفيذها. لا تحتوي على أي أعضاء مُنفذة بشكل صحيح. يجب إنشاء كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم أبدًا بإنشاء نسخة من هذا النوع على المكدس أو باستخدام المشغل new، لأن ذلك سيؤدي إلى أخطاء وقت التشغيل أو أخطاء تأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدم هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HttpContext : public System::Object
```
