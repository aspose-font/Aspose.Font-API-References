## Array::crend method


يعيد مُكرِّر عكسي إلى العنصر الذي يلي العنصر الأخير في الحاوية المعكوسة. وهو يطابق العنصر الذي يسبق العنصر الأول في الحاوية غير المعكوسة. هذا العنصر يعمل كعنصر نائب، ومحاولة الوصول إليه تؤدي إلى سلوك غير معرف.

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```


### ReturnValue

مكرّر يشير إلى العنصر النظري المؤهل بـ const والذي يسبق العنصر الأول في الحاوية.
