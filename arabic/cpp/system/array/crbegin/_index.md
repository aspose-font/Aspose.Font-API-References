## Array::crbegin method


يعيد مكرِّرًا عكسيًا إلى العنصر الأول في الحاوية المعكوسة. يتطابق مع العنصر الأخير في الحاوية غير المعكوسة. إذا كانت الحاوية فارغة، فإن المكرِّر المعاد يساوي [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

مكرِّر يشير إلى العنصر الأخير المؤهل بـ const في الحاوية.
