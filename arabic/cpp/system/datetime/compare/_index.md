## DateTime::Compare method

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| معامل | نوع | الوصف |
| --- | --- | --- |
| t1 | DateTime | المقارن الأول |
| t2 | DateTime | المقارن الثاني |

### ReturnValue

قيمة أقل من 0 إذا كان **t1** أسبق من **t2**؛ 0 إذا كان **t1** مساويًا لـ **t2**؛ قيمة أكبر من 0 إذا كان **t1** لاحقًا على **t2**.
