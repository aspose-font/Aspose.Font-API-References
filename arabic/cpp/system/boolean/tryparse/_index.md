## Boolean::TryParse method


يقوم بتحويل السلسلة المحددة إلى قيمة من نوع bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| معامل | نوع | الوصف |
| --- | --- | --- |
| قيمة | const String\& | السلسلة للتحويل. |
| result | bool\& | المرجع إلى متغيّر bool الذي تُوضع فيه نتيجة التحويل؛ تكون النتيجة true إذا كانت السلسلة المحددة مساوية لـ "True" وfalse إذا كانت السلسلة المحددة مساوية لـ "False". |

### ReturnValue

True إذا كانت السلسلة المحددة مساوية إما لـ "True" أو "False"، وإلا - false.
