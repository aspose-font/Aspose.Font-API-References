## EventArgs class


الفئة الأساسية للفئات التي تمثل سياقًا يتم تمريره إلى مشتركي الحدث عندما يتم تشغيل حدث. يجب إنشاء كائنات هذه الفئة باستخدام الدالة [System::MakeObject()](../makeobject/) فقط. لا تقم بإنشاء نسخة من هذا النوع على المكدس أو باستخدام عامل new، حيث سيؤدي ذلك إلى أخطاء وقت التشغيل و/أو أخطاء التأكيد. احwrap دائمًا هذه الفئة في مؤشر [System::SmartPtr](../smartptr/) واستخدم هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class EventArgs : public System::Object
```
