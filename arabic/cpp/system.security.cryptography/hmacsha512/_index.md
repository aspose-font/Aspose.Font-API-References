## HMACSHA512 class


رمز المصادقة الرسائل القائم على التجزئة [Authentication](../../system.security.authentication/) الذي يستخدم دالة التجزئة [SHA512](../sha512/). تم تنفيذ جزء منه فقط. يجب تخصيص كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/) function. لا تقم بإنشاء نسخة من هذا النوع على المكدس أو باستخدام المشغل new، لأن ذلك سيتسبب في أخطاء وقت التشغيل أو أعطال التأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HMACSHA512 : public System::Security::Cryptography::HashAlgorithm
```
