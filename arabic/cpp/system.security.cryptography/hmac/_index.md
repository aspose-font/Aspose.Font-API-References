## HMAC class


جميع تطبيقات الرسالة المستندة إلى التجزئة [Authentication](../../system.security.authentication/) Code ([HMAC](./)) يجب أن ترث هذه الفئة المجردة. يجب إنشاء كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم أبدًا بإنشاء نسخة من هذا النوع على المكدس أو باستخدام عامل new، لأن ذلك سيسبب أخطاء وقت التشغيل أو أخطاء تأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدم هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HMAC : public System::Security::Cryptography::HashAlgorithm
```
