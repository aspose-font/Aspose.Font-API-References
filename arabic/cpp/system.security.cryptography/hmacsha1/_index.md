## HMACSHA1 class


رمز توثيق الرسائل القائم على التجزئة [Authentication](../../system.security.authentication/) الذي يستخدم دالة التجزئة [SHA1](../sha1/). غير مُنفّذ. يجب تخصيص كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم بإنشاء نسخة من هذا النوع على المكدس أو باستخدام عامل new، لأن ذلك سيؤدي إلى أخطاء وقت التشغيل أو أخطاء التأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HMACSHA1 : public System::Security::Cryptography::HashAlgorithm
```
