توفر مساحة الاسم **Aspose.Font.Exceptions** الفئات التي تصف الاستثناءات المستخدمة في Aspose.Font.

## الفئات

| فئة | الوصف |
| --- | --- |
| [EncodingException](./encodingexception/) | يمثل استثناء الترميز. يمكن إلقاء الاستثناء في حالة حدوث مشكلة في ترميز/فك ترميز النص. |
| [FontAgrumentException](./fontagrumentexception/) | يمثل استثناء حجة الخط. يمكن إلقاء الاستثناء في حالة استخدام غير صحيح للمعاملات. |
| [FontConversionException](./fontconversionexception/) | يمثل استثناء تحويل الخط. يمكن إلقاء الاستثناء في حالة حدوث أخطاء أثناء عملية تحويل الخط. |
| [FontCreationException](./fontcreationexception/) | يمثل استثناء إنشاء الخط. يمكن إلقاء الاستثناء في حالة حدوث أخطاء أثناء عملية إنشاء الخط. |
| [FontException](./fontexception/) | يمثل استثناءً شائعًا مرتبطًا بمعالجة الخط. |
| [FontMergeException](./fontmergeexception/) | يمثل استثناء دمج الخط. قد يتم إلقاء الاستثناء في حالة حدوث أخطاء أثناء دمج الخطوط. |
| [FontNotSupportedOperationException](./fontnotsupportedoperationexception/) | يمثل استثناء العملية غير المدعومة. يمكن إلقاء الاستثناء في حالة عدم دعم بعض العمليات لنوع Font المحدد. |
| [IncorrectFontDataException](./incorrectfontdataexception/) | يمثل الاستثناءات للحالات التي تكون فيها بعض قيم كائن Font غير صالحة. |
| [LicenseRestrictionException](./licenserestrictionexception/) | يمثل الاستثناء الذي يمكن إلقاؤه عند محاولة تنفيذ وظيفة مقيدة في وضع التقييم. |
| [SvgConversionException](./svgconversionexception/) | يمثل استثناء تحويل Font لتنسيق SVG. يمكن إلقاء الاستثناء في حالة حدوث أخطاء أثناء عملية تحويل الخط. |
