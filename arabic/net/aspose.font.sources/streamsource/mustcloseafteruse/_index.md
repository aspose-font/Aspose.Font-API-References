## StreamSource.MustCloseAfterUse method

قد يمنع الورثون إغلاق التدفق. يرجع true إذا كان مصدر التدفق يريد إغلاق التدفق بعد الاستخدام. وإلا يرجع false.

```csharp
public virtual bool MustCloseAfterUse()
```

### قيمة الإرجاع

True إذا كان مصدر التدفق يرغب في إغلاق التدفق بعد الاستخدام، وإلا false.
