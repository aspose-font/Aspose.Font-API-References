توفر مساحة الاسم **Aspose.Font.CffDataProviders** الفئات المستخدمة لاسترجاع وتحديث البيانات من هياكل CFF المختلفة.

## الفئات

| فئة | الوصف |
| --- | --- |
| [NameIndexDataProvider](./nameindexdataprovider/) | يعلن عن وظيفة للوصول إلى بنية CFF Name INDEX. |
| [PrivateDictDataProvider](./privatedictdataprovider/) | يعلن عن وظيفة لقراءة/تحديث بنية CFF Private DICT. |
| [StringIndexDataProvider](./stringindexdataprovider/) | يعلن عن وظيفة للوصول إلى بنية CFF String INDEX. |
| [TopDictDataProvider](./topdictdataprovider/) | يعلن عن وظيفة لقراءة/تحديث بنية CFF Top DICT. |
## الواجهات

| الواجهة | الوصف |
| --- | --- |
| [ICffIndexDataProvider](./icffindexdataprovider/) | واجهة أساسية للوصول إلى بنى INDEX لخطوط CFF. |
## التعداد

| التعداد | الوصف |
| --- | --- |
| [CffIndexProviderType](./cffindexprovidertype/) | يحدد بنى INDEX المدعومة من قبل عائلة واجهة مزود الفهرس. |
| [CffUpdateStringIndexStrategy](./cffupdatestringindexstrategy/) | يحدد كيفية إضافة السلاسل إلى تخزين فهرس سلاسل CFF. عندما نحاول إضافة سلسلة ما إلى فهرس سلاسل CFF، قد يحدث أن تكون هذه السلسلة موجودة بالفعل في الفهرس. باستخدام الخيار SearchForDuplicates يمكننا إجبار البحث عبر الفهرس لاكتشاف ما إذا كانت هذه السلسلة موجودة بالفعل أم لا. هذا النهج يوفر مساحة في بنية فهرس السلاسل، لكنه يتطلب وقتًا أكبر لإضافة السلسلة. |
