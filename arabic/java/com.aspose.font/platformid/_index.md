### ISO {#ISO}
```
public static final PlatformId ISO
```


Value2 مواصفة Apple: (محجوز؛ لا تستخدم) مواصفة MS: ترميز ISO. لاحظ أن معرف المنصة 2 (ISO) تم إهماله اعتبارًا من مواصفة OpenType Specification v1.3. كان المقصود منه تمثيل ISO/IEC 10646، على عكس Unicode؛ كلا المعيارين لهما تعيينات رموز أحرف متطابقة.
