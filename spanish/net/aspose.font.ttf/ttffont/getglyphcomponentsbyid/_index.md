## GetGlyphComponentsById(GlyphId, GlyphIdList) {#getglyphcomponentsbyid}

Obtiene un glifo por identificador de glifo pasado y completa la lista pasada de identificadores de glifo con componentes de este glifo. El ID de glifo es un número único para un glifo, que depende del tipo de fuente. El ID de glifo de fuente TTF puede ser una instancia de ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) clase o ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) ) class. El direccionamiento de glifos de nombre (cadena) es compatible con las fuentes TTF a través de la asignación de tablas de publicación. En el caso de la fuente CFF interna, las estructuras CFF se utilizan para direccionar los glifos por nombre.

```csharp
public virtual void GetGlyphComponentsById(GlyphId id, GlyphIdList componentsToPopulate)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| id | GlyphId | Identificación de glifo. |
| componentsToPopulate | GlyphIdList | Lista de identificadores de glifos para completar. |

### Observaciones

Se debe pasar la colección vacía componentsToPopulate que contendrá la lista de ID de componentes de glifo.
