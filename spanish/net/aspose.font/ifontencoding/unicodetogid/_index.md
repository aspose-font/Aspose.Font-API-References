## IFontEncoding.UnicodeToGid method

Descodifica un Unicode y devuelve el id de glifo. El id de glifo es un número único para un glifo, que depende del tipo de fuente. Por ejemplo: El id de Type1 es un nombre de glifo, instancia de ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. El id de TTF es un índice int, instancia de ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) clase.

public GlyphId UnicodeToGid(uint unicode)
