## CffEncoding.DecodeToGid method

Obtiene Gid para charCode pasado. Este método está diseñado para CFF CIDFonts, donde charCode debe ser un valor CID válido. Glyph id es un número único para un glifo, que depende del tipo de fuente. CFF Font glyph id puede ser una instancia de ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) clase o ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) clase.
