## Type1Encoding.GidToUnicode method

Decodifica Gid a Unicode. Glyph id es un número único para un glifo, que depende del tipo de fuente. Por ejemplo: Type1's id es un nombre de glifo, instancia de ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. El id de TTF es un índice int, instancia de ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) clase.
