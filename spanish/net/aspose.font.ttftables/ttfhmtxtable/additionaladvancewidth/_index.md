## TtfHmtxTable.AdditionalAdvanceWidth property

En la tabla hmtx podrían darse casos en los que el número total de glifos no sea igual a hhea.numberOfHMetrics. Para estos casos, la tabla hmtx contiene array adicional 'leftSideBearing' que corresponde a la propiedad[`LeftSidebearings`](../leftsidebearings) . Pero los glifos con índices desde hhea.numOfLongHorMetrics hasta maxp.numGlyphs también tienen anchos. Y estos anchos de acuerdo con la especificación para la tabla hmtx tienen tales valores: "Aquí se supone que el ancho de avance es el mismo que el ancho de avance para el último entrada anterior".

public ushort AdditionalAdvanceWidth { get; }
