## Version16Dot16.MinorNumber property

Obtiene o establece el número de versión menor El valor tiene sentido solo en notación hexadecimal, por ejemplo la versión 0.5 para 'maxp' en archivos de fuentes reales es de 4 bytes: {0, 0, 80, 0}, lo que tiene representación hexadecimal 0x00005000. Después leyendo esta versión del archivo de fuente, números mayores y menores para el objeto[`Version16Dot16`](../../version16dot16) serán 0 y 20480 respectivamente. Y estos valores en forma hexadecimal son 0x0000 y 0x5000.

public ushort MinorNumber { get; set; }
