## Version16Dot16.MajorNumber property

Obtiene o establece el número de versión principal. El valor solo tiene sentido en notación hexadecimal, por ejemplo, versión 0.5 para 'maxp' en archivos de fuentes reales es de 4 bytes: {0, 0, 80, 0}, lo que tiene representación hexadecimal 0x00005000. Después de leer esta versión del archivo de fuentes , Números mayores y menores para objeto[`Version16Dot16`](../../version16dot16) serán 0 y 20480 respectivamente. Y estos valores en forma hexadecimal son 0x0000 y 0x5000.

public ushort MajorNumber { get; set; }
