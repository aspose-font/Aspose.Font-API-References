## IGlyphRenderer.RenderIndependentGlyphPath method

Representa el glifo utilizando una ruta de glifo independiente. La familia de funciones RenderGlyph() cambia la ruta del glifo en el renderizado. Luego conduce a la necesidad de volver a cargar este glifo nuevamente. Esta función usa una copia de la ruta del glifo y no cambia la ruta del glifo original, por lo que el mismo glifo podría reutilizarse varias veces. Esta versión de la función está pensada para usar con caché de glifos.

```csharp
public void RenderIndependentGlyphPath(IFont font, GlyphId glyphId, Glyph glyph, 
    TransformationMatrix glyphPlacementMatrix)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| font | IFont | La fuente que contiene el glifo. |
| glyphId | GlyphId | Índice de glifos físicos dentro de la fuente. Tenga en cuenta que esto no es un Unicode. |
| glyph | Glyph | Glifo para renderizar. |
| glyphPlacementMatrix | TransformationMatrix | Matriz que se aplica a las coordenadas del glifo. |
