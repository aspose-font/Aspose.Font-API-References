## CookieCollection::get_IsOtherVersionSeen method


Devuelve un valor que indica si la colección contiene una cookie con una versión que no es igual a [Cookie::MaxSupportedVersion](../../cookie/maxsupportedversion/).

```cpp
bool System::Net::CookieCollection::get_IsOtherVersionSeen()
```


### ReturnValue

True cuando la colección contiene una cookie con una versión que no es igual a [Cookie::MaxSupportedVersion](../../cookie/maxsupportedversion/), de lo contrario false.
