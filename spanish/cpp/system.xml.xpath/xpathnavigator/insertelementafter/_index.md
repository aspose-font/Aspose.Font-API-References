## XPathNavigator::InsertElementAfter method


Crea un nuevo elemento hermano después del nodo actual usando el prefijo de espacio de nombres, el nombre local y el URI del espacio de nombres especificados, con el valor especificado.

```cpp
virtual void System::Xml::XPath::XPathNavigator::InsertElementAfter(String prefix, String localName, String namespaceURI, String value)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| prefijo | Cadena | El prefijo de espacio de nombres del nuevo elemento hijo (si lo hay). |
| localName | Cadena | El nombre local del nuevo elemento hijo (si lo hay). |
| namespaceURI | String | El URI del espacio de nombres del nuevo elemento hijo (si lo hay). [String::Empty](../../../system/string/empty/) y **nullptr** son equivalentes. |
| value | String | El valor del nuevo elemento hijo. Si se pasa [String::Empty](../../../system/string/empty/) o **nullptr**, se crea un elemento vacío. |
