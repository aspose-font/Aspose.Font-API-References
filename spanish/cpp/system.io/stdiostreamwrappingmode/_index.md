## STDIOStreamWrappingMode enum


Especifica el modo de operaciones de E/S que los contenedores realizarán sobre flujos similares a std::iostreams.

```cpp
enum class STDIOStreamWrappingMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Binario | 0 | Un modo que permite que las operaciones de entrada decodifiquen datos de flujo del tipo char_type a bytes, y codifiquen bytes en datos del tipo char_type para operaciones de salida. |
| Conversión | 1 | Un modo que permite que las operaciones de entrada conviertan datos de flujo del tipo char_type al tipo uint8_t y viceversa para operaciones de salida. |
