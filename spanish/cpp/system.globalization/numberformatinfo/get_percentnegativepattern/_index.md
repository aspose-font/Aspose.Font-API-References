## NumberFormatInfo::get_PercentNegativePattern method


Obtiene el patrón negativo de porcentaje.

```cpp
int System::Globalization::NumberFormatInfo::get_PercentNegativePattern() const
```


### ReturnValue

Índice del patrón en la matriz {"-n %", "-n%", "-%n", "%-n", "%n-", "n-%", "n%-", "-% n", "n %-", "% n-", "% -n", "n- %"}, donde "n" es número, "%" es el signo de porcentaje y "-" es el signo menos.
