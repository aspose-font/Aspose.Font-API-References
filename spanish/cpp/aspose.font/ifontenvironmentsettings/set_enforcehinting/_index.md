## IFontEnvironmentSettings::set_EnforceHinting method


Esta bandera controla la lógica del uso de sugerencias de bytecode. Si es verdadera, la fuente debe habilitar las sugerencias de bytecode; si es falsa, las sugerencias de bytecode se activarán/desactivarán según la lógica predeterminada. Por defecto, false.

```cpp
virtual void Aspose::Font::IFontEnvironmentSettings::set_EnforceHinting(bool value)=0
```
