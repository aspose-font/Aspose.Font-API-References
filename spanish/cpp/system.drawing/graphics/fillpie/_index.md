

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| pincel | const SharedPtr\<Brush\>\& | Un pincel para usar al rellenar el pie |
| x | float | La coordenada X de la esquina superior izquierda del rectángulo que define la elipse |
| y | float | La coordenada Y de la esquina superior izquierda del rectángulo que define la elipse |
| width | float | El ancho del rectángulo que define la elipse |
| height | float | El alto del rectángulo que define la elipse |
| startAngle | float | Ángulo en grados medido en sentido horario desde el eje X hasta el punto de inicio del pie |
| sweepAngle | float | Ángulo en grados medido en sentido horario desde el **startAngle** hasta el punto final del pie |
