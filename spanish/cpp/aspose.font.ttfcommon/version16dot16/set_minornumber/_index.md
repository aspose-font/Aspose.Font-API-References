## Version16Dot16::set_MinorNumber method


Establece el número de versión menor. El valor solo tiene sentido en notación hexadecimal, por ejemplo la versión 0.5 para 'maxp' en los archivos de fuentes reales son 4 bytes: {0, 0, 80, 0}, que tiene la representación hexadecimal 0x00005000. Después de leer esta versión del archivo de fuentes, los números mayor y menor para el objeto [Version16Dot16](../) serán 0 y 20480 respectivamente. Y estos valores en forma hexadecimal son 0x0000 y 0x5000.
