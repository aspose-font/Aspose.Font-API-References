## XmlSchemaEnumerationFacet class


Representa la faceta **enumeration** del XML [Schema](../) según lo especificado por el World Wide [Web](../../system.web/) Consortium (W3C). Esta clase especifica una lista de valores válidos para un elemento **simpleType**. La declaración está contenida dentro de una declaración **restriction**.

```cpp
class XmlSchemaEnumerationFacet : public System::Xml::Schema::XmlSchemaFacet
```
