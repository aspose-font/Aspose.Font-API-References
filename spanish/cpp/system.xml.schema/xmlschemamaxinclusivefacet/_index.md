## XmlSchemaMaxInclusiveFacet class


Representa el elemento **maxInclusive** de XML [Schema](../) según lo especificado por el Consorcio World Wide [Web](../../system.web/) (W3C). Esta clase puede usarse para especificar una restricción sobre el valor máximo de un elemento **simpleType**. El valor del elemento debe ser menor o igual al valor del elemento **maxInclusive**.

```cpp
class XmlSchemaMaxInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
