## XmlSchemaMaxExclusiveFacet class


Representa el elemento **maxExclusive** del XML [Schema](../) según lo especificado por el World Wide [Web](../../system.web/) Consortium (W3C). Esta clase puede usarse para especificar una restricción sobre el valor máximo de un elemento **simpleType**. El valor del elemento debe ser menor que el valor del elemento **maxExclusive**.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
