## XmlSchemaDerivationMethod enum


Proporciona diferentes métodos para prevenir la derivación.

```cpp
enum class XmlSchemaDerivationMethod
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Vacío | 0 | Sobrescribe el método de derivación predeterminado para permitir cualquier derivación. |
| Sustitución | 1 | Se refiere a derivaciones mediante **Substitution**. |
| Extensión | 2 | Se refiere a derivaciones mediante **Extension**. |
| Restricción | 4 | Se refiere a derivaciones mediante **Restriction**. |
| Lista | 8 | Se refiere a derivaciones mediante **List**. |
| Union | 16 | Se refiere a derivaciones mediante **Union**. |
| Todos | 255 | **#all**. Se refiere a todos los métodos de derivación. |
| Ninguno | 256 | Acepta el método de derivación predeterminado. |
