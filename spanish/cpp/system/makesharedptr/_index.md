## System::MakeSharedPtr(const X *) method


Convierte un puntero crudo a un puntero inteligente. Sobrecarga para punteros const. Útil, p. ej., cuando se usa la variable 'this' en métodos de C# traducidos como const.

```cpp
template<class X> SmartPtr<X> System::MakeSharedPtr(const X *p)
```


| Parámetro | Descripción |
| --- | --- |
| X | Tipo apuntado. |

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| p | const X * | Puntero crudo a objeto. |

### ReturnValue

Puntero inteligente compartido a objeto.
