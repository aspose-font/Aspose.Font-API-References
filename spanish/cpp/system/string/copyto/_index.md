## String::CopyTo method


Copia los caracteres de la cadena en los elementos existentes del arreglo. No se realiza redimensionamiento.

```cpp
void System::String::CopyTo(int sourceIndex, const ArrayPtr<char_t> &destination, int destinationIndex, int count) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| sourceIndex | int | Índice en la cadena desde donde comenzar a leer. |
| destination | const ArrayPtr\<char_t\>\& | Matriz de destino. |
| destinationIndex | int | Índice en el arreglo desde donde comenzar a escribir. |
| count | int | Número de caracteres a copiar. |
