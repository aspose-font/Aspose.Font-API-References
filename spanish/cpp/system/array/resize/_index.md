## Array::Resize method


Cambia el tamaño del array especificado al valor especificado o crea un nuevo array con el tamaño especificado.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) para redimensionar. Si **arr** es un puntero nulo, se creará la nueva matriz |
| new_size | int | El nuevo tamaño de la matriz, o el tamaño de la nueva matriz si **arr** es nulo |
