## SmartPtr::get_shared_count method


Obtiene el número de punteros compartidos existentes al objeto referenciado, incluido el actual. Afirma que el puntero actual está en modo compartido.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Número de punteros compartidos existentes al objeto referenciado, si los hay. Si el puntero es nulo, devuelve 0.
