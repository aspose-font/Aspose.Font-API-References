## DateTime::AddMonths method


Devuelve una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora que es la suma del valor representado por el objeto actual y el número especificado de meses.

```cpp
DateTime System::DateTime::AddMonths(int value) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | int | El número de meses a añadir; el valor puede ser positivo o negativo |

### ReturnValue

Una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora que es la suma del valor representado por el objeto actual y **value** meses
