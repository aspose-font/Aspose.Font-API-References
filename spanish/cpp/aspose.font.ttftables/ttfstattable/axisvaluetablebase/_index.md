## AxisValueTableBase class


Clase base para la estructura de la tabla de valores de eje. Especificación: Las tablas de valores de eje proporcionan detalles sobre un valor de atributo de estilo específico en algún eje de variación de diseño, o una combinación de valores de ejes de variación de diseño, y la relación de esos valores con las etiquetas usadas como elementos en los nombres de subfamilia.
