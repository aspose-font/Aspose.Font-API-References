## RangeGaspBehaviorFlags enum


Indicadores que describen el comportamiento deseado del rasterizador.

```cpp
enum class RangeGaspBehaviorFlags : uint16_t
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| ninguno | n/a | Opcional para tamaños muy grandes, típicamente ppem>2048. |
| GASP_GRIDFIT | n/a | Utiliza el ajuste de cuadrícula. |
| GASP_DOGRAY | n/a | Utiliza renderizado en escala de grises. |
| GASP_SYMMETRIC_GRIDFIT | n/a | Utiliza el ajuste de cuadrícula con suavizado simétrico ClearType. Solo compatible con la versión 1 'gasp'. |
| GASP_SYMMETRIC_SMOOTHING | n/a | Utiliza suavizado a lo largo de varios ejes con ClearType®. Solo compatible con la versión 1 'gasp'. |
| Reservado | n/a | Banderas reservadas — establecer a 0. |
