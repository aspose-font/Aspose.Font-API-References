## TtfPostTable::GetGlyphIndex method


Obtiene el índice del glifo por nombre de glifo.

```cpp
uint32_t Aspose::Font::TtfTables::TtfPostTable::GetGlyphIndex(System::String glyphName)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| glyphName | System::String | Nombre del glifo. |

### ReturnValue

Índice de glifo. Cuando no se proporciona información de nombres PostScript para los glifos en este archivo de fuente, devuelve el índice 0, que es el glifo indefinido o el glifo \".notdef\".
