## IGlyphAccessor::GetAllGlyphIds method


Devuelve todos los ids de glifos, disponibles en el [Font](../../../aspose.font/font/). El id de [Glyph](../../glyph/) es un número único para un glifo, que depende del tipo de fuente. Por ejemplo: el id de [Type1](../../../aspose.font.type1/) es un nombre de glifo, instancia de la clase ([GlyphStringId](../../glyphstringid/)). El id de TTF es un índice entero, instancia de la clase ([GlyphUInt32Id](../../glyphuint32id/)).

```cpp
virtual System::ArrayPtr<System::SharedPtr<GlyphId>> Aspose::Font::Glyphs::IGlyphAccessor::GetAllGlyphIds()=0
```


### ReturnValue

[Glyph](../../glyph/) identifiers.
