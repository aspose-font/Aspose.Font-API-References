## TraceLevel enum


Especifica qué mensajes se deben emitir para las clases [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) y System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Off | 0 | No emitir mensajes de trazado ni depuración. |
| Error | 1 | Emitir mensajes de manejo de errores. |
| Warning | 2 | Emitir mensajes de advertencias y manejo de errores. |
| Info | 3 | Emitir mensajes informativos, advertencias y de manejo de errores. |
| Verbose | 4 | Emitir todos los mensajes de depuración y trazado. |
