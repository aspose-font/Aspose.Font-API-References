## XmlNodeOrder enum


Describe el orden del documento de un nodo comparado con un segundo nodo.

```cpp
enum class XmlNodeOrder
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Antes | 0 | El nodo actual de este navegador está antes del nodo actual del navegador suministrado. |
| Después | 1 | El nodo actual de este navegador está después del nodo actual del navegador suministrado. |
| Mismo | 2 | Los dos navegadores están posicionados en el mismo nodo. |
| Desconocido | 3 | No se pueden determinar las posiciones de los nodos en el orden del documento, respecto a cada uno. Esto podría ocurrir si los dos nodos residen en árboles diferentes. |
