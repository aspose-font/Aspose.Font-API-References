## XmlNamespaceManager::RemoveNamespace method


Elimina el espacio de nombres dado para el prefijo dado.

```cpp
virtual void System::Xml::XmlNamespaceManager::RemoveNamespace(String prefix, String uri)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| prefijo | Cadena | El prefijo del espacio de nombres. |
| uri | Cadena | El espacio de nombres a eliminar para el prefijo dado. El espacio de nombres eliminado proviene del alcance actual del espacio de nombres. Los espacios de nombres fuera del alcance actual se ignoran. |
