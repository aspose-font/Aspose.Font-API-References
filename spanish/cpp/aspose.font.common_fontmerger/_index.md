| [FontCharactersMerger](./fontcharactersmerger/) | Declara la funcionalidad para combinar fuentes de diferentes tipos. Se necesita un par de fuentes para la operación de combinación. Esto significa que muchas características, relacionadas con la fuente combinada final, como métricas, estructura de codificación y otras, se tomarán de esta fuente primaria. La otra fuente del par (secundaria) proporciona solo glifos para la fuente combinada final. |
