## Type1Encoding.UnicodeToGid method

Returnerar GlyphId för unicode. Eller notdef om teckensnittet inte innehåller glyf för unicode. Glyf-id är ett unikt nummer för en glyf, vilket är teckensnittsberoende. Till exempel: Typ1s id är ett glyfnamn, instans av ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. TTF:s id är ett int-index, instans av ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) class.

```csharp
public GlyphId UnicodeToGid(uint unicode)
```

| unicode | UInt32 | Unicode för att få glyph-identifierare för. |

### Returvärde

Glyfidentifierare relaterad till unicode har godkänts.
