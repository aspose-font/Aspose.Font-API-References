## TtfEncoding.DecodeToGid method

TTF Fonts DecodeToGlyphId-implementering hittar unicode-tabellen och returnerar glyph id för unicode char. Glyph id är ett unikt nummer för en glyph, vilket är teckensnittsberoende. Till exempel: Type1:s namn är i instans av glyph id ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. TTF:s id är ett int-index, instans av ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) class.
