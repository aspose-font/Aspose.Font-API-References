## CffFont.GetAllGlyphIds method

Returnerar array av alla glyph id, tillgängliga i Font. Glyph id är ett unikt nummer för en glyph, vilket är teckensnittsberoende. CFF Font glyph id kan vara en instans av ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) klass eller ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) class.
