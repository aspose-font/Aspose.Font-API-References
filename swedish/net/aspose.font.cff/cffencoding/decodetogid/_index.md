## CffEncoding.DecodeToGid method

Får Gid för godkänd charCode. Den här metoden utformad för en CFF CIDFonts, där charCode måste vara ett giltigt CID-värde. Glyph id är ett unikt nummer för en glyph, vilket är teckensnittsberoende. CFF Font glyph id kan vara en instans av ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) klass eller ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) class.
