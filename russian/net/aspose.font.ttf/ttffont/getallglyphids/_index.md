## TtfFont.GetAllGlyphIds method

Возвращает массив всех идентификаторов глифов, доступных в Шрифте. Идентификатор глифа — это уникальный номер глифа, который зависит от типа шрифта. Идентификатор глифа шрифта TTF может быть экземпляром класса ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) или класса ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id)) . Адресация глифа имени (строки) поддерживается для шрифтов TTF через сопоставление таблицы Post. В случае, если CFF Font внутри, структуры CFF используются для адресации глифов по имени.

```csharp
public override GlyphId[] GetAllGlyphIds()
```

### Возвращаемое значение

Идентификаторы глифов.
