## Type1MetricFont class

Реализация метрического шрифта Type1. Этот шрифт type1 создан с использованием только метрик. Функции извлечения глифов и некоторые другие, требующие реального шрифта, не разрешены, не разрешенные функции вызывают исключениеType1NotSupportedException. Другие свойства (FontName, Weight, Metrics и Encoding) используются из файла метрик.

```csharp
public class Type1MetricFont : Type1Font
```

```csharp
string[] zapfDingbatsEncoding = new string[256] {null, null, ... , "space", "a1", ...};
FontEnvironment.Current.FontSpecificEncodings.RegisterEncoding("ZapfDingbats", zapfDingbatsEncoding);
```
