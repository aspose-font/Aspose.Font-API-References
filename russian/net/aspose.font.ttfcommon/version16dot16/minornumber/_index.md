## Version16Dot16.MinorNumber property

Получает или устанавливает младший номер версии Значение имеет смысл только в шестнадцатеричном представлении, например версия 0.5 для 'maxp' в реальных файлах шрифтов составляет 4 байта: {0, 0, 80, 0}, что имеет шестнадцатеричное представление 0x00005000. После чтение этой версии из файла шрифта, основные и второстепенные номера для объекта[`Version16Dot16`](../../version16dot16) будет 0 и 20480 соответственно. И эти значения в шестнадцатеричной форме 0x0000 и 0x5000.

public ushort MinorNumber { get; set; }
