**Aspose.Font.Sources**пространство имен обеспечивает функциональность для доступа к данным шрифта из различных источников (файлов, потоков памяти и т. д.).

## Классы

| Учебный класс | Описание |
| --- | --- |
| [ByteContentStreamSource](./bytecontentstreamsource) | Представляет источник потока на основе потока контента. |
| [FileSystemStreamSource](./filesystemstreamsource) | Представляет источник потока на основе файловой системы. |
| [FontDefinition](./fontdefinition) | Представляет определение набора файлов шрифтов. Этот класс содержит поля, которые не относятся к внутренним данным шрифта. Эти поля описывают размещение шрифта и другие данные, необходимые для загрузки шрифта из какого-либо источника шрифта (файла, памяти и т. д.). |
| [FontFileDefinition](./fontfiledefinition) | Представляет определение файла шрифта. |
| [StreamSource](./streamsource) | Определяет способ получения файлового потока, когда это необходимо. |
