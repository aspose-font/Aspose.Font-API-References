## StreamSource.MustCloseAfterUse method

Наследники могут предотвратить закрытие потока. Возвращает true, если источник потока хочет, чтобы поток был закрыт после использования. В противном случае возвращает false.

```csharp
public virtual bool MustCloseAfterUse()
```

### Возвращаемое значение

True, если источник потока хочет, чтобы поток был закрыт после использования, в противном случае false.
