## IFontEncoding.DecodeToGid method

Декодирует код символа и возвращает идентификатор глифа. Идентификатор глифа — это уникальный номер глифа, который зависит от типа шрифта.[`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. Идентификатор TTF является индексом int, экземпляром ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) ) class. Примечание: код символа не обязательно юникод. Код символа — это индекс символа в кодировке шрифта «таблица».

```csharp
public GlyphId DecodeToGid(uint charCode)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| charCode | UInt32 | Код символа, для которого нужно получить идентификатор глифа. |

### Возвращаемое значение

Идентификатор глифа, связанный с переданным charCode.
