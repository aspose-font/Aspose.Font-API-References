## IFontEncoding.UnicodeToGid method

Декодирует юникод и возвращает идентификатор глифа. Идентификатор глифа — это уникальный номер глифа, который зависит от типа шрифта. Например: Идентификатор Type1 — это имя глифа, экземпляр класса ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)). Идентификатор TTF является индексом int, экземпляром класса ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id)).

```csharp
public GlyphId UnicodeToGid(uint unicode)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| unicode | UInt32 | Юникод, для которого нужно получить идентификатор глифа. |

### Возвращаемое значение

Идентификатор глифа, связанный с переданным юникодом.
