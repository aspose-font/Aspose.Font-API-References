## IFontEncoding.UnicodeToGid method

Декодирует Unicode и возвращает идентификатор глифа. Идентификатор глифа — это уникальный номер глифа, который зависит от типа шрифта. Например: id Type1 — это имя глифа, экземпляр[`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. Идентификатор TTF является индексом int, экземпляром ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) класс.

```csharp
public GlyphId UnicodeToGid(uint unicode)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| unicode | UInt32 | Unicode для получения идентификатора глифа. |

### Возвращаемое значение

Идентификатор глифа, связанный с переданным юникодом.
