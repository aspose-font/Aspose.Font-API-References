## TransformationMatrix class

Представляет матрицу преобразования 3x3 &#x7C; АБ 0 &#x7C; &#x7C; КД 0 &#x7C; &#x7C; ТХ ТИ 1 &#x7C;. Преобразует координаты следующим образом: x1 = A*x + C*y + TX; y1 = B*x + D*y + TY.
