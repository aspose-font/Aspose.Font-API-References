## SocketFlags enum


Предоставляет константные значения для сообщений сокета.

```cpp
enum class SocketFlags
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Нет | 0 | Для этого вызова не используются флаги. |
| OutOfBand | 1 | Выполняется обработка out-of-band данных. |
| Peek | 2 | Посмотреть входящее сообщение. |
| DontRoute | 4 | Отправить сообщение без использования таблиц маршрутизации. |
| Обрезано | 256 | Сообщение слишком велико, чтобы поместиться в указанный буфер. Оно было обрезано. |
| ControlDataTruncated | 512 | Контрольные данные превышают 64 КБ и не помещаются во внутренний буфер. Они были обрезаны. |
| Broadcast | 1024 | Широковещательный пакет. |
| Мультикаст | 2048 | Мультикастовый пакет. |
| Частичный | 32768 | Сообщение, отправленное или полученное частично. |
