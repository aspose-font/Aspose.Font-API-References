## XmlSchemaComplexType::get_BlockResolved method


Возвращает значение после того, как тип был скомпилирован в набор информации после проверки схемы (infoset). Это значение указывает, как тип применяется, когда **xsi:type** используется в документе экземпляра.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

Значение пост-валидационного infoset. Значением по умолчанию является **BlockDefault** на элементе **schema**.
