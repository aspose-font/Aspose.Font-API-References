## Timer::set_AutoReset method


Устанавливает таймер в режим авто‑сброса или отключает его.



| Параметр | Тип | Описание |
| --- | --- | --- |
| автосброс | bool | true, чтобы таймер вызывался в цикле, false — чтобы остановить его после первого вызова. |
