## FileAttributes enum


Представляет атрибуты каталога или файла.

```cpp
enum class FileAttributes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Архив | 32 | Элемент файловой системы является кандидатом на удаление или резервное копирование. |
| Сжатый | 2048 | Элемент файловой системы сжат. |
| Устройство | 64 | Не используется. |
| Каталог | 16 | Элемент файловой системы является каталогом. |
| Зашифровано | 16384 | Элемент файловой системы зашифрован. |
| Скрытый | 2 | Элемент файловой системы скрыт. |
| IntegrityStream | 32768 | Элемент файловой системы содержит поддержку целостности данных. |
| Normal | 128 | Элемент файловой системы — файл без специальных атрибутов. |
| NotContentIndexed | 8192 | Элемент файловой системы не будет индексироваться службой индексирования содержимого. |
| NoScrubData | 131072 | Элемент файловой системы не включён в сканирование целостности данных. |
| Offline | 4096 | Элемент файловой системы находится в офлайн-режиме, и его данные недоступны сразу. |
| ReadOnly | 1 | Элемент файловой системы доступен только для чтения. |
| ReparsePoint | 1024 | Элемент файловой системы — файл, содержащий точку повторного разбора. |
| SparseFile | 512 | Элемент файловой системы — разрежённый файл. |
| System | 4 | Элемент файловой системы — системный файл. |
| Temporary | 256 | Элемент файловой системы — временный файл. |
| Invalid | 0 | Недопустимое значение. |
