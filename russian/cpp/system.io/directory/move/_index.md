## Directory::Move method


Перемещает указанную сущность в новое место. Если перемещаемая сущность является каталогом, она перемещается со всем её содержимым.

```cpp
static void System::IO::Directory::Move(const String &sourceDirName, const String &destDirName)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| sourceDirName | const String\& | Каталог или файл для перемещения |
| destDirName | const String\& | Новое расположение **sourceDirName** |
