## SystemIOStreamWrappingMode enum


Указывает режим операций ввода/вывода, которые обёртки будут выполнять над потоками, похожими на [System::IO::Stream](../stream/).

```cpp
enum class SystemIOStreamWrappingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Binary | 0 | Режим, позволяющий операциям ввода кодировать байты потока в данные типа char_type и декодировать данные типа char_type в байты потока для операций вывода. |
| Conversion | 1 | Режим, позволяющий операциям ввода преобразовывать байты потока из типа uint8_t в тип char_type и наоборот для операций вывода. |
