## FileMode enum


Указывает, как должен быть открыт файл.

```cpp
enum class FileMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| CreateNew | 1 | Создать новый файл. Если файл уже существует, будет выброшено исключение. |
| Create | 2 | Создайте новый файл. Если файл уже существует, он будет перезаписан. |
| Open | 3 | Откройте существующий файл. Если файл не существует, будет выброшено исключение. |
| OpenOrCreate | 4 | Откройте существующий файл или создайте новый, если он не существует. |
| Обрезка | 5 | Откройте существующий файл и усеките его, чтобы он стал пустым. Если файл не существует, будет выброшено исключение. |
| Добавить | 6 | Откройте существующий файл и переместите указатель в конец, или создайте новый, если он не существует. |
