## Color::get_IsNamedColor method


Возвращает значение, определяющее, представляет ли структура [Color](../) именованный цвет или элемент перечисления [KnownColor](../../knowncolor/).

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```


### ReturnValue

True, если [Color](../) был создан с помощью метода FromName(String) или метода [FromKnownColor(KnownColor)](../fromknowncolor/); в противном случае возвращает false.
