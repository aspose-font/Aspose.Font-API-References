## PointF::Subtract(const PointF\&, const Size\&) method


Вычитает значения ширины и высоты указанного объекта [Size](../../size/) из значений координат X и Y указанного объекта [PointF](../) соответственно.

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const Size &size)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| точка | const PointF\& | Точка для переноса |
| size | const Size\& | Объект [Size](../../size/), который задаёт значения для вычитания из значений координат **point** |

### ReturnValue

Новый объект [PointF](../), значение координаты X которого равно результату вычитания значения ширины **size** из значения координаты X **point**, а значение координаты Y равно результату вычитания значения высоты **size** из значения координаты Y **point**
