## Rectangle::Offset(const Point\&) method


Смещает положение прямоугольника, представленного текущим объектом, на указанные величины.

```cpp
void System::Drawing::Rectangle::Offset(const Point &pos)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| pos | const Point\& | Объект [Point](../../point/) указывающий величину, на которую смещается прямоугольник; значение координаты X точки определяет величину, которую следует добавить к координате X верхнего левого угла прямоугольника, а значение координаты Y точки определяет величину, которую следует добавить к координате Y верхнего левого угла прямоугольника. |
