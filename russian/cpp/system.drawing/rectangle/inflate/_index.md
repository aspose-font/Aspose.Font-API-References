## Rectangle::Inflate(const Size\&) method


Увеличивает ширину и высоту прямоугольника, представленного текущим объектом, сохраняя положение его геометрического центра. Ширина и высота увеличиваются в обеих направлениях на величины, указанные соответствующими значениями ширины и высоты указанного объекта размера.

```cpp
void System::Drawing::Rectangle::Inflate(const Size &size)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| size | const Size\& | Объект [Size](../../size/) указывающий величины, на которые следует увеличить ширину и высоту прямоугольника |
