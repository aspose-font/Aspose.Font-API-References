## UnicodePlatformSpecificId enum


Представляет платформенно-специфичное перечисление Unicode.

```cpp
enum class UnicodePlatformSpecificId
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Значение 0 Семантика по умолчанию (Unicode 1.0). |
| Unicode1_1 | 1 | Значение 1 Семантика Unicode 1.1. |
| ISO10646_1993 | 2 | Значение 2 Семантика ISO 10646 1993 (устарело). |
| Unicode2_0 | 3 | Значение 3 Семантика Unicode 2.0 и более поздних версий. Только BMP Unicode. |
| Unicode2_0_Full | 4 | Значение 4 Семантика Unicode 2.0 и последующих версий (разрешены символы за пределами BMP), полный набор Unicode. |
