## TtfPostTable::GetGlyphIndex method


Получает индекс глифа по имени глифа.

```cpp
uint32_t Aspose::Font::TtfTables::TtfPostTable::GetGlyphIndex(System::String glyphName)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| glyphName | System::String | Имя глифа. |

### ReturnValue

Индекс глифа. Когда информация о PostScript именах для глифов в этом файле шрифта не предоставлена, возвращает индекс 0, который соответствует неопределённому глифу или глифу ".notdef".
