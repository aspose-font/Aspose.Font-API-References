## RangeGaspBehaviorFlags enum


Флаги, описывающие желаемое поведение растеризатора.

```cpp
enum class RangeGaspBehaviorFlags : uint16_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ни один | n/a | Необязательно для очень больших размеров, обычно ppam>2048. |
| GASP_GRIDFIT | n/a | Использовать привязку к сетке. |
| GASP_DOGRAY | n/a | Использовать отображение в градациях серого. |
| GASP_SYMMETRIC_GRIDFIT | n/a | Использовать привязку к сетке с симметричным сглаживанием ClearType. Поддерживается только в версии 1 'gasp'. |
| GASP_SYMMETRIC_SMOOTHING | n/a | Использовать сглаживание по нескольким осям с ClearType®. Поддерживается только в версии 1 'gasp'. |
| Зарезервировано | n/a | Зарезервированные флаги — установить в 0. |
