## Boolean::TryParse method


Преобразует указанную строку в значение типа bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| значение | const String\& | Строка для преобразования. |
| result | bool\& | Ссылка на переменную типа bool, в которую помещается результат преобразования; результат будет true, если указанная строка равна "True", и false, если указанная строка равна "False". |

### ReturnValue

True, если указанная строка равна либо "True", либо "False", иначе - false.
