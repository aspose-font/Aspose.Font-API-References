## Char::IsLetter(char_t) method


Определяет, классифицируется ли указанный символ как буква Unicode.

```cpp
static bool System::Char::IsLetter(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True, если указанный символ является буквой Unicode, иначе — false
