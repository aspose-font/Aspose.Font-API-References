## Char::IsWhiteSpace(char_t) method


Определяет, классифицируется ли указанный символ как пробельный символ.

```cpp
static bool System::Char::IsWhiteSpace(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True, если указанный символ является пробельным символом, иначе — false
