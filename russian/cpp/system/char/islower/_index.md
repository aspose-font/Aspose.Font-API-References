## Char::IsLower(char_t) method


Определяет, классифицируется ли указанный символ как строчная буква.

```cpp
static bool System::Char::IsLower(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True, если указанный символ является строчной буквой, иначе — false
