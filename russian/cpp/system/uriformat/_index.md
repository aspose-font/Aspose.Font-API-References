## UriFormat enum


Указывает, как происходит экранирование URI.

```cpp
enum class UriFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| UriEscaped | 1 | Экранирование согласно RFC-2396. |
| Неэкранировано | 2 | Без экранирования. |
| SafeUnescaped | 3 | Только символы, имеющие зарезервированное значение в URI, экранируются. |
