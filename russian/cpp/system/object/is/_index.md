## Object::Is method


Проверяет, представляет ли объект экземпляр типа, описанного targetType. Аналог оператора C# 'is'.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| targetType | const TypeInfo\& | [TypeInfo](../../typeinfo/) структура, описывающая тип, против которого проверяется текущий объект. |

### ReturnValue

True, если объект имеет помеченный тип или его подкласс, иначе false.
