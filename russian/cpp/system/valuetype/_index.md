## ValueType class


Базовый класс для типовых значений с наследованием от [Object](../object/) наследованием, усечённым в целях производительности. Этот тип должен быть выделен в стеке и передаваться в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../smartptr/) для управления объектами этого типа.

```cpp
class ValueType
```
