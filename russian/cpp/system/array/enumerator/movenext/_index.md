## Enumerator::MoveNext method


Проверяет, указывает ли индекс текущего элемента не на последний элемент в массиве, и продвигает его, если это не так.

```cpp
virtual bool System::Array<T>::Enumerator::MoveNext() override
```


### ReturnValue

True если индекс текущего элемента не указывал на последний элемент массива до его перемещения, иначе — false
