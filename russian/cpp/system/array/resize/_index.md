## Array::Resize method


Изменяет размер указанного массива до заданного значения или создает новый массив с указанным размером.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) для изменения размера. Если **arr** является нулевым указателем, будет создан новый массив |
| new_size | int | Новый размер массива, или размер нового массива, если **arr** равен null |
