## Array::cbegin method


Возвращает итератор к первому const‑квалифицированному элементу контейнера. Если контейнер пуст, возвращаемый итератор будет равен [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### ReturnValue

Итератор, указывающий на первый const‑квалифицированный элемент контейнера.
