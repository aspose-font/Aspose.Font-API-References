## StringComparison enum


### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| CurrentCulture | 0 | Использовать текущую культуру. |
| CurrentCultureIgnoreCase | 1 | Использовать текущую культуру, но игнорировать регистр. |
| InvariantCulture | 2 | Использовать инвариантную культуру. |
| InvariantCultureIgnoreCase | 3 | Использовать инвариантную культуру, но игнорировать регистр. |
| Ordinal | 4 | Использовать порядковые правила сортировки. |
| OrdinalIgnoreCase | 5 | Использовать порядковые правила сортировки, но игнорировать регистр. |
