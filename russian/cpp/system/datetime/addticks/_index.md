## DateTime::AddTicks method

```cpp
DateTime System::DateTime::AddTicks(int64_t value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| значение | int64_t | Количество интервалов по 100 наносекунд для добавления; значение может быть положительным или отрицательным. |
