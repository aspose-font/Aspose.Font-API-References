## WeakReference class


Представляет слабую ссылку, которая ссылается на объект, позволяя при этом объекту быть удалённым.

```cpp
template<typename...>class WeakReference
```


| Параметр | Описание |
| --- | --- |
| Аргументы | Аргументы шаблона слабой ссылки, объявляющие тип ссылочного объекта; количество аргументов может быть ноль или один. |
