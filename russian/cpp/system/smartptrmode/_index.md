## SmartPtrMode enum


[SmartPtr](../smartptr/) pointer type: weak or shared. Defines whether pointer is being counted when it is being decided whether to delete object or not.

```cpp
enum class SmartPtrMode : char
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Совместный | 0 | Режим совместного использования: указатель участвует в подсчёте ссылок. |
| Weak | 1 | Режим слабого указателя: указатель не участвует в подсчёте ссылок. |
