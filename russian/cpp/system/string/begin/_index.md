## String::begin method


Возвращает указатель на начало фактического буфера строки. Никогда ничего не переаллоцирует. Не гарантирует, что буфер будет завершён нулём.

```cpp
const UChar * System::String::begin() const
```


### ReturnValue

Указатель на фактический буфер строки, нуль‑терминированный или нет.
