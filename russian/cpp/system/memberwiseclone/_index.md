## System::MemberwiseClone method


Выполняет клонирование членов с использованием конструктора копирования.

```cpp
template<typename T> SmartPtr<Object> System::MemberwiseClone(T *ptr)
```


| Параметр | Описание |
| --- | --- |
| T | Класс, который будет скопирован конструктором. Информация о подклассе теряется. |

| Параметр | Тип | Описание |
| --- | --- | --- |
| ptr | T * | Указатель на объект для клонирования. |

### ReturnValue

Указатель на клонированный объект.
