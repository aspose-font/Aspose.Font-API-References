## Decimal::Truncate method

```cpp
static Decimal System::Decimal::Truncate(const Decimal &d)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| d | const Decimal\& | Десятичное число |

### ReturnValue

Объект [Decimal](../), представляющий десятичное значение, у которого целая часть равна целой части указанного значения, а дробные цифры все равны 0.
