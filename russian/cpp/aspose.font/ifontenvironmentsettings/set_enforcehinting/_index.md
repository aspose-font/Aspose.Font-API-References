## IFontEnvironmentSettings::set_EnforceHinting method


Этот флаг управляет логикой использования подсказок байткода. Если он установлен в true, шрифт должен включать подсказки байткода, если false — подсказки байткода будут включаться/выключаться в зависимости от логики по умолчанию. По умолчанию false.

```cpp
virtual void Aspose::Font::IFontEnvironmentSettings::set_EnforceHinting(bool value)=0
```
