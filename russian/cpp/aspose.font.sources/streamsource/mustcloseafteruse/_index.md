## StreamSource::MustCloseAfterUse method


Наследники могут препятствовать закрытию потока. Возвращает true, если источник потока хочет, чтобы поток закрывался после использования. В противном случае возвращает false.

```cpp
virtual bool Aspose::Font::Sources::StreamSource::MustCloseAfterUse()
```


### ReturnValue

True, если источник потока требует закрыть поток после использования, иначе false.
