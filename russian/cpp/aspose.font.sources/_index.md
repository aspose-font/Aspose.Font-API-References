## Классы

| Класс | Описание |
| --- | --- |
| [ByteContentStreamSource](./bytecontentstreamsource/) | Представляет источник потока, основанный на потоке содержимого. |
| [FileSystemStreamSource](./filesystemstreamsource/) | Представляет источник потока, основанный на файловой системе. |
| [FontDefinition](./fontdefinition/) | Представляет определение набора файлов [Font](../aspose.font/font/). Этот класс содержит поля, которые не связаны с внутренними данными шрифта. Эти поля описывают размещение шрифта и другие данные, необходимые для загрузки шрифта из некоторого источника шрифта (файл, память и т.д.). |
| [FontFileDefinition](./fontfiledefinition/) | Представляет определение файла [Font](../aspose.font/font/). |
| [StreamSource](./streamsource/) | Определяет способ получения файлового потока, когда он нужен. |
