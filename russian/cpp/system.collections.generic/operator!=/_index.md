## System::Collections::Generic::operator!= method


Сравнивает две пары ключ‑значение, используя обратную семантику 'equals'.

```cpp
template<typename TKey,typename TValue> bool System::Collections::Generic::operator!=(const KeyValuePair<TKey, TValue> &left, const KeyValuePair<TKey, TValue> &right)
```


| Параметр | Описание |
| --- | --- |
| TKey | Тип ключа. |
| TValue | Тип значения. |

| Параметр | Тип | Описание |
| --- | --- | --- |
| лево | const KeyValuePair\<TKey, TValue\>\& | Операнд LHS. |
| право | const KeyValuePair\<TKey, TValue\>\& | Операнд RHS. |

### ReturnValue

True, если оба ключа и значения не совпадают, иначе false.
