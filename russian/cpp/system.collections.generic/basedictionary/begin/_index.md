## BaseDictionary::begin method


Возвращает итератор к обёртке KVPair для элемента ключ‑значение контейнера. Реализовано в стиле C# — итератор должен возвращать объект KVPair с интерфейсом get_Key() и get_Value(). Если контейнер пуст, возвращённый итератор будет равен [end()](../../ienumerable/end/).

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```


### ReturnValue

Итератор, указывающий на первый элемент коллекции.
