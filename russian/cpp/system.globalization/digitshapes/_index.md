## DigitShapes enum


Указывает отображение цифр.


### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Context | 0 | Форма цифры зависит от предыдущего текста. |
| Нет | 1 | Форма цифры не изменяется. |
| NativeNational | 2 | ASCII-цифры 0..9 заменяются национальными цифрами. |
