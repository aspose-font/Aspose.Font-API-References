## LicenseFlags class


Представляет вспомогательную обёртку для флагов внедрения из таблицы 'OS/2' (поле fsType).

```cpp
class LicenseFlags : public System::Object
```

## Методы

| Метод | Описание |
| --- | --- |
| [get_FSType](./get_fstype/)() const | Получает необработанное значение fsType из таблицы OS/2 или 0, если это поле отсутствует в таблице. |
| [get_FSTypeAbsent](./get_fstypeabsent/)() const | Возвращает true, если флаг fsType отсутствует в таблице 'OS/2'. |
| [get_IsBitmapOnlyEmbedding](./get_isbitmaponlyembedding/)() | Определяет, разрешает ли fsType встраивание только в виде BitmapOnly. |
| [get_IsEditableEmbedding](./get_iseditableembedding/)() | Определяет, разрешает ли fsType встраивание Editable. |
| [get_IsInstallableEmbedding](./get_isinstallableembedding/)() | Определяет, является ли fsType встраиванием Installable. |
| [get_IsNoSubsettingEmbedding](./get_isnosubsettingembedding/)() | Определяет, разрешает ли fsType встраивание NoSubsetting. |
| [get_IsPreviewAndPrintEmbedding](./get_ispreviewandprintembedding/)() | Определяет, разрешает ли fsType встраивание Preview и Print. |
| [get_IsReservedType](./get_isreservedtype/)() | Определяет, установлен ли зарезервированный бит (бит 0) для fsType. |
| [get_IsRestrictedLicenseEmbedding](./get_isrestrictedlicenseembedding/)() | Определяет, является ли fsType встраиванием Restricted [Лицензия](../../aspose.font/license/). |
