## AuthenticationLevel enum


Флаги аутентификации, специфичные для WebRequest.

```cpp
enum class AuthenticationLevel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Нет | 0 | Аутентификация не требуется ни для клиента, ни для сервера. |
| MutualAuthRequested | 1 | Запрос не завершится с ошибкой, если сервер не аутентифицирован. |
| MutualAuthRequired | 2 | Текущее приложение получит 'IOException', когда сервер не аутентифицирован. |
