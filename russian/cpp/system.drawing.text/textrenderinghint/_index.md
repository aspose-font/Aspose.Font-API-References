## TextRenderingHint enum


### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SystemDefault | 0 | Глиф с системной подсказкой рендеринга. |
| SingleBitPerPixelGridFit | 1 | Битовая карта глифа с подсказками. |

| AntiAlias | 4 | Битовая карта глифа со сглаживанием без подсказок. |
