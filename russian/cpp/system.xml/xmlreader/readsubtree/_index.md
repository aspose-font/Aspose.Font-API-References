## XmlReader::ReadSubtree method


Возвращает новый экземпляр [XmlReader](../), который можно использовать для чтения текущего узла и всех его потомков.

```cpp
virtual SharedPtr<XmlReader> System::Xml::XmlReader::ReadSubtree()
```


### ReturnValue

Новый экземпляр XML‑читателя, установленный в состояние [ReadState::Initial](../../readstate/). Вызов метода [XmlReader::Read](../read/) позиционирует новый читатель на узле, который был текущим до вызова метода [XmlReader::ReadSubtree](./).
