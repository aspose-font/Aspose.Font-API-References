## XmlWriterSettings::get_NewLineChars method


Возвращает строку символов, используемую для разрывов строк.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

Строка символов, используемая для разрывов строк. Ее можно установить в любое строковое значение. Однако, чтобы обеспечить корректный XML, следует указывать только допустимые символы пробельных знаков, такие как пробелы, табуляции, возвраты каретки или переводы строки. По умолчанию используется **\r\n** (возврат каретки, новая строка).
