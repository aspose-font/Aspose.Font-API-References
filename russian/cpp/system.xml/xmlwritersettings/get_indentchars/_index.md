## XmlWriterSettings::get_IndentChars method


Возвращает строку символов, используемую для отступов. Этот параметр используется, когда значение [XmlWriterSettings::set_Indent](../set_indent/) установлено в **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Строка символов, используемая для отступов. Её можно задать любым строковым значением. Однако, чтобы обеспечить корректный XML, следует указывать только допустимые символы пробела, такие как пробелы, табуляции, возвраты каретки или переводы строки. По умолчанию используется два пробела.
