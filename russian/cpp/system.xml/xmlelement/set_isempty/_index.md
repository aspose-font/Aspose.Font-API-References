## XmlElement::set_IsEmpty method


Устанавливает формат тега элемента.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| значение | bool | Значение для установки. При установке этого значения, если оно равно **true**, дочерние элементы удаляются, и элемент сериализуется в формате короткого тега. Если оно равно **false**, значение изменяется (независимо от того, имеет ли элемент содержимое); если элемент пуст, он сериализуется в длинном формате. |
