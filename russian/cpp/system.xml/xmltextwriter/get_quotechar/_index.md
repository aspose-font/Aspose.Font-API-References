## XmlTextWriter::get_QuoteChar method


Возвращает, какой символ использовать для кавычек значений атрибутов.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### ReturnValue

Символ, используемый для заключения значений атрибутов в кавычки. Это должна быть одинарная кавычка **'** или двойная кавычка **\"**. По умолчанию используется двойная кавычка.
