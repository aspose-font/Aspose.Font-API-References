## WhitespaceHandling enum


Указывает, как обрабатываются пробельные символы.

```cpp
enum class WhitespaceHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Все | 0 | Возвращать узлы **Whitespace** и **SignificantWhitespace**. Это значение по умолчанию. |
| Значимый | 1 | Возвращать только узлы **SignificantWhitespace**. |
| Нет | 2 | Не возвращать узлы **Whitespace** и **SignificantWhitespace**. |
