## Классы

| Класс | Описание |
| --- | --- |
| [FontSpecificEncodings](./fontspecificencodings/) | Представляет специфические кодировки для ориентированных на потребителя шрифтов. |
| [Type1Encoding](./type1encoding/) | Представляет кодировку [Type1](./)[Font](../aspose.font/font/). |
| [Type1Font](./type1font/) | Представляет [Type1](./)[Font](../aspose.font/font/). |
| [Type1FontMetrics](./type1fontmetrics/) | Представляет метрики [Type1](./)[Font](../aspose.font/font/). |
| [Type1MetricFont](./type1metricfont/) | [Type1](./) реализация метрического шрифта. Этот шрифт type1 создаётся только на основе метрик. Функции получения [Glyphs](../aspose.font.glyphs/) и некоторые другие, требующие реального шрифта, не разрешены; запрещённые функции бросают исключение [Type1NotSupportedException](../). Другие свойства (FontName, Weight, Metrics и Encoding) берутся из файла метрик. |
