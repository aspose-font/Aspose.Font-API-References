## Version16Dot16::get_MinorNumber method


Получает номер минорной версии. Значение имеет смысл только в шестнадцатеричной нотации, например версия 0.5 для 'maxp' в реальных файлах шрифтов состоит из 4 байтов: {0, 0, 80, 0}, что имеет шестнадцатеричное представление 0x00005000. После чтения этой версии из файла шрифта, номера старшего и младшего для объекта [Version16Dot16](../) будут 0 и 20480 соответственно. И эти значения в шестнадцатеричной форме равны 0x0000 и 0x5000.
