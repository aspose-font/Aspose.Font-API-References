## FlatStyle enum


Определяет, будет ли элемент управления отрисован плоским или трёхмерным.

```cpp
enum class FlatStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) отрисован плоским. |
| Popup | 1 | [Control](../control/) отрисован плоским, если только мышь не перемещается над ним, когда он отрисован как 3D. |
| Standard | 2 | [Control](../control/) отрисован как 3D. |
| System | 3 | Стиль контролируется ОС. |
