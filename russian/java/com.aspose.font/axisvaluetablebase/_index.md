Базовый класс для структуры Axis Value Table. Спецификация: Axis Value Tables предоставляют детали относительно конкретного значения атрибута стиля на определённой оси вариации дизайна, либо комбинации значений осей вариации дизайна, и взаимосвязи этих значений с метками, используемыми как элементы в названиях подсемейств.
