## FontType enumeration

Указывает тип шрифта.

```csharp
public enum FontType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| TTF | `0` | Тип шрифта TTF (TrueType) и связанные параметры. |
| Type1 | `1` | Тип шрифта Type1. |
| CFF | `2` | Тип шрифта CFF (Compact font format). |
| OTF | `3` | Шрифт OpenType. Формат шрифта OpenType является расширением формата шрифта TrueType, добавляя поддержку данных шрифта PostScript. |
