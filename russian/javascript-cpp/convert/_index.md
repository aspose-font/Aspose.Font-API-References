## Convert functions

| Функция | Описание |
| -------- | ----------- |
| [AsposeFontConvert](./asposefontconvert) | Конвертировать шрифт в поддерживаемые форматы. |
| [AsposeFontConvertToTTF](./asposefontconverttottf/) | Преобразовать шрифт в формат TTF. |
| [AsposeFontConvertToWOFF](./asposefontconverttowoff/) | Преобразовать шрифт в формат WOFF. |
| [AsposeFontConvertToWOFF2](./asposefontconverttowoff2/) | Преобразовать шрифт в формат WOFF2. |
| [AsposeFontConvertToSVG](./asposefontconverttosvg/) | Преобразовать шрифт в формат SVG. |

Функции конвертации шрифтов в форматы TrueType.
