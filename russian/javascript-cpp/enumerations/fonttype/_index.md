## FontType enumeration

Указывает тип шрифта.

```csharp
public enum FontType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| TTF | `0` | TTF (TrueType) тип шрифта и связанные параметры. |
| Type1 | `1` | Тип шрифта Type1. |
| CFF | `2` | CFF (Compact font format) тип шрифта. |
| OTF | `3` | OpenType Font. Формат шрифта OpenType является расширением формата шрифта TrueType, добавляя поддержку данных шрифтов PostScript. |
