> Aspose.Font for JavaScript via C++ allows developers manipulate them Font files directly in the Web.
>
> Such operations are very time consuming, so we recommend using Web Worker. 


## Convert functions

| Функция | Описание |
| -------- | ----------- |
| [AsposeFontConvert](./convert/) | Преобразовать шрифт в TrueType шрифты. |
| [AsposeFontConvertToTTF](./convert/asposefontconverttottf/) | Преобразовать шрифт в формат TTF. |
| [AsposeFontConvertToWOFF](./convert/asposefontconverttowoff/) | Преобразовать шрифт в формат WOFF. |
| [AsposeFontConvertToWOFF2](./convert/asposefontconverttowoff2/) | Преобразовать шрифт в формат WOFF2. |
| [AsposeFontConvertToSVG](./convert/asposefontconverttosvg/) | Преобразовать шрифт в формат SVG. |
