## XmlSchemaDerivationMethod enum


Fournit différentes méthodes pour empêcher la dérivation.

```cpp
enum class XmlSchemaDerivationMethod
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Vide | 0 | Remplace la méthode de dérivation par défaut pour autoriser toute dérivation. |
| Substitution | 1 | Fait référence aux dérivations par **Substitution**. |
| Extension | 2 | Fait référence aux dérivations par **Extension**. |
| Restriction | 4 | Fait référence aux dérivations par **Restriction**. |
| List | 8 | Fait référence aux dérivations par **List**. |
| Union | 16 | Fait référence aux dérivations par **Union**. |
| Tous | 255 | **#all**. Fait référence à toutes les méthodes de dérivation. |
| Aucun | 256 | Accepte la méthode de dérivation par défaut. |
