## XmlSchemaMaxInclusiveFacet class


Représente l'élément **maxInclusive** du XML [Schema](../) tel que spécifié par le World Wide [Web](../../system.web/) Consortium (W3C). Cette classe peut être utilisée pour spécifier une restriction sur la valeur maximale d'un élément **simpleType**. La valeur de l'élément doit être inférieure ou égale à la valeur de l'élément **maxInclusive**.

```cpp
class XmlSchemaMaxInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
