## XmlSchemaComplexType::get_BlockResolved method


Renvoie la valeur après que le type a été compilé vers l'ensemble d'informations post-validation du schéma (infoset). Cette valeur indique comment le type est appliqué lorsque **xsi:type** est utilisé dans le document d'instance.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

La valeur de l'infoset post-validation du schéma. La valeur par défaut est **BlockDefault** sur l'élément **schema**.
