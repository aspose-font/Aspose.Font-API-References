## System::MakeSharedPtr(const X *) method


Convertit un pointeur brut en pointeur intelligent. Surcharge pour les pointeurs const. Utile, par ex., lors de l’utilisation de la variable 'this' dans les méthodes C# traduites en const.

```cpp
template<class X> SmartPtr<X> System::MakeSharedPtr(const X *p)
```


| Paramètre | Description |
| --- | --- |
| X | Type pointeur. |

| Paramètre | Type | Description |
| --- | --- | --- |
| p | const X * | Pointeur brut vers l'objet. |

### ReturnValue

Pointeur intelligent partagé vers l'objet.
