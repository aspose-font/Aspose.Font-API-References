## Object::Is method


Vérifie si l'objet représente une instance du type décrit par targetType. Analogue de l'opérateur C# 'is'.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```


| Paramètre | Type | Description |
| --- | --- | --- |
| targetType | const TypeInfo\& | Structure [TypeInfo](../../typeinfo/) décrivant le type contre lequel tester l'objet actuel. |

### ReturnValue

Vrai si l'objet est du type balisé ou de l'une de ses sous‑classes, sinon faux.
