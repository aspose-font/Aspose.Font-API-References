## Array::cbegin method


Renvoie un itérateur vers le premier élément const‑qualifié du conteneur. Si le conteneur est vide, l'itérateur retourné sera égal à [cend()](../cend/).

```cpp

```


### ReturnValue

Un itérateur pointant vers le premier élément const‑qualifié du conteneur.
