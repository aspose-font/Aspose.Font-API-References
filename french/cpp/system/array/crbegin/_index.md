## Array::crbegin method


Renvoie un itérateur inverse vers le premier élément du conteneur inversé. Il correspond au dernier élément du conteneur non inversé. Si le conteneur est vide, l'itérateur retourné est égal à [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

Un itérateur pointant vers le dernier élément const‑qualifié du conteneur.
