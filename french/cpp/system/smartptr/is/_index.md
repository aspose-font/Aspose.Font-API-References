## SmartPtr::Is method


Vérifie si l’objet pointé est d’un type spécifique ou d’un de ses sous‑types. Suit la sémantique C# 'is'.

```cpp
bool System::SmartPtr<T>::Is(const System::TypeInfo &target) const
```


| Paramètre | Type | Description |
| --- | --- | --- |
| target | const System::TypeInfo\& | Spécifie le type cible à vérifier. |

### ReturnValue

Vrai si la vérification de type à la C# 'is' est positive, sinon faux.
