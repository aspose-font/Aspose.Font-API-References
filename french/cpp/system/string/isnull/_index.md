## String::IsNull method


Vérifie si la chaîne est considérée comme nulle. [String](../) est nulle uniquement si elle est construite via le constructeur [String()](../string/), déplacée, copiée ou assignée à partir d'une chaîne nulle ou si la méthode [reset()](../reset/) a été appelée.

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true si la chaîne est considérée comme nulle, false sinon.
