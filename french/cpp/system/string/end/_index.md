## String::end method


Renvoie un pointeur vers la fin du tampon de chaîne réel. Ne réalloue jamais rien. Ne garantit pas que le tampon soit terminé par un caractère nul.

```cpp
const UChar * System::String::end() const
```


### ReturnValue

Pointeur vers le premier élément après le tampon de chaîne réel, terminé par nul ou non.
