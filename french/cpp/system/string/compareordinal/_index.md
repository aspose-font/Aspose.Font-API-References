## String::CompareOrdinal(const String\&, const String\&) method


Less-equal-greater compare deux chaînes en mode ordinal.

```cpp
static int System::String::CompareOrdinal(const String &strA, const String &strB)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| strA | const String\& | Première chaîne à comparer. |
| strB | const String\& | Deuxième chaîne à comparer. |

### ReturnValue

Valeur négative si la première sous-chaîne est inférieure à la seconde, zéro si elles correspondent, valeur positive sinon.
