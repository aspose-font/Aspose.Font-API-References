## Uri::GetComponents method


Renvoie les composants spécifiés de l'URI représenté par l'objet actuel en utilisant l'échappement spécifié.

```cpp
String System::Uri::GetComponents(UriComponents components, UriFormat format) const
```


| Paramètre | Type | Description |
| --- | --- | --- |
| components | UriComponents | Une combinaison binaire des valeurs de [UriComponents](../../uricomponents/) qui spécifie quelles parties de l'URI retourner |
| format | UriFormat | Spécifie comment les caractères spéciaux sont échappés |

### ReturnValue

Les composants demandés
