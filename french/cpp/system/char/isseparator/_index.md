## Char::IsSeparator(char_t) method


Détermine si le caractère spécifié est classé comme un caractère séparateur.



| Paramètre | Type | Description |
| --- | --- | --- |
| c | char_t | Le caractère à tester |

### ReturnValue

Vrai si le caractère spécifié est un caractère séparateur, sinon - false
