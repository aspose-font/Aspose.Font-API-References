## Char::IsLetterOrDigit(char_t) method


Détermine si le caractère spécifié est classé comme une lettre Unicode ou un chiffre décimal.

```cpp
static bool System::Char::IsLetterOrDigit(char_t c)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| c | char_t | Le caractère à tester |

### ReturnValue

Vrai si le caractère spécifié est une lettre Unicode ou un chiffre décimal, sinon - faux
