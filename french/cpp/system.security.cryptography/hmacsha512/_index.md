## HMACSHA512 class


Code de message basé sur le hachage [Authentication](../../system.security.authentication/) qui utilise la fonction de hachage [SHA512](../sha512/). Implémentation partielle. Les objets de cette classe doivent être alloués uniquement à l'aide de la fonction [System::MakeObject()](../../system/makeobject/) function. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this class into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.
