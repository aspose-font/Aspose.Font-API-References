## StringCollection::rbegin() const method


Renvoie un itérateur inverse vers le premier élément du conteneur inversé. Il correspond au dernier élément du conteneur non inversé. Si le conteneur est vide, l'itérateur renvoyé est égal à [rend()](../rend/).

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::rbegin() const noexcept
```


### ReturnValue

Un itérateur pointant vers le dernier élément du conteneur qualifié const.
