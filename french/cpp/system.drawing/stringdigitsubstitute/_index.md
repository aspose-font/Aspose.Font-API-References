## StringDigitSubstitute enum


Spécifie comment les chiffres d'une chaîne sont substitués selon la locale ou la langue.

```cpp
enum class StringDigitSubstitute
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Utilisateur | 0 | Substitution définie par l'utilisateur. |
| Aucun | 1 | La substitution est désactivée. |
| National | 2 | Substituer les chiffres qui correspondent à la langue nationale officielle de la locale de l'utilisateur. |
| Traditionnel | 3 | Substituer les chiffres qui correspondent à l'écriture ou à la langue native de l'utilisateur, qui peuvent ne pas être les mêmes que la langue nationale officielle de la locale de l'utilisateur. |
