## Color::get_IsNamedColor method


Retourne une valeur qui indique si la structure [Color](../) représente une couleur nommée ou un membre de l'énumération [KnownColor](../../knowncolor/).

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```


### ReturnValue

Vrai si le [Color](../) a été créé en utilisant soit la méthode FromName(String), soit la méthode [FromKnownColor(KnownColor)](../fromknowncolor/) ; sinon, il retourne false.
