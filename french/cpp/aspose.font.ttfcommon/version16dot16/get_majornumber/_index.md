## Version16Dot16::get_MajorNumber method


Obtient le numéro de version majeur. La valeur n'a de sens qu'en notation hexadécimale, par exemple la version 0,5 pour 'maxp' dans les fichiers de police réels est de 4 octets : {0, 0, 80, 0}, ce qui a la représentation hexadécimale 0x00005000. Après lecture de cette version depuis le fichier de police, les numéros majeur et mineur pour l'objet [Version16Dot16](../) seront 0 et 20480 respectivement. Et ces valeurs sous forme hexadécimale sont 0x0000 et 0x5000.
