## CultureTypes enum


Entrées du masque de bits des catégories culturelles.

```cpp
enum class CultureTypes : int32_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NeutralCultures | 1 | Culture spécifique à une langue mais pas à un pays ou une région. |
| SpecificCultures | 2 | Spécifique à un pays/région. |
| InstalledWin32Cultures | 4 | Cultures installées dans le système d'exploitation. |
| AllCultures | n/a | Toutes les cultures disponibles. |
| UserCustomCulture | 8 | Cultures définies par l'utilisateur. |
| ReplacementCultures | 16 | Remplacements définis par l'utilisateur pour les cultures existantes. |
| WindowsOnlyCultures | 32 | Obsolète. |
| FrameworkCultures | 64 | Obsolète. |
