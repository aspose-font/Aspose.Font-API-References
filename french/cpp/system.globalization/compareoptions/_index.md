## CompareOptions enum


[String](../../system/string/) comparison options.

```cpp
enum class CompareOptions : int32_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Aucun | 0 | Aucune option spéciale. |
| IgnoreCase | 1 | Ignorer la casse. |
| IgnoreNonSpace | 2 | Ignorer les caractères combinés non espacés, p. ex. les diacritiques. |
| IgnoreSymbols | 4 | Inclure les espaces, les signes de ponctuation, etc. |
| IgnoreKanaType | 8 | Ignorer le type kana (japonais). |
| IgnoreWidth | 16 | Ignorer la largeur des caractères lors de la comparaison des chaînes. |
| OrdinalIgnoreCase | 268435456 | Comparaison ordinale en ignorant la différence de casse. |
| StringSort | 536870912 | Utiliser l'algorithme de tri de chaînes pour comparer les caractères. |
| Ordinal | 1073741824 | Comparer les codes UTF directement pour la première comparaison. |
