Obtient le modèle négatif du pourcentage.



### ReturnValue

Indice du modèle dans le tableau {"-n %", "-n%", "-%n", "%-n", "%n-", "n-%", "n%-", "-% n", "n %-", "% n-", "% -n", "n- %"}, où "n" est le nombre, "%" est le signe de pourcentage et "-" est le signe moins.
