| Carré | 1 | Terminaison de ligne carrée. |
| Arrondi | 2 | Terminaison de ligne ronde. |
| Triangle | 3 | Terminaison de ligne triangulaire. |
| NoAnchor | 16 | Pas d'ancre. |
| SquareAnchor | 17 | Extrémité de ligne d'ancre carrée. |
| RoundAnchor | 18 | Extrémité de ligne d'ancre ronde. |
