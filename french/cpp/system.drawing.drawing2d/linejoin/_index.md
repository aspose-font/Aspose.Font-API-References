## LineJoin enum


Spécifie comment les lignes ou courbes consécutives sont jointes.

```cpp
enum class LineJoin
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Miter | 0 | Jointure en onglet qui produit un coin aigu ou tronqué selon que la longueur de l'onglet dépasse la limite d'onglet. |
| Bevel | 1 | Jointure biseautée qui produit un coin diagonal. |
| Arrondi | 2 | Jointure arrondie qui produit un arc circulaire lisse entre les lignes. |
| MiterClipped | 3 | Jointure en onglet qui produit un coin aigu ou biseauté selon que la longueur de l'onglet dépasse la limite d'onglet. |
