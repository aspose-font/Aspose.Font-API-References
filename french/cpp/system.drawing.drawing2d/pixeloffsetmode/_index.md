## PixelOffsetMode enum


Spécifie comment les pixels sont décalés lors du rendu.

```cpp
enum class PixelOffsetMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |

| Default | n/a | Le mode par défaut. |
