## NamespaceHandling enum


Spécifie s'il faut supprimer les déclarations d'espaces de noms en double dans le [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Spécifie que les déclarations d'espaces de noms en double ne seront pas supprimées. |
| OmettreDoublons | 1 | Spécifie que les déclarations d'espaces de noms en double seront supprimées. Pour que l'espace de noms en double soit supprimé, le préfixe et l'espace de noms doivent correspondre. |
