## WhitespaceHandling enum


Spécifie comment les espaces blancs sont gérés.

```cpp
enum class WhitespaceHandling
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Tous | 0 | Retourne les nœuds **Whitespace** et **SignificantWhitespace**. C’est la valeur par défaut. |
| Significatif | 1 | Retourne uniquement les nœuds **SignificantWhitespace**. |
| Aucun | 2 | Ne retourne aucun nœud **Whitespace** ni **SignificantWhitespace**. |
