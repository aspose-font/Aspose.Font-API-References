## XmlNodeOrder enum


Décrit l'ordre du document d'un nœud par rapport à un second nœud.

```cpp
enum class XmlNodeOrder
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Avant | 0 | Le nœud actuel de ce navigateur est avant le nœud actuel du navigateur fourni. |
| Après | 1 | Le nœud actuel de ce navigateur est après le nœud actuel du navigateur fourni. |
| Identique | 2 | Les deux navigateurs sont positionnés sur le même nœud. |
| Inconnu | 3 | Les positions des nœuds ne peuvent pas être déterminées dans l'ordre du document, l'une par rapport à l'autre. Cela peut se produire si les deux nœuds se trouvent dans des arbres différents. |
