## XmlTextWriter::get_Indentation method


Retourne le nombre de IndentChars à écrire pour chaque niveau dans la hiérarchie lorsque [XmlTextWriter::set_Formatting](../set_formatting/) est défini sur [Formatting::Indented](../../formatting/).

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```

### ReturnValue

Nombre de **IndentChars** pour chaque niveau. La valeur par défaut est 2.
