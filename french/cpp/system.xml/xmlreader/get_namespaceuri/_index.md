## XmlReader::get_NamespaceURI method


Lorsqu'il est remplacé dans une classe dérivée, obtient l'URI d'espace de noms (tel que défini dans la spécification d'espace de noms du W3C) du nœud sur lequel le lecteur est positionné.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```


### ReturnValue

L'URI d'espace de noms du nœud actuel ; sinon une chaîne vide.
