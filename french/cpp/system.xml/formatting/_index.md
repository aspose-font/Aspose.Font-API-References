## Formatting enum


Spécifie les options de mise en forme pour le [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Aucun | 0 | Aucun formatage spécial n'est appliqué. C'est la valeur par défaut. |
| Indented | 1 | Provoque l'indentation des éléments enfants selon les paramètres [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) et [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |
