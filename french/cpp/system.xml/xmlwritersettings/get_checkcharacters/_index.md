## XmlWriterSettings::get_CheckCharacters method


Renvoie une valeur indiquant si l’écrivain XML doit vérifier que tous les caractères du document sont conformes à la section "2.2 Characters" de la [Recommandation XML 1.0](https://www.w3.org/TR/REC-xml/#charsets) du W3C.

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```


### ReturnValue

**true** to do character checking; otherwise, **false**. The default is **true**.
