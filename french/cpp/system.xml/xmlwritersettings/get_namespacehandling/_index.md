## XmlWriterSettings::get_NamespaceHandling method


Renvoie une valeur indiquant si le [XmlWriter](../../xmlwriter/) doit supprimer les déclarations d'espaces de noms en double lors de l'écriture du contenu XML. Le comportement par défaut est que le XmlWriter émet toutes les déclarations d'espaces de noms présentes dans le résolveur d'espaces de noms du XmlWriter.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

L'énumération [NamespaceHandling](../../namespacehandling/) utilisée pour spécifier s'il faut supprimer les déclarations d'espaces de noms en double dans le [XmlWriter](../../xmlwriter/).
