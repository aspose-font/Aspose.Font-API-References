## XmlNodeChangedEventArgs::get_Action method


Renvoie une valeur indiquant le type d'événement de modification de nœud qui se produit.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```


### ReturnValue

L'une des valeurs d'énumération qui décrit l'événement de modification de nœud. La valeur **[Action](../../../system/action/)** ne différencie pas le moment où l'événement s'est produit (avant ou après). Vous pouvez créer des gestionnaires d'événements distincts pour gérer les deux cas.
