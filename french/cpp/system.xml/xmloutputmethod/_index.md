## XmlOutputMethod enum


Spécifie la méthode utilisée pour sérialiser la sortie du [XmlWriter](../xmlwriter/).

```cpp
enum class XmlOutputMethod
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Xml | 0 | Sérialiser selon les règles XML 1.0. |
| Html | 1 | Sérialiser selon les règles HTML spécifiées par XSLT. |
| Texte | 2 | Sérialiser uniquement les blocs de texte. |
| AutoDetect | 3 | Utilisez les règles XSLT pour choisir entre les méthodes de sortie [XmlOutputMethod::Xml](./) et [XmlOutputMethod::Html](./) au moment de l'exécution. |
