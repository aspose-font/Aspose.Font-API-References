## XmlReaderSettings::set_MaxCharactersInDocument method


Définit une valeur indiquant le nombre maximal de caractères autorisés dans un document XML. Une valeur zéro (0) signifie qu'il n'y a aucune limite sur la taille du document XML. Une valeur non nulle spécifie la taille maximale, en caractères.

void System::Xml::XmlReaderSettings::set_MaxCharactersInDocument(int64_t value)
