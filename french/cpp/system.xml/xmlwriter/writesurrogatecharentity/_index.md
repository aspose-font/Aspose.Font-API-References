## XmlWriter::WriteSurrogateCharEntity method


Lorsqu'il est remplacé dans une classe dérivée, génère et écrit l'entité de caractère de substitution pour la paire de caractères de substitution.

```cpp
virtual void System::Xml::XmlWriter::WriteSurrogateCharEntity(char16_t lowChar, char16_t highChar)=0
```


| Paramètre | Type | Description |
| --- | --- | --- |
| lowChar | char16_t | Le substitut bas. Cela doit être une valeur comprise entre 0xDC00 et 0xDFFF. |
| highChar | char16_t | Le substitut haut. Cela doit être une valeur comprise entre 0xD800 et 0xDBFF. |
