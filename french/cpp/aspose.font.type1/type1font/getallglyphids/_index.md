## Type1Font::GetAllGlyphIds method


Retourne un tableau de tous les identifiants de glyphes disponibles dans la [Font](../../../aspose.font/font/). L'identifiant de glyphe est un nombre unique pour un glyphe, qui dépend du type de police. [Type1](../../)[Font](../../../aspose.font/font/) identifiant de glyphe peut être une instance de la classe ([GlyphStringId](../)) ou de la classe ([GlyphUInt32Id](../)).

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Type1::Type1Font::GetAllGlyphIds() override
```


### ReturnValue

Tous les identifiants de glyphes, disponibles dans la [Font](../../../aspose.font/font/).
