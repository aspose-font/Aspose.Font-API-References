## SearchOption enum


Spécifie qu'une recherche doit être effectuée uniquement dans le répertoire actuel, ou dans le répertoire actuel et tous ses sous‑répertoires.

```cpp
enum class SearchOption
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| AllDirectories | 0 | Rechercher dans le répertoire actuel et tous ses sous‑répertoires. |
| TopDirectoryOnly | 1 | Rechercher uniquement dans le répertoire actuel. |
