## FileNotFoundException typedef


L'exception qui est levée lorsqu'une tentative d'accès à un fichier qui n'existe pas sur le disque échoue. Ne jamais encapsuler les instances de la classe [FileNotFoundException](./) dans [System::SmartPtr](../../system/smartptr/).

```cpp
using System::IO::FileNotFoundException =  System::ExceptionWrapper<Details_FileNotFoundException>
```
