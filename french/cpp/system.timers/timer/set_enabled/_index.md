## Timer::set_Enabled method


Démarre ou arrête la minuterie. Démarrer la minuterie ne redémarre pas le comptage du temps si la minuterie est déjà en cours d'exécution.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| valeur | bool | true pour démarrer le minuteur, false pour l'arrêter. |
