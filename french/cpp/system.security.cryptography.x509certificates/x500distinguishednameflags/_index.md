## X500DistinguishedNameFlags enum


Règles de formatage du nom distinctif du certificat X509.

```cpp
enum class X500DistinguishedNameFlags
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Aucun | 0 | Aucune caractéristique spéciale. |
| Inversé | 1 | Le nom est réservé. |
| UtiliserPointsVirgules | 16 | Utiliser des points-virgules. |
| NePasUtiliserSignePlus | 32 | Le nom n'utilise pas le signe plus. |
| NePasUtiliserGuillemets | 64 | Désactive les guillemets dans le nom. |
| UtiliserVirgules | 128 | Autorise l'utilisation de virgules. |
| UtiliserNouvellesLignes | 256 | Autorise l'utilisation de sauts de ligne. |
| UtiliserEncodageUTF8 | 4096 | Passe de l'utilisation d'Unicode à l'utilisation du codage UTF-8. |
| UtiliserEncodageT61 | 8192 | Passe au codage T61. |
| ForcerEncodageUTF8 | 16384 | Force l'utilisation de UTF-8 lors du codage de clés X500 spécifiques. |
