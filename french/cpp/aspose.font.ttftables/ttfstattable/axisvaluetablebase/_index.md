## AxisValueTableBase class


Classe de base pour la structure de la table de valeurs d'axe. Spécification : les tables de valeurs d'axe fournissent des détails concernant une valeur d'attribut de style spécifique sur un axe particulier de variation de conception, ou une combinaison de valeurs d'axe de variation de conception, ainsi que la relation de ces valeurs avec les libellés utilisés comme éléments dans les noms de sous-famille.

```cpp
class AxisValueTableBase : public System::Object
```
