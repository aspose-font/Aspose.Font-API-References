## TtfStatTable::get_DesignAxisCount method


Renvoie le nombre d'enregistrements d'axe. Spec : dans une police contenant une table 'fvar', cette valeur doit être supérieure ou égale à la valeur axisCount de la table 'fvar'. Dans toutes les polices, elle doit être supérieure à zéro si axisValueCount est supérieur à zéro.
