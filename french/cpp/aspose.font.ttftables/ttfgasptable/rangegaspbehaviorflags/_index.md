## RangeGaspBehaviorFlags enum


Indicateurs décrivant le comportement souhaité du rasteriseur.

```cpp
enum class RangeGaspBehaviorFlags : uint16_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| aucun | n/a | Optionnel pour les très grandes tailles, généralement ppem>2048. |
| GASP_GRIDFIT | n/a | Utiliser le lissage de grille. |
| GASP_DOGRAY | n/a | Utiliser le rendu en niveaux de gris. |
| GASP_SYMMETRIC_GRIDFIT | n/a | Utiliser le lissage de grille avec lissage symétrique ClearType. Pris en charge uniquement dans la version 1 'gasp'. |
| GASP_SYMMETRIC_SMOOTHING | n/a | Utiliser le lissage le long de plusieurs axes avec ClearType®. Pris en charge uniquement dans la version 1 'gasp'. |
| Réservé | n/a | Indicateurs réservés — définis à 0. |
