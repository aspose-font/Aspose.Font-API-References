## IFontEnvironmentSettings::get_EnforceHinting method


Ce drapeau contrôle la logique d'utilisation du bytecode hinting. S'il est vrai, la police doit activer le bytecode hinting, si false - le bytecode hinting sera activé/désactivé selon la logique par défaut. Par défaut false.

virtual bool Aspose::Font::IFontEnvironmentSettings::get_EnforceHinting()=0
