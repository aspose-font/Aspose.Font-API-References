## Font::GetAllGlyphIds method


Retourne tous les identifiants de glyphes disponibles dans le [Font](../). Un identifiant de glyphe est un nombre unique pour un glyphe, dépendant du type de police. Par exemple : l'id de [Type1](../../../aspose.font.type1/)'s est un nom de glyphe, instance de la classe ([GlyphStringId](../)). L'id de TTF est un indice entier, instance de la classe ([GlyphUInt32Id](../)).

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Font::GetAllGlyphIds() override=0
```


### ReturnValue

Identifiants de glyphes.
