## StreamSource::MustCloseAfterUse method


Les classes dérivées peuvent empêcher la fermeture du flux. Renvoie true si la source du flux souhaite que le flux soit fermé après utilisation. Sinon, renvoie false.

```cpp
virtual bool Aspose::Font::Sources::StreamSource::MustCloseAfterUse()
```


### ReturnValue

Vrai si la source du flux souhaite que le flux soit fermé après utilisation, sinon faux.
