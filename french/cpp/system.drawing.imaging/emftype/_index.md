## EmfType enum


Spécifie les types d'enregistrements qui sont placés dans un fichier EMF.

```cpp
enum class EmfType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) métafichier amélioré. Contient des commandes GDI. Les métafichiers de ce type sont appelés fichier EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) métafichier amélioré plus. Contient des commandes GDI+. Les métafichiers de ce type sont appelés fichier EMF+. |
| EmfPlusDual | n/a | Métafichier amélioré double [Windows](../../system.windows/). Contient des commandes GDI et GDI+ équivalentes. Les métafichiers de ce type sont appelés fichier EMF+. |
