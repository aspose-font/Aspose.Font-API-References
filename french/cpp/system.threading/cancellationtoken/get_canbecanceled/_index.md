## CancellationToken::get_CanBeCanceled method


Obtient si ce jeton peut être dans l'état annulé.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true si ce jeton peut être dans l'état annulé ; sinon, false.
## Remarques



Les jetons créés à partir de [CancellationTokenSource](../../cancellationtokensource/) renverront true, tandis que le jeton None renverra toujours false.
