## SoapHeaderDirection enum


Énumère les directions de l'en-tête SOAP.

```cpp
enum class SoapHeaderDirection
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| In | 1 | Le destinataire de l'en-tête SOAP est le service XML [Web](../../system.web/). |
| InOut | 3 | Les destinataires de l'en-tête SOAP sont à la fois le service XML [Web](../../system.web/) et le client du service XML [Web](../../system.web/). |
| Out | 2 | Le destinataire de l'en-tête SOAP est le client du service XML [Web](../../system.web/). |
| Fault | 4 | L'en-tête SOAP est envoyé au client du service XML [Web](../../system.web/) lorsqu'une exception est levée par la méthode du service XML [Web](../../system.web/). |
