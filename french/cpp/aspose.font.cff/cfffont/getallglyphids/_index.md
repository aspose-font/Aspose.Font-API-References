## CffFont::GetAllGlyphIds method


Retourne un tableau de tous les identifiants de glyphes disponibles dans la [Font](../../../aspose.font/font/). L'identifiant de glyphe est un numéro unique pour un glyphe, dépendant du type de police. L'identifiant de glyphe CFF [Font](../../../aspose.font/font/) peut être une instance de la classe ([GlyphStringId](../)) ou de la classe ([GlyphUInt32Id](../)).

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Cff::CffFont::GetAllGlyphIds() override
```


### ReturnValue

Tous les identifiants de glyphes, disponibles dans la [Font](../../../aspose.font/font/).
