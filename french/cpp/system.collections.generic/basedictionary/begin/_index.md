## BaseDictionary::begin method


Retourne un itérateur vers le wrapper KVPair pour l'élément clé‑valeur du conteneur. Implémenté dans le style C# – l'itérateur doit renvoyer l'objet KVPair avec les interfaces get_Key() et get_Value(). Si le conteneur est vide, l'itérateur retourné sera égal à [end()](../../ienumerable/end/).

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```


### ReturnValue

Un itérateur pointant vers le premier élément de la collection.
