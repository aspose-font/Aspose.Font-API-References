## AutoSizeMode enum


Définit le fonctionnement de l'ajustement automatique.


### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| GrowAndShrink | 0 | Le [Control](../control/) grandit et rétrécit selon les besoins. Il ne peut pas être redimensionné manuellement. |
| GrowOnly | 1 | Le [Control](../control/) grandit selon les besoins mais ne peut pas rétrécir en dessous de la taille minimale. Il peut être redimensionné manuellement. |
