## EncryptionPolicy enum


Énumère les politiques de chiffrement.

```cpp
enum class EncryptionPolicy
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ExigerChiffrement | 0 | Exiger le chiffrement et ne jamais autoriser un chiffre 'Null'. |
| AutoriserPasDeChiffrement | 1 | Préférer utiliser le chiffrement complet mais un chiffre 'Null' peut être utilisé si le serveur accepte. |
| PasDeChiffrement | 2 | Autoriser aucun cryptage et demander qu'un chiffre 'Null' soit utilisé si l'autre extrémité peut gérer un chiffre 'Null'. |
