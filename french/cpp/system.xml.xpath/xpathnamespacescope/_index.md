## XPathNamespaceScope enum


Définit la portée de l'espace de noms.

```cpp
enum class XPathNamespaceScope
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Tous | 0 | Renvoie tous les espaces de noms définis dans la portée du nœud actuel. Cela inclut l'espace de noms **xmlns:xml** qui est toujours déclaré implicitement. L'ordre des espaces de noms renvoyés n'est pas défini. |
| ExcludeXml | 1 | Renvoie tous les espaces de noms définis dans la portée du nœud actuel, à l'exception de l'espace de noms **xmlns:xml**. L'espace de noms **xmlns:xml** est toujours déclaré implicitement. L'ordre des espaces de noms renvoyés n'est pas défini. |
| Local | 2 | Renvoie tous les espaces de noms définis localement au nœud actuel. |
