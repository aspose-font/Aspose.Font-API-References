## XPathNavigator::MoveToId method


Lorsqu'elle est remplacée dans une classe dérivée, se déplace vers le nœud qui possède un attribut de type **ID** dont la valeur correspond à la [String](../../../system/string/) spécifiée.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```


| Paramètre | Type | Description |
| --- | --- | --- |
| id | String | Une [String](../../../system/string/) représentant la valeur **ID** du nœud vers lequel vous souhaitez vous déplacer. |

### ReturnValue

**true** if the [XPathNavigator](../) is successful moving; otherwise, **false**. If **false**, the position of the navigator is unchanged.
