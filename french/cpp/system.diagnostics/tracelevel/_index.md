## TraceLevel enum


Spécifie quels messages afficher pour les classes [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) et System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Désactivé | 0 | Ne génère aucun message de traçage ni de débogage. |
| Erreur | 1 | Génère les messages de gestion des erreurs. |
| Avertissement | 2 | Génère les avertissements et les messages de gestion des erreurs. |
| Info | 3 | Génère les messages d'information, les avertissements et les messages de gestion des erreurs. |
| Verbeux | 4 | Génère tous les messages de débogage et de traçage. |
