## TransformationMatrix class

Représente la matrice de transformation 3x3 &#x7C; AB 0 &#x7C; &#x7C; CD 0 &#x7C; &#x7C; TX TY 1 &#x7C;. Transforme les coordonnées de la manière suivante : x1 = A*x + C*y + TX ; y1 = B*x + D*y + TY.
