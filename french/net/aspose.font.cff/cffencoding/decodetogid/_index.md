## CffEncoding.DecodeToGid method

Obtient Gid pour charCode passé. Cette méthode est conçue pour un CFF CIDFonts, où charCode doit être une valeur CID valide. Glyph id est un numéro unique pour un glyphe, qui dépend du type de police. CFF Font glyph id can be instance of ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) classe ou ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) classe.
