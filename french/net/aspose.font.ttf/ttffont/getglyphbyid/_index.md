## GetGlyphById(GlyphId) {#getglyphbyid}

Renvoie le glyphe par identifiant de glyphe. L'identifiant de glyphe est un numéro unique pour un glyphe, qui dépend du type de police. L'identifiant de glyphe de police TTF peut être une instance de ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) classe ou ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) ) class. L'adressage de glyphe de nom (chaîne) est pris en charge pour les polices TTF via le mappage de table Post. Dans le cas où la police CFF est à l'intérieur, les structures CFF sont utilisées pour adresser les glyphes par leur nom.

```csharp
public override Glyph GetGlyphById(GlyphId id)
```
