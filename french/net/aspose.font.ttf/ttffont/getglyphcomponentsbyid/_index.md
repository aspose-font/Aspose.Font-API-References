## GetGlyphComponentsById(GlyphId, GlyphIdList) {#getglyphcomponentsbyid}

Obtient un glyphe par identifiant de glyphe passé et remplit la liste passée des identifiants de glyphe avec les composants de ce glyphe. L'identifiant de glyphe est un numéro unique pour un glyphe, qui dépend du type de police. TTF L'identifiant de glyphe de police peut être une instance de ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) classe ou ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) ) class. L'adressage de glyphe de nom (chaîne) est pris en charge pour les polices TTF via le mappage de table Post. Dans le cas où la police CFF est à l'intérieur, les structures CFF sont utilisées pour adresser les glyphes par leur nom.

```csharp
public virtual void GetGlyphComponentsById(GlyphId id, GlyphIdList componentsToPopulate)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| id | GlyphId | ID de glyphe. |
| componentsToPopulate | GlyphIdList | Liste des identifiants de glyphe à remplir. |

### Remarques

La collection vide componentsToPopulate doit être transmise et contiendra la liste des identifiants des composants de glyphe.
