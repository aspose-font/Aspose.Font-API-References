## Version16Dot16.MinorNumber property

Obtient ou définit le numéro de version mineure La valeur n'a de sens qu'en notation hexadécimale, par exemple version 0.5 pour 'maxp' dans les fichiers de police réels est de 4 octets : {0, 0, 80, 0}, ce qui a une représentation hexadécimale 0x00005000. Après lecture de cette version à partir du fichier de police, numéros majeurs et mineurs pour l'objet[`Version16Dot16`](../../version16dot16) seront 0 et 20480 respectivement. Et ces valeurs sous forme hexadécimale sont 0x0000 et 0x5000.

public ushort MinorNumber { get; set; }
