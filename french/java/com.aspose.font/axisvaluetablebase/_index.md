Classe de base pour la structure Axis Value Table. Spec: Les Axis Value Tables fournissent des détails concernant une valeur d'attribut de style spécifique sur un axe de variation de conception donné, ou une combinaison de valeurs d'axes de variation de conception, et la relation de ces valeurs aux libellés utilisés comme éléments dans les noms de sous-familles.
