## FontType enumeration

指定字体类型。

```csharp
public enum FontType
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| TTF | `0` | TTF（TrueType）字体类型及相关内容。 |
| Type1 | `1` | Type1 字体类型。 |
| CFF | `2` | CFF（紧凑字体格式）字体类型。 |
| OTF | `3` | OpenType 字体。OpenType 字体格式是 TrueType 字体格式的扩展，增加了对 PostScript 字体数据的支持。 |
