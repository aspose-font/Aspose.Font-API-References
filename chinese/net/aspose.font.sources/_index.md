**Aspose.Font.Sources** 命名空间提供从不同来源（文件、内存流等）访问字体数据的功能。

## 类

| 类 | 描述 |
| --- | --- |
| [ByteContentStreamSource](./bytecontentstreamsource/) | 表示基于内容流的流源。 |
| [FileSystemStreamSource](./filesystemstreamsource/) | 表示基于文件系统的流源。 |
| [FontDefinition](./fontdefinition/) | 表示字体文件集定义。此类包含与字体内部数据无关的字段。这些字段描述字体的放置以及从某些字体来源（文件、内存等）加载字体所需的其他数据。 |
| [FontFileDefinition](./fontfiledefinition/) | 表示 Font 文件定义。 |
| [StreamSource](./streamsource/) | 定义了一种在需要时获取文件流的方法。 |
