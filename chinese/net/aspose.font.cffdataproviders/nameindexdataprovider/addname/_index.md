## NameIndexDataProvider.AddName method

向 Name INDEX 结构添加字体名称。请谨慎使用此方法，因为根据 CFF 规范，CFF Name INDEX 结构中的每个字体名称必须在 Top DICT 索引中拥有相应的 DICT 结构。

```csharp
public abstract void AddName(string name)
```

| name | String | 字体名称 |
