**Aspose.Font.Exceptions** 命名空间提供描述在 Aspose.Font 中使用的异常的类。

## 类

| 类 | 描述 |
| --- | --- |
| [EncodingException](./encodingexception/) | 表示编码异常。当出现编码/解码文本问题时可能抛出此异常。 |
| [FontAgrumentException](./fontagrumentexception/) | 表示 Font 参数异常。当参数使用不当时可能抛出此异常。 |
| [FontConversionException](./fontconversionexception/) | 表示 Font 转换异常。当字体转换过程中出现错误时可能抛出此异常。 |
| [FontCreationException](./fontcreationexception/) | 表示 Font 创建异常。当字体创建过程中出现错误时可能抛出此异常。 |
| [FontException](./fontexception/) | 表示通用的 Font 处理相关异常。 |
| [FontMergeException](./fontmergeexception/) | 表示 Font 合并异常。当合并字体时出现错误可能抛出此异常。 |
| [FontNotSupportedOperationException](./fontnotsupportedoperationexception/) | 表示不支持的操作异常。如果对特定字体类型的某些操作不受支持，则可能抛出此异常。 |
| [IncorrectFontDataException](./incorrectfontdataexception/) | 表示当 Font 对象的某些值无效时的异常。 |
| [LicenseRestrictionException](./licenserestrictionexception/) | 表示在尝试执行在评估模式下受限的功能时可能抛出的异常。 |
| [SvgConversionException](./svgconversionexception/) | 表示针对 SVG 格式的字体转换异常。如果在字体转换过程中出现错误，则可能抛出此异常。 |
