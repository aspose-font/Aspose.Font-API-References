**Aspose.Font.Glyphs** 命名空间提供描述字形对象的类。

## 类

| 类 | 描述 |
| --- | --- |
| [CompositeGlyph](./compositeglyph/) | 表示字体复合字形。 |
| [CompositeGlyphComponent](./compositeglyphcomponent/) | 表示复合字形组件（带有定位矩阵的字形）。 |
| [CompositeGlyphComponentList](./compositeglyphcomponentlist/) | 表示复合字形组件列表。 |
| [Glyph](./glyph/) | 表示字体字形。 |
| [GlyphId](./glyphid/) | 表示字体中可用的字形 ID。字形 ID 是字形的唯一编号，取决于字体类型。例如：Type1 的 ID 是字形名称，是 ([`GlyphStringId`](../aspose.font.glyphs/glyphstringid/)) 类的实例。TTF 的 ID 是整数索引，是 ([`GlyphUInt32Id`](../aspose.font.glyphs/glyphuint32id/)) 类的实例。 |
| [GlyphIdList](./glyphidlist/) | 表示字形 ID 列表。 |
| [GlyphStringId](./glyphstringid/) | 表示字符串字形 ID。 |
| [GlyphUInt32Id](./glyphuint32id/) | 表示整数字形 ID。 |
## 接口

| 接口 | 描述 |
| --- | --- |
| [IGlyphAccessor](./iglyphaccessor/) | 定义检索指定字形标识符和字形的功能。 |
## 枚举

| 枚举 | 描述 |
| --- | --- |
| [GlyphIdType](./glyphidtype/) | 指定字形 ID 的类型。 |
| [GlyphState](./glyphstate/) | 指定字形状态。 |
