## IGlyphAccessor.GetAllGlyphIds method

返回字体中可用的所有字形 ID。字形 ID 是字形的唯一编号，取决于字体类型。例如：Type1 的 ID 是字形名称，实例为 ([`GlyphStringId`](../../glyphstringid/)) 类。TTF 的 ID 是整数索引，实例为 ([`GlyphUInt32Id`](../../glyphuint32id/)) 类。

```csharp
public GlyphId[] GetAllGlyphIds()
```

### 返回值

字形标识符。
