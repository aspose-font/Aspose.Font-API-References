**Aspose.Font** 命名空间提供描述常见 Aspose.Font 对象的类。

## 类

| 类 | 描述 |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | 提供有关当前产品名称和版本的信息。 |
| [Font](./font/) | 表示基础 Font 类。 |
| [FontBBox](./fontbbox/) | 表示 Font 边界框 (BBox)。 |
| [FontEnvironment](./fontenvironment/) | 提供有关当前环境和平台的信息。 |
| [FontFactory](./fontfactory/) | 包含用于打开不同类型字体的功能以及用于创建各种对象的其他方法。 |
| [FontMetrics](./fontmetrics/) | 表示 Font 度量。 |
| [License](./license/) | 提供对组件进行授权的方法。 |
| [Metered](./metered/) | 提供设置计量密钥的方法。 |
| [MultiLanguageString](./multilanguagestring/) | 表示多语言字符串。 |
| [NameToCodeMap](./nametocodemap/) | 表示名称到代码的映射。 |
| [TransformationMatrix](./transformationmatrix/) | 表示 3x3 变换矩阵 &#x7C; A B 0 &#x7C; &#x7C; C D 0 &#x7C; &#x7C; TX TY 1 &#x7C;。以以下方式转换坐标：x1 = A*x + C*y + TX; y1 = B*x + D*y + TY。 |
## 接口

| 接口 | 描述 |
| --- | --- |
| [IEncodingParameters](./iencodingparameters/) | 支持编码参数的通用接口。某些 Font 类型可以拥有多个编码算法/映射。因此，应使用此接口来创建具体的字体编码参数。 |
| [IFont](./ifont/) | 声明所有字体格式的通用功能。由 Font 类实现。 |
| [IFontEncoding](./ifontencoding/) | 定义 Font 编码的接口。 |
| [IFontMetrics](./ifontmetrics/) | 定义 Font 度量工具的接口。 |
| [IFontSaver](./ifontsaver/) | 定义 Font 保存功能的接口。 |
| [ISupportsNameAddressing](./isupportsnameaddressing/) | 定义特定于支持字形名称寻址的编码的成员。 |
