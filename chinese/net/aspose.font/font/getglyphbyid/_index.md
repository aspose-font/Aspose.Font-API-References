## Font.GetGlyphById method

根据 glyph id 返回字形。Glyph id 是针对字形的唯一编号，取决于字体类型。GlyphId - 派生对象。例如：Type1 的 id 是字形名称，属于 ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid/)) 类的实例。TTF 的 id 是整数索引，属于 ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id/)) 类的实例。

```csharp
public abstract Glyph GetGlyphById(GlyphId id)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| id | GlyphId | 字形 id。 |

### 返回值

字形。
