## FontEnvironment.Strictness property

某些字体可能包含意外数据、未指定的特性，或可能被粗略裁剪。建议容错设置，以便消费者在不考虑字体可能不足的情况下打开任何字体。字体内部结构不保证一致。建议严格设置，以便消费者打开大多数有效且稳固的字体。

public ParsingStrictness Strictness { get; set; }
