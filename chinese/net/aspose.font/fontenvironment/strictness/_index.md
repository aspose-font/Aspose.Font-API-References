## FontEnvironment.Strictness property

某些字体可能包含意外数据、未指定的功能，或者可能被粗略裁剪。 容错设置推荐给想要打开任何字体而不考虑字体不足的消费者。字体内部结构不保证一致。 对于想要打开大部分有效和实体字体的消费者，建议进行严格设置。

public ParsingStrictness Strictness { get; set; }
