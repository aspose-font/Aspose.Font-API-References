**Aspose.Font.Type1** 命名空间提供描述与 Type1 字体格式相关对象的类。

## 类

| 类 | 描述 |
| --- | --- |
| [FontSpecificEncodings](./fontspecificencodings/) | 表示面向消费者的字体的特定编码。 |
| [Type1Encoding](./type1encoding/) | 表示 Type1 字体编码。 |
| [Type1Font](./type1font/) | 表示 Type1 字体。 |
| [Type1FontMetrics](./type1fontmetrics/) | 表示 Type1 字体度量。 |
| [Type1MetricFont](./type1metricfont/) | Type1 度量字体实现。此 Type1 字体仅使用度量创建。检索字形的函数以及其他需要真实字体的功能均不被允许，不允许的函数会抛出 Type1NotSupportedException 异常。其他属性（FontName、Weight、Metrics 和 Encoding）取自度量文件。 |
