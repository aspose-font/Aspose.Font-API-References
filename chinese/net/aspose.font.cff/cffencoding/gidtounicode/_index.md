## CffEncoding.GidToUnicode method

将 Gid 解码为 Unicode。Glyph id 是字形的唯一编号，取决于字体类型。CFF Font 的 glyph id 可以是 ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid/)) 类或 ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id/)) 类的实例。

```csharp
public uint GidToUnicode(GlyphId gid)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| gid | GlyphId | 要解码的符号的 glyph 标识符。 |

### 返回值

传入的 glyph id 对应的 Unicode 值。
