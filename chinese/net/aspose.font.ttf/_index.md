**Aspose.Font.Ttf** 命名空间提供描述与 TTF 字体格式相关对象的类。

## 类

| 类 | 描述 |
| --- | --- |
| [LicenseFlags](./licenseflags/) | 表示用于 'OS/2' 表（字段 fsType）中嵌入标志的辅助包装器。 |
| [TtcFontFileDefinition](./ttcfontfiledefinition/) | 表示 TTC 字体的文件定义。 |
| [TtcFontSource](./ttcfontsource/) | 表示 TTC 字体源。 |
| [TtfEncoding](./ttfencoding/) | 表示 TTF 字体编码。 |
| [TtfEncodingParameters](./ttfencodingparameters/) | 表示 TTF 编码参数。应当用于从 TTF 字体请求特定编码。 |
| [TtfFont](./ttffont/) | 表示 TrueType 字体（TTF）。 |
| [TtfFontMetrics](./ttffontmetrics/) | 表示 TTF 字体度量。 |
