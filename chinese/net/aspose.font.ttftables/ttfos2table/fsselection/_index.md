## TtfOs2Table.FSSelection property

包含有关字体模式性质的信息。0 位 1 ITALIC 字体包含斜体字符，否则为正体。1 UNDERSCORE 字符带下划线。2 NEGATIVE 字符的前景色和背景色颠倒。3 OUTLINED 轮廓（空心）字符，否则为实心。4 STRIKEOUT 字符被划线。5 位 0 BOLD 字符加粗。6 REGULAR 字符采用字体的标准粗细/样式。

public ushort FSSelection { get; }
