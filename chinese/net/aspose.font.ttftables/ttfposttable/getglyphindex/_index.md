## TtfPostTable.GetGlyphIndex method

获取按字形名称的字形索引。

```csharp
public uint GetGlyphIndex(string glyphName)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| glyphName | String | 字形名称。 |

### 返回值

字形索引。当此字体文件的字形未提供 PostScript 名称信息时，返回索引 0，即未定义字形或 ".notdef" 字形
