## Version16Dot16.MajorNumber property

获取或设置主要版本号。 值仅在十六进制表示法中有意义，例如实际字体文件中 'maxp' 的 版本 0.5 为 4 个字节：{0, 0, 80, 0}，十六进制表示为 0x00005000. 从字体文件中读取此版本后, 对象的主要和次要数字[`Version16Dot16`](../../version16dot16)将分别为 0 和 20480。 而这些十六进制形式的值是 0x0000 和 0x5000.

```csharp
public ushort MajorNumber { get; set; }
```
