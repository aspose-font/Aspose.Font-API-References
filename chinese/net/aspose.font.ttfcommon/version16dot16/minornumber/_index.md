## Version16Dot16.MinorNumber property

获取或设置次要版本号。该值仅在十六进制表示中有意义，例如实际字体文件中 'maxp' 的版本 0.5 为 4 字节：{0, 0, 80, 0}，其十六进制表示为 0x00005000。读取此版本自字体文件后，[`Version16Dot16`](../) 对象的主版本号和次版本号分别为 0 和 20480。这些值的十六进制形式为 0x0000 和 0x5000。

```csharp
public ushort MinorNumber { get; set; }
```
