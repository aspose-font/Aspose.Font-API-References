## RangeGaspBehaviorFlags enum


描述所需光栅化器行为的标志。

```cpp
enum class RangeGaspBehaviorFlags : uint16_t
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 均不 | n/a | 可选用于非常大的尺寸，通常 ppem>2048。 |
| GASP_GRIDFIT | n/a | 使用网格拟合。 |
| GASP_DOGRAY | n/a | 使用灰度渲染。 |
| GASP_SYMMETRIC_GRIDFIT | n/a | 使用带有 ClearType 对称平滑的网格拟合，仅在版本 1 的 'gasp' 中受支持。 |
| GASP_SYMMETRIC_SMOOTHING | n/a | 使用 ClearType® 多轴平滑，仅在版本 1 的 'gasp' 中受支持。 |
| Reserved | n/a | 保留标志 — 设置为 0。 |
