## XmlSchemaMinExclusiveFacet class


表示来自 XML [Schema](../) 的 **minExclusive** 元素，按照万维网联盟（W3C）的规定。此类可用于对 **simpleType** 元素的最小值进行限制。该元素的值必须大于 **minExclusive** 元素的值。

```cpp
class XmlSchemaMinExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
