## XmlElement::set_IsEmpty method


设置元素的标签格式。

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| 值 | bool | 要设置的值。设置此值时，如果设为 **true**，元素的子节点将被移除，且元素以短标签格式序列化。如果设为 **false**，值会被更改（无论元素是否有内容）；如果元素为空，则以长格式序列化。 |
