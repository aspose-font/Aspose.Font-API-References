## NamespaceHandling enum


指定是否在 [XmlWriter](../xmlwriter/) 中删除重复的命名空间声明。

```cpp
enum class NamespaceHandling
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 默认 | 0 | 指定不会删除重复的命名空间声明。 |
| OmitDuplicates | 1 | 指定将删除重复的命名空间声明。要删除的重复命名空间，其前缀和命名空间必须匹配。 |
