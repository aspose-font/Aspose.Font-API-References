## WhitespaceHandling enum


指定空白字符的处理方式。

```cpp
enum class WhitespaceHandling
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 全部 | 0 | 返回 **Whitespace** 和 **SignificantWhitespace** 节点。这是默认设置。 |
| Significant | 1 | 仅返回 **SignificantWhitespace** 节点。 |
| 无 | 2 | 不返回 **Whitespace** 和 **SignificantWhitespace** 节点。 |
