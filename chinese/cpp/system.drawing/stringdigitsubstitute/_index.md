## StringDigitSubstitute enum


指定根据区域设置或语言对字符串中的数字进行替换的方式。

```cpp
enum class StringDigitSubstitute
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| User | 0 | 用户定义的替换。 |
| 无 | 1 | 替换已禁用。 |
| National | 2 | 替换对应于用户区域设置的官方国家语言的数字。 |
| Traditional | 3 | 替换对应于用户本土文字或语言的数字，这可能与用户区域设置的官方国家语言不同。 |
