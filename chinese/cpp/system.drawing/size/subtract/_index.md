## Size::Subtract method


返回一个新的 [Size](../) 对象，该对象是 **size2** 从 **size1** 中减去的结果，即其宽度值是 **size1's** 宽度值减去 **size2's** 宽度值的结果，且高度值是 **size1's** 高度值减去 **size2's** 高度值的结果。

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| size1 | const Size\& | 用于减去的 [Size](../) 对象 |
| size2 | const Size\& | 要减去的 [Size](../) 对象 |

### ReturnValue

从 **size1** 中减去 **size2** 的结果的 [Size](../) 对象
