## SystemColors class


该类提供一组预先创建的 [Color](../color/) 对象，表示 [Windows](../../system.windows/) 显示元素的颜色。这是一个没有实例服务的静态类型。您绝不应以任何方式创建其实例。

```cpp
class SystemColors
```
