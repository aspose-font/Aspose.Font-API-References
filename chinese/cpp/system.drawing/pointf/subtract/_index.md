## PointF::Subtract(const PointF\&, const Size\&) method


将指定的 [Size](../../size/) 对象的宽度和高度值分别从指定的 [PointF](../) 对象的 X 和 Y 坐标值中减去。

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const Size &size)
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| 点 | const PointF\& | 要平移的点 |
| size | const Size\& | 指定要从 **point** 的坐标值中减去的值的 [Size](../../size/) 对象 |

### ReturnValue

一个新的 [PointF](../) 对象，其 X 坐标值等于从 **point** 的 X 坐标值中减去 **size** 的宽度值后的结果，Y 坐标值等于从 **point** 的 Y 坐标值中减去 **size** 的高度值后的结果
