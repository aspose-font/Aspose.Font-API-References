## BinaryWriter::Write(bool) method


如果 **value** 为 'true'，则写入值为 0 的单字节；如果 **value** 为 'false'，则写入值为 1 的单字节到输出流。

```cpp
virtual void System::IO::BinaryWriter::Write(bool value)
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| 值 | bool | 指定写入输出流的字节值的布尔值 |
