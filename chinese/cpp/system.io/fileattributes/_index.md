```cpp
enum class FileAttributes
```

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 归档 | 32 | 文件系统实体是删除或备份的候选对象。 |
| 已压缩 | 2048 | 文件系统实体已压缩。 |
| 设备 | 64 | 未使用。 |
| 目录 | 16 | 文件系统实体是目录。 |
| 已加密 | 16384 | 文件系统实体已加密。 |
| 隐藏 | 2 | 文件系统实体是隐藏的。 |

| 离线 | 4096 | 文件系统实体处于离线状态，其数据无法立即获取。 |
| 只读 | 1 | 文件系统实体是只读的。 |
| 重新解析点 | 1024 | 文件系统实体是包含重新解析点的文件。 |
