## FileShare enum


指定其他 [FileStream](../filestream/) 对象在打开文件时可以拥有何种访问权限。

```cpp
enum class FileShare
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 无 | 0 | 无访问权限。 |
| Read | 1 | 只读访问。 |
| Write | 2 | 只写访问。 |
| 读写 | 3 | 读取和写入访问权限。 |
| Delete | 4 | 文件可以被删除。 |
| 可继承 | 16 | 使文件句柄可被子进程继承。 |
