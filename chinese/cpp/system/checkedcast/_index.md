## System::CheckedCast method


确定指定值是否在类型 **TTo** 的取值范围内，如果是，则将其转换为类型 **TTo**。

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| 参数 | 描述 |
| --- | --- |
| TTo | 要将指定值转换为的类型 |
| TFrom | 指定值的类型 |

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| 值 | TFrom | 要转换的值 |

### ReturnValue

类型为 **TTo** 的值等价于 **value**
