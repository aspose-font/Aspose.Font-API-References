## DateTime::CompareTo method


比较当前对象和指定的 [DateTime](../) 类实例所表示的两个日期和时间值，并返回指示这些值在时间轴上相对位置的值。

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| value | DateTime | 用于与当前对象比较的 [DateTime](../) 类实例 |

### ReturnValue

如果当前对象表示的值早于 **value** 所表示的值，则返回小于 0 的值；如果两个对象表示的值相同，则返回 0；如果当前对象表示的值晚于 **value** 所表示的值，则返回大于 0 的值
