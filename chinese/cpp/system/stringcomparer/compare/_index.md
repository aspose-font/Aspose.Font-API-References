## StringComparer::Compare method

```cpp
virtual int System::StringComparer::Compare(args_type x, args_type y) const override
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | args_type | 左侧字符串值。 |
| y | args_type | 右侧字符串值。 |

### ReturnValue
