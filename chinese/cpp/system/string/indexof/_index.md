## String::IndexOf(char_t, int, int) const method


子串中的字符向前查找。

```cpp
int System::String::IndexOf(char_t c, int startIndex, int count) const
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| c | char_t | 要查找的字符。 |
| startIndex | int | 开始查找的索引。 |
| count | int | 要遍历的字符数。 |

### ReturnValue

自 startIndex 起的第一个字符位置索引，如果未找到则为 -1。
