## Char::IsLetterOrDigit(char_t) method


确定指定字符是否被分类为 Unicode 字母或十进制数字。

```cpp
static bool System::Char::IsLetterOrDigit(char_t c)
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| c | char_t | 要测试的字符 |

### ReturnValue

如果指定字符是 Unicode 字母或十进制数字，则为 True，否则为 false
