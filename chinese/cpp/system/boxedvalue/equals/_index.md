## BoxedValue::Equals method


确定当前对象和指定对象所表示的装箱值是否相等。

```cpp
bool System::BoxedValue<T>::Equals(ptr obj) override
```


| 参数 | 类型 | 描述 |
| --- | --- | --- |
| obj | ptr | 表示要与当前对象表示的装箱值进行比较的装箱值的对象 |

### ReturnValue

如果当前对象表示的装箱值与 **obj** 相等，则为 true；否则为 false
