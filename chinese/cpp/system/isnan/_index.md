## System::IsNaN method

```cpp
template<typename T> bool System::IsNaN(const T &value)
```


| 参数 | 描述 |
| --- | --- |
| T | 函数检查的值的类型 |

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| 值 | const T\& | 要检查的值 |

### ReturnValue
