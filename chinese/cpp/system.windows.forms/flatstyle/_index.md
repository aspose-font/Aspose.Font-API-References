## FlatStyle enum


定义控件是平面绘制还是三维绘制。

```cpp
enum class FlatStyle
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) 被绘制为平面。 |
| Popup | 1 | [Control](../control/) 被绘制为平面，除非当鼠标移动到其上时它被绘制为 3D 样式。 |
| Standard | 2 | [Control](../control/) 被绘制为 3D 样式。 |
| System | 3 | 样式由操作系统控制。 |
