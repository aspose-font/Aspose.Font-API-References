## PixelOffsetMode enum


指定像素在渲染时的偏移方式。

```cpp
enum class PixelOffsetMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Invalid | n/a | 无效值。 |
| 默认 | n/a | 默认模式。 |
| HighSpeed | n/a | 低质量（高性能）模式。 |
| HighQuality | n/a | 高质量（较低性能）模式。 |
| 无 | n/a | 无像素偏移。 |
| 半 | n/a | 像素在水平和垂直方向上均偏移 -0.5 单位，以实现高性能抗锯齿。 |
