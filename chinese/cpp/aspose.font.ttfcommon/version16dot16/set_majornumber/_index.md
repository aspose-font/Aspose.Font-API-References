## Version16Dot16::set_MajorNumber method


设置主版本号。该值仅在十六进制表示下有意义，例如实际字体文件中 'maxp' 的版本 0.5 为 4 字节：{0, 0, 80, 0}，其十六进制表示为 0x00005000。读取该版本后，对象 [Version16Dot16](../) 的主版本号和次版本号分别为 0 和 20480。这些值的十六进制形式为 0x0000 和 0x5000。
