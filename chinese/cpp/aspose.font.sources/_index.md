## 类

| 类 | 描述 |
| --- | --- |
| [ByteContentStreamSource](./bytecontentstreamsource/) | 表示基于内容流的流源。 |
| [FileSystemStreamSource](./filesystemstreamsource/) | 表示基于文件系统的流源。 |
| [FontDefinition](./fontdefinition/) | 表示 [Font](../aspose.font/font/) 文件集定义。此类包含与字体内部数据无关的字段。这些字段描述字体的放置以及加载字体所需的其他数据（来自某些字体源（文件、内存等））。 |
| [FontFileDefinition](./fontfiledefinition/) | 表示 [Font](../aspose.font/font/) 文件定义。 |
| [StreamSource](./streamsource/) | 定义在需要时获取文件流的方式。 |
