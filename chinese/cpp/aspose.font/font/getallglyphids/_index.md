## Font::GetAllGlyphIds method


返回 [Font](../) 中可用的所有字形 ID。字形 ID 是字形的唯一编号，取决于字体类型。例如：[Type1](../../../aspose.font.type1/) 的 ID 是字形名称，属于 ([GlyphStringId](../)) 类的实例。TTF 的 ID 是整数索引，属于 ([GlyphUInt32Id](../)) 类的实例。

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Font::GetAllGlyphIds() override=0
```


### ReturnValue

字形标识符。
