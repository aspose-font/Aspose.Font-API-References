## 类

| 类 | 描述 |
| --- | --- |
| [FontSpecificEncodings](./fontspecificencodings/) | 表示面向消费者的字体的特定编码。 |
| [Type1Encoding](./type1encoding/) | 表示 [Type1](./)[Font](../aspose.font/font/) 编码。 |
| [Type1Font](./type1font/) | 表示 [Type1](./)[Font](../aspose.font/font/)。 |
| [Type1FontMetrics](./type1fontmetrics/) | 表示 [Type1](./)[Font](../aspose.font/font/) 度量。 |
| [Type1MetricFont](./type1metricfont/) | [Type1](./) 度量字体实现。此 type1 字体仅使用度量创建。不允许使用 [Glyphs](../aspose.font.glyphs/) 检索函数以及其他需要真实字体的功能，不允许的函数会抛出异常 [Type1NotSupportedException](../)。其他属性（FontName、Weight、Metrics 和 Encoding）均来自度量文件。 |
