## Type1Encoding.UnicodeToGid method

Restituisce GlyphId per unicode. Oppure notdef se il carattere non contiene glifo per l'unicode. L'ID glifo è un numero univoco per un glifo, che dipende dal tipo di carattere. Ad esempio: L'ID di Type1 è un nome di glifo, istanza di ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. L'id di TTF è un int index, istanza di ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) classe.

```csharp
public GlyphId UnicodeToGid(uint unicode)
```

| Parametro | Tipo | Descrizione |

Identificatore del glifo relativo a unicode passato.
