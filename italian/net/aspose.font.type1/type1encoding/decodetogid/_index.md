## Type1Encoding.DecodeToGid method

Decodifica Gid in unicode. Glyph id è un numero univoco per un glifo, che dipende dal tipo di carattere. Ad esempio: L'id di Type1 è un nome di glifo, istanza di ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. L'id di TTF è un int index, istanza di ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) classe.
