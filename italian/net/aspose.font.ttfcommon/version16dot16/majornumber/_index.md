## Version16Dot16.MajorNumber property

Ottiene o imposta il numero di versione principale. Il valore ha senso solo nella notazione esademica, ad esempio versione 0.5 per 'maxp' nei file di font effettivi è 4 byte: {0, 0, 80, 0}, cosa ha la rappresentazione esademica 0x00005000. Dopo aver letto questa versione dal file di font , Numeri maggiori e minori per oggetto[`Version16Dot16`](../../version16dot16) saranno rispettivamente 0 e 20480. E questi valori in forma esademica sono 0x0000 e 0x5000.

```csharp
public ushort MajorNumber { get; set; }
```
