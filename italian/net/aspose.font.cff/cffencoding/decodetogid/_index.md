## CffEncoding.DecodeToGid method

Ottiene Gid per charCode passato. Questo metodo progettato per un CFF CIDFonts, dove charCode deve essere un valore CID valido. Glyph id è un numero univoco per un glifo, che dipende dal tipo di carattere. CFF Font glyph id può essere un'istanza di ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) classe o ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) classe.
