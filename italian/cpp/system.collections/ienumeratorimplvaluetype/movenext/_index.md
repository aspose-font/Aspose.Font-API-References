## IEnumeratorImplValueType::MoveNext method


Sposta l'enumeratore al prossimo elemento. Se non è stato referenziato alcun elemento prima, imposta il riferimento al primo elemento disponibile. Se si raggiunge la fine del contenitore, non fa nulla.

```cpp
virtual bool System::Collections::IEnumeratorImplValueType<T>::MoveNext() override
```


### ReturnValue

True se qualche elemento è disponibile dopo la chiamata, false altrimenti.
