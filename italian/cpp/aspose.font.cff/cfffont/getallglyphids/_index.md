## CffFont::GetAllGlyphIds method


Restituisce un array di tutti gli ID glifo disponibili nel [Font](../../../aspose.font/font/). L'ID glifo è un numero unico per un glifo, dipendente dal tipo di font. L'ID glifo del [Font](../../../aspose.font/font/) CFF può essere un'istanza della classe ([GlyphStringId](../)) o della classe ([GlyphUInt32Id](../)).

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Cff::CffFont::GetAllGlyphIds() override
```


### ReturnValue

Tutti gli identificatori dei glifi disponibili nel [Font](../../../aspose.font/font/).
