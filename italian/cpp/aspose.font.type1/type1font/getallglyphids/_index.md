## Type1Font::GetAllGlyphIds method


Restituisce un array di tutti gli ID glifo disponibili nel [Font](../../../aspose.font/font/). L'ID glifo è un numero univoco per un glifo, dipendente dal tipo di font. [Type1](../../)[Font](../../../aspose.font/font/) ID glifo può essere un'istanza della classe ([GlyphStringId](../)) o della classe ([GlyphUInt32Id](../)).
