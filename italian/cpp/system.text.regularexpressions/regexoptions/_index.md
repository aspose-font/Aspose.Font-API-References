## RegexOptions enum


[Regex](../regex/) options.

```cpp
enum class RegexOptions
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | 0 | Comportamento predefinito. |
| Compilato | 1 | Compila l'espressione regolare per le prestazioni. Sempre eseguito per impostazione predefinita. |
| CultureInvariant | 2 | Usa il confronto invariabile rispetto alla cultura. Ignorato. |
| ECMAScript | 4 | Usa la sintassi ECMAScript. Ignorato. |
| ExplicitCapture | 8 | Solo cattura esplicita. Ignorato. |
| IgnoreCase | 16 | Ignora maiuscole/minuscole durante il confronto. |
| IgnorePatternWhitespace | 32 | Ignora gli spazi bianchi nel modello. Non supportato. |
| Multiline | 64 | Considera '^' e '$' come inizio e fine riga, non dell'intera stringa. |
| RightToLeft | 128 | Corrispondenza da destra a sinistra. Non supportato. |
| Singleline | 256 | Fa sì che '.' corrisponda a qualsiasi carattere senza eccezioni (normalmente, i caratteri di nuova riga non sono corrisposti). |
