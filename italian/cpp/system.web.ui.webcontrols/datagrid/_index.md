## DataGrid class


Classe fittizia che rende possibile collegare il codice tradotto con i riferimenti a [DataGrid](./), ma non eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usare questo puntatore per passarlo alle funzioni come argomento.

```cpp
class DataGrid : public System::Object
```
