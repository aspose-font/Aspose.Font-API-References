## Classi

| Classe | Descrizione |
| --- | --- |
| [DataGrid](./datagrid/) | Classe fittizia che rende possibile collegare il codice tradotto con i riferimenti a [DataGrid](./datagrid/), ma senza eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe dovrebbero essere allocati solo usando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
