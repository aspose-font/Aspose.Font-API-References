## UriParser class


Utilizzata per analizzare un nuovo schema URI. Gli oggetti di questa classe devono essere allocati solo mediante la funzione [System::MakeObject()](../makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../smartptr/) e utilizza tale puntatore per passarlo alle funzioni come argomento.

```cpp
class UriParser
```
