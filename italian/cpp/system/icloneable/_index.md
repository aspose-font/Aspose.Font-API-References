## ICloneable class


Definisce un metodo che consente la clonazione di oggetti - creare una copia di un oggetto. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../smartptr/) e utilizzare tale puntatore per passarla alle funzioni come argomento.

```cpp
class ICloneable : public virtual System::Object
```
