## Array::begin() const method


Restituisce un iteratore al primo elemento del contenitore const-qualificato. Se il contenitore è vuoto, l'iteratore restituito sarà uguale a [end()](../end/).

```cpp

```


### ReturnValue

Un iteratore che punta al primo elemento del contenitore const-qualificato.
