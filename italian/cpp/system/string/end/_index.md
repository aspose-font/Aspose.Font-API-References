## String::end method


Restituisce un puntatore alla fine del buffer della stringa reale. Non riassegna mai nulla. Non garantisce che il buffer sia terminato con null.

```cpp
const UChar * System::String::end() const
```


### ReturnValue

Puntatore al primo elemento dopo il buffer della stringa effettiva, terminato o meno con null.
