## Boolean::Parse method


Converte la stringa specificata in un valore di tipo bool.

```cpp
static bool System::Boolean::Parse(const String &value)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| valore | const String\& | La stringa da convertire. |

### ReturnValue

La variabile bool in cui viene inserito il risultato della conversione; il risultato è true se la stringa specificata è uguale a "True" e false se la stringa specificata è uguale a "False".
