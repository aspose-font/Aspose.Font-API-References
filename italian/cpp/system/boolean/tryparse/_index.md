## Boolean::TryParse method


Converte la stringa specificata in un valore di tipo bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| valore | const String\& | La stringa da convertire. |
| risultato | bool\& | Il riferimento a una variabile bool in cui viene inserito il risultato della conversione; il risultato è true se la stringa specificata è uguale a "True" e false se la stringa specificata è uguale a "False". |

### ReturnValue

True se la stringa specificata è uguale sia a "True" sia a "False", altrimenti - false.
