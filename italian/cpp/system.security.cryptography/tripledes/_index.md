## TripleDES class


Classe base dell'algoritmo Triple [Data](../../system.data/) Encryption Standard. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/) function. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this class into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class TripleDES : public System::Security::Cryptography::SymmetricAlgorithm
```
