## HMACSHA1 class


Codice di [Autenticazione](../../system.security.authentication/) dei messaggi basato su hash che utilizza la funzione hash [SHA1](../sha1/). Non implementato. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento.

```cpp
class HMACSHA1 : public System::Security::Cryptography::HashAlgorithm
```
