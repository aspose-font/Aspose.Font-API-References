## HMAC class


Tutte le implementazioni di Hash-based Message [Authentication](../../system.security.authentication/) Code ([HMAC](./)) devono ereditare questa classe astratta. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../../system/makeobject/) function. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this class into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class HMAC : public System::Security::Cryptography::HashAlgorithm
```
