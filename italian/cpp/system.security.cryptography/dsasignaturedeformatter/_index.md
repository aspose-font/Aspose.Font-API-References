## DSASignatureDeformatter class


Utilizzato per verificare le firme [DSA](../dsa/). Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento.

```cpp
class DSASignatureDeformatter : public System::Security::Cryptography::AsymmetricSignatureDeformatter
```
