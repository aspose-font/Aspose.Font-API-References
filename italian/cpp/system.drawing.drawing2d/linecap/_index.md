## LineCap enum


Indica gli stili di cappuccio disponibili con cui un oggetto [Pen](../../system.drawing/pen/) può terminare una linea.

```cpp
enum class LineCap
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Piatto | 0 | Cappuccio di linea piatto. |
| Quadrato | 1 | Cappuccio di linea quadrato. |
| Round | 2 | Cappuccio di linea rotondo. |
| Triangolo | 3 | Cappuccio di linea triangolare. |
| NoAnchor | 16 | Nessuna ancora. |
| SquareAnchor | 17 | Cap di linea con ancora quadrata. |
| RoundAnchor | 18 | Cap di linea con ancora rotonda. |
| DiamondAnchor | 19 | Cap di linea con ancora a diamante. |
| ArrowAnchor | 20 | Cap di linea con ancora a freccia. |
| AnchorMask | 240 | Una maschera usata per verificare se un cap di linea è un cap di ancora. |
| Personalizzato | 255 | Cap personalizzato. |
