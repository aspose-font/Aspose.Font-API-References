## DashCap enum


Specifica il tipo di cappuccio usato alle due estremità di un trattino in una linea tratteggiata.

```cpp
enum class DashCap
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Piatto | 0 | Un cap quadrato che rende quadrate entrambe le estremità di un tratto. |
| Round | 2 | Un cap circolare che arrotonda entrambe le estremità di un tratto. |
| Triangolo | 3 | Un cap triangolare che punta entrambe le estremità di un tratto. |
