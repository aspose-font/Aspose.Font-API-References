## FtpWebResponse class


Classe fittizia che consente di collegare il codice tradotto con riferimenti a [FtpWebResponse](./), ma non lo esegue. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai istanze di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e utilizzare tale puntatore per passarla alle funzioni come argomento.

```cpp
class FtpWebResponse : public System::Net::WebResponse
```
