## CookieVariant enum


Enumera le specifiche dei cookie. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza tale puntatore per passarlo alle funzioni come argomento.

```cpp
enum class CookieVariant
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Sconosciuto | 0 | La specifica del cookie sconosciuta. |
| Plain | 1 | La specifica del cookie plain. |
| Rfc2109 | 2 | La specifica del cookie 'RFC 2109'. |
| Rfc2965 | 3 | La specifica del cookie 'RFC 2965'. |
| Predefinito | n/a | Il valore predefinito è uguale a 'Rfc2109'. |
