## DecompressionMethods enum


Enumera l'algoritmo di compressione-decompressione. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza tale puntatore per passarlo alle funzioni come argomento.

```cpp
enum class DecompressionMethods
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | 0 | L'algoritmo di compressione-decompressione non è utilizzato. |
| GZip | 1 | L'algoritmo di compressione-decompressione gZip. |
| Deflate | 2 | L'algoritmo di compressione-decompressione deflate. |
