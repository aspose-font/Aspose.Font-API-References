## StreamSource::MustCloseAfterUse method


Gli eredi possono impedire la chiusura del flusso. Restituisce true se lo stream source desidera che il flusso venga chiuso dopo l'uso. Altrimenti restituisce false.

```cpp
virtual bool Aspose::Font::Sources::StreamSource::MustCloseAfterUse()
```


### ReturnValue

True se la sorgente dello stream desidera che lo stream venga chiuso dopo l'uso, altrimenti false.
