## Classi

| Classe | Descrizione |
| --- | --- |
| [ByteContentStreamSource](./bytecontentstreamsource/) | Rappresenta una sorgente di flusso basata sul flusso di contenuto. |
| [FileSystemStreamSource](./filesystemstreamsource/) | Rappresenta una sorgente di flusso basata sul file system. |
| [FontDefinition](./fontdefinition/) | Rappresenta la definizione del set di file [Font](../aspose.font/font/). Questa classe contiene campi che non sono correlati ai dati interni del carattere. Questi campi descrivono il posizionamento del carattere e altri dati necessari per caricare il carattere da qualche fonte di caratteri (file, memoria, ecc). |
| [FontFileDefinition](./fontfiledefinition/) | Rappresenta la definizione del file [Font](../aspose.font/font/). |
| [StreamSource](./streamsource/) | Definisce un modo per ottenere un flusso di file quando è necessario. |
