## STDIOStreamWrappingMode enum


Specifica la modalità delle operazioni I/O che i wrapper eseguiranno su stream simili a std::iostreams.

```cpp
enum class STDIOStreamWrappingMode
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Binary | 0 | Una modalità che consente alle operazioni di input di decodificare i dati del flusso di tipo char_type in byte e di codificare i byte in dati di tipo char_type per le operazioni di output. |
| Conversion | 1 | Una modalità che consente alle operazioni di input di convertire i dati del flusso dal tipo char_type al tipo uint8_t e viceversa per le operazioni di output. |
