## Classi

| Classe | Descrizione |
| --- | --- |
| [HttpContext](./httpcontext/) | Classe fittizia che rende possibile collegare il codice tradotto con riferimenti a [HttpContext](./httpcontext/), ma senza eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o con l'operatore new, poiché ciò può causare errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza questo puntatore per passarla alle funzioni come argomento. |
| [HttpResponse](./httpresponse/) | Classe fittizia che rende possibile collegare il codice tradotto con riferimenti a [HttpResponse](./httpresponse/), ma senza eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o con l'operatore new, poiché ciò può causare errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza questo puntatore per passarla alle funzioni come argomento. |
| [HttpUtility](./httputility/) | Classe di servizio che codifica e decodifica parti di URL in e da frammenti di escape esadecimali. |
