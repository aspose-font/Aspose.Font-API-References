## Classi

| Classe | Descrizione |
| --- | --- |
| [IXmlSerializable](./ixmlserializable/) | Fornisce una formattazione personalizzata per la serializzazione e la deserializzazione XML. Gli oggetti di questa classe dovrebbero essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza questo puntatore per passarlo alle funzioni come argomento. |
| [XmlAttributeOverrides](./xmlattributeoverrides/) | Consente di sovrascrivere gli attributi quando [XmlSerializer](./xmlserializer/) viene utilizzato per serializzare o deserializzare un oggetto.
