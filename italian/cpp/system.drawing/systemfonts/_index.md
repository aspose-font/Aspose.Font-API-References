## SystemFonts class


Fornisce un insieme di oggetti [Font](../font/) pre-creati che rappresentano i caratteri utilizzati per visualizzare il testo negli elementi di visualizzazione di [Windows](../../system.windows/). Si tratta di un tipo statico senza servizi di istanza. Non dovresti mai creare istanze di questo tipo in alcun modo.

```cpp
class SystemFonts : public System::Object
```
