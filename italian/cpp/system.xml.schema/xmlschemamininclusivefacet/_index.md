## XmlSchemaMinInclusiveFacet class


Rappresenta l'elemento **minInclusive** da XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere usata per specificare una restrizione sul valore minimo di un elemento simpleType. Il valore dell'elemento deve essere maggiore o uguale al valore dell'elemento **minInclusive**.

```cpp
class XmlSchemaMinInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
