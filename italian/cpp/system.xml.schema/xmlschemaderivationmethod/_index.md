## XmlSchemaDerivationMethod enum


Fornisce diversi metodi per impedire la derivazione.

```cpp
enum class XmlSchemaDerivationMethod
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Vuoto | 0 | Sovrascrivi il metodo di derivazione predefinito per consentire qualsiasi derivazione. |
| Sostituzione | 1 | Si riferisce alle derivazioni tramite **Substitution**. |
| Estensione | 2 | Si riferisce a derivazioni tramite **Extension**. |
| Restrizione | 4 | Si riferisce a derivazioni tramite **Restriction**. |
| List | 8 | Si riferisce a derivazioni tramite **List**. |
| Unione | 16 | Si riferisce a derivazioni tramite **Union**. |
| Tutti | 255 | **#all**. Si riferisce a tutti i metodi di derivazione. |
| None | 256 | Accetta il metodo di derivazione predefinito. |
