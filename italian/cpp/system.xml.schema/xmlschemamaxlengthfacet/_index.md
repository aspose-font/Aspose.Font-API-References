## XmlSchemaMaxLengthFacet class


Rappresenta l'elemento **maxLength** dallo Schema XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere utilizzata per specificare una restrizione sulla lunghezza massima del valore dati di un elemento **simpleType**. La lunghezza deve essere inferiore al valore dell'elemento **maxLength**.

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
