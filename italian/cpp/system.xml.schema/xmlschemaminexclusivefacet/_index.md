## XmlSchemaMinExclusiveFacet class


Rappresenta l'elemento **minExclusive** dello XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere utilizzata per specificare una restrizione sul valore minimo di un elemento **simpleType**. Il valore dell'elemento deve essere maggiore del valore dell'elemento **minExclusive**.

```cpp
class XmlSchemaMinExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
