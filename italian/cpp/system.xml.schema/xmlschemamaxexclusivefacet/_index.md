## XmlSchemaMaxExclusiveFacet class


Rappresenta l'elemento **maxExclusive** dello XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere usata per specificare una restrizione sul valore massimo di un elemento **simpleType**. Il valore dell'elemento deve essere inferiore al valore dell'elemento **maxExclusive**.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
