## XmlSchemaMinLengthFacet class


Rappresenta l'elemento **minLength** da XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere usata per specificare una restrizione sulla lunghezza minima del valore dati di un elemento **simpleType**. La lunghezza deve essere maggiore del valore dell'elemento **minLength**.

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
