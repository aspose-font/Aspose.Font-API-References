## DataObject class


Definisce un meccanismo indipendente dal formato per il trasferimento dei dati. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/) . Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usarlo per passarlo alle funzioni come argomento.

```cpp
class DataObject : public System::Object
```
