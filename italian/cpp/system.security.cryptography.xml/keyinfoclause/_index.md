## KeyInfoClause class


Tutte le implementazioni dei sottoelementi [KeyInfo](../keyinfo/) devono ereditare da questa classe astratta. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/) . Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usarlo per passarlo alle funzioni come argomento.

```cpp
class KeyInfoClause : public System::Object
```
