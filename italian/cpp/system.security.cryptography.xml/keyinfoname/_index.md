## KeyInfoName class


Rappresenta il sottoelemento XMLDSIG 'KeyName' o l'elemento XML encryption '[KeyInfo](../keyinfo/)' . Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò proverà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usa questo puntatore per passarla alle funzioni come argomento.

```cpp
class KeyInfoName : public System::Security::Cryptography::Xml::KeyInfoClause
```
