## Version16Dot16::get_MinorNumber method


Ottiene il numero di versione minore. Il valore ha senso solo nella notazione esadecimale, per esempio la versione 0,5 per 'maxp' nei file di font reali è di 4 byte: {0, 0, 80, 0}, che ha la rappresentazione esadecimale 0x00005000. Dopo aver letto questa versione dal file di font, i numeri maggiore e minore per l'oggetto [Version16Dot16](../) saranno rispettivamente 0 e 20480. E questi valori in forma esadecimale sono 0x0000 e 0x5000.
