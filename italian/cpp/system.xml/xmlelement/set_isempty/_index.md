## XmlElement::set_IsEmpty method


Imposta il formato del tag dell'elemento.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| valore | bool | Il valore da impostare. Quando si imposta questo valore, se impostato su **true**, i figli dell'elemento vengono rimossi e l'elemento viene serializzato nel formato di tag breve. Se impostato su **false**, il valore viene modificato (indipendentemente dal fatto che l'elemento abbia o meno contenuto); se l'elemento è vuoto, viene serializzato nel formato lungo. |
