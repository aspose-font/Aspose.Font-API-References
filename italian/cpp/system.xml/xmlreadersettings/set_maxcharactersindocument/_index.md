## XmlReaderSettings::set_MaxCharactersInDocument method


Imposta un valore che indica il numero massimo consentito di caratteri in un documento XML. Un valore zero (0) significa nessun limite alla dimensione del documento XML. Un valore diverso da zero specifica la dimensione massima, in caratteri.
