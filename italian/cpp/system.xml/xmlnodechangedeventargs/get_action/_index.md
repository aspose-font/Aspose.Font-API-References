### ReturnValue

Uno dei valori dell'enumerazione che descrive l'evento di modifica del nodo. Il valore **[Action](../../../system/action/)** non differenzia quando l'evento si è verificato (prima o dopo). È possibile creare gestori di eventi separati per gestire entrambe le istanze.
