## XmlTextWriter::get_QuoteChar method


Restituisce quale carattere utilizzare per citare i valori degli attributi.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### ReturnValue

Il carattere da utilizzare per racchiudere tra virgolette i valori degli attributi. Deve essere un apice singolo **'** o un apice doppio **\"**. Il valore predefinito è l'apice doppio.
