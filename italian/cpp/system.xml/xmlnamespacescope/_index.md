## XmlNamespaceScope enum


Definisce l'ambito del namespace.

```cpp
enum class XmlNamespaceScope
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Tutti | 0 | Tutti i namespace definiti nell'ambito del nodo corrente. Include il namespace xmlns:xml che è sempre dichiarato implicitamente. L'ordine dei namespace restituiti non è definito. |
| ExcludeXml | 1 | Tutti i namespace definiti nell'ambito del nodo corrente, escluso il namespace xmlns:xml, che è sempre dichiarato implicitamente. L'ordine dei namespace restituiti non è definito. |
| Local | 2 | Tutti i namespace definiti localmente nel nodo corrente. |
