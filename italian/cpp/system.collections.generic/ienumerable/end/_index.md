## IEnumerable::end() method


Ottiene un iteratore che punta subito dopo l'ultimo elemento (se presente) della collezione. Questo iteratore non può essere usato per modificare un oggetto di riferimento perché [GetEnumerator()](../getenumerator/) restituisce un oggetto copia di T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```


### ReturnValue

Un iteratore che punta subito dopo l'ultimo elemento (se presente) della collezione
