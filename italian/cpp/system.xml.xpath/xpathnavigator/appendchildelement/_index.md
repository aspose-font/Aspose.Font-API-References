## XPathNavigator::AppendChildElement method


Crea un nuovo nodo elemento figlio alla fine dell'elenco dei nodi figlio del nodo corrente utilizzando il prefisso di namespace, il nome locale e l'URI di namespace specificati con il valore indicato.

```cpp
virtual void System::Xml::XPath::XPathNavigator::AppendChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| prefisso | Stringa | Il prefisso di namespace del nuovo nodo elemento figlio (se presente). |
| localName | Stringa | Il nome locale del nuovo nodo elemento figlio (se presente). |
| namespaceURI | String | L'URI di namespace del nuovo nodo elemento figlio (se presente). [String::Empty](../../../system/string/empty/) e **nullptr** sono equivalenti. |
| value | String | Il valore del nuovo nodo elemento figlio. Se viene passato [String::Empty](../../../system/string/empty/) o **nullptr**, viene creato un elemento vuoto. |
