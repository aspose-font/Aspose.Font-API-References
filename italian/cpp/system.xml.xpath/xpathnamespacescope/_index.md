## XPathNamespaceScope enum


Definisce l'ambito del namespace.

```cpp
enum class XPathNamespaceScope
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Tutti | 0 | Restituisce tutti i namespace definiti nell'ambito del nodo corrente. Include il namespace **xmlns:xml**, che è sempre dichiarato implicitamente. L'ordine dei namespace restituiti non è definito. |
| ExcludeXml | 1 | Restituisce tutti i namespace definiti nell'ambito del nodo corrente, escludendo il namespace **xmlns:xml**. Il namespace **xmlns:xml** è sempre dichiarato implicitamente. L'ordine dei namespace restituiti non è definito. |
| Local | 2 | Restituisce tutti i namespace definiti localmente nel nodo corrente. |
