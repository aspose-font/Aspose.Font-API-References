## Classi

| --- | --- |
| [CmsSigner](./cmssigner/) | Fornisce un'API per firmare oggetti usando CMS. Non firma gli oggetti da solo, usa la classe SignedCMS per farlo. Non implementato. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione.
