## RangeGaspBehaviorFlags enum


Flag che descrivono il comportamento desiderato del rasterizzatore.

```cpp
enum class RangeGaspBehaviorFlags : uint16_t
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| nessuno | n/a | Opzionale per dimensioni molto grandi, tipicamente ppem>2048. |
| GASP_GRIDFIT | n/a | Usa il grid fitting. |
| GASP_DOGRAY | n/a | Usa il rendering in scala di grigi. |
| GASP_SYMMETRIC_GRIDFIT | n/a | Usa il grid fitting con smussatura simmetrica ClearType. Supportato solo nella versione 1 'gasp'. |
| GASP_SYMMETRIC_SMOOTHING | n/a | Usa la smussatura lungo più assi con ClearType®. Supportato solo nella versione 1 'gasp'. |
| Riservato | n/a | Flag riservati — impostati a 0. |
