Die **Aspose.Font.Sources**Namespace bietet Funktionen für den Zugriff auf Schriftdaten aus verschiedenen Quellen (Dateien, Speicherstreams usw.).

| Klasse | Beschreibung |

| [FileSystemStreamSource](./filesystemstreamsource) | Repräsentiert eine Stream-Quelle basierend auf dem Dateisystem. |
| [FontDefinition](./fontdefinition) | Repräsentiert die Definition des Schriftdateisatzes. Diese Klasse enthält Felder, die sich nicht auf interne Schriftartdaten beziehen.
