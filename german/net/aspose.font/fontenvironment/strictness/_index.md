## FontEnvironment.Strictness property

Einige Schriftarten können unerwartete Daten oder nicht spezifizierte Merkmale enthalten oder grob beschnitten sein. Eine tolerante Einstellung wird für Benutzer empfohlen, die jede Schriftart unabhängig von der möglichen Unzulänglichkeit der Schriftart öffnen möchten. Es kann nicht garantiert werden, dass interne Schriftstrukturen konsistent sind. Eine strenge Einstellung wird für Verbraucher empfohlen, die möglichst gültige und solide Schriften öffnen möchten.
