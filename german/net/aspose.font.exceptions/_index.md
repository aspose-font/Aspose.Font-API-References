Die **Aspose.Font.Exceptions** namespace stellt Klassen bereit, die Ausnahmen beschreiben, die in Aspose.Font verwendet werden.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [EncodingException](./encodingexception) | Stellt eine Codierungsausnahme dar. Die Ausnahme kann ausgelöst werden, falls ein Problem mit der Codierung/Decodierung von Text besteht. |
| [FontAgrumentException](./fontagrumentexception) | Repräsentiert eine Font-Argument-Ausnahme. Die Ausnahme kann bei unsachgemäßer Verwendung von Agrumen ausgelöst werden. |
| [FontCreationException](./fontcreationexception) | Stellt eine Ausnahme bei der Schriftarterstellung dar. Die Ausnahme kann bei Fehlern während der Schriftarterstellung ausgelöst werden. |
| [FontException](./fontexception) | Stellt eine häufige Ausnahme bei der Verarbeitung von Schriftarten dar. |
| [FontNotSupportedOperationException](./fontnotsupportedoperationexception) | Stellt eine Ausnahme für nicht unterstützte Operationen dar. Die Ausnahme kann ausgelöst werden, falls einige Operationen für einen bestimmten Schriftarttyp nicht unterstützt werden. |
