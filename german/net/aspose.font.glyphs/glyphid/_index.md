## GlyphId class

Stellt Glyphen-IDs dar, die in der Schriftart verfügbar sind. Die Glyphen-ID ist eine eindeutige Nummer für eine Glyphe, die vom Schriftarttyp abhängig ist. Zum Beispiel: Die ID von Typ1 ist ein Glyphenname, Instanz von ([`GlyphStringId`](../glyphstringid)) class. TTF-ID ist ein int-Index, Instanz von ([`GlyphUInt32Id`](../glyphuint32id) Klasse.
