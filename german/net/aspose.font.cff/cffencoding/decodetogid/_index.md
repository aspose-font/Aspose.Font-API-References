## CffEncoding.DecodeToGid method

Ruft Gid für übergebenen charCode ab. Diese Methode wurde für eine CFF-CIDFonts entwickelt, wobei charCode ein gültiger CID-Wert sein muss. Die Glyph-ID ist eine eindeutige Nummer für eine Glyphe, die vom Schriftarttyp abhängig ist. Die Glyph-ID der CFF-Schriftart kann eine Instanz von sein ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) Klasse oder ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) Klasse.
