## CffEncoding.UnicodeToGid method

Dekodiert einen Unicode und gibt die Glyphen-ID zurück. Die Glyphen-ID ist eine eindeutige Nummer für eine Glyphe, die vom Schriftarttyp abhängig ist. Die Glyphen-ID der CFF-Schriftart kann eine Instanz von sein ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) Klasse oder ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) Klasse.
