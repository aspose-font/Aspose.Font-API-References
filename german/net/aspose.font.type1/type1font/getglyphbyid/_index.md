## GetGlyphById(GlyphId) {#getglyphbyid}

Gibt Glyphe nach Glyphen-ID zurück. Die Glyphen-ID ist eine eindeutige Nummer für eine Glyphe, die vom Schriftarttyp abhängig ist. Typ1 Schriftart-Glyphen-ID kann eine Instanz von sein ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) Klasse oder ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) Klasse.
