## IGlyphRenderer.RenderIndependentGlyphPath method

Rendert Glyphe mit unabhängigem Glyphenpfad. Die Funktionsfamilie RenderGlyph() ändert den Glyphenpfad beim Rendern. Dies führt dann dazu, dass diese Glyphe erneut geladen werden muss. Diese Funktion verwendet eine Kopie des Glyphenpfads und ändert nicht den ursprünglichen Glyphenpfad, sodass dieselbe Glyphe mehrmals wiederverwendet werden kann. Diese Version der Funktion ist für die Verwendung mit dem Cache von Glyphen bestimmt.

```csharp
public void RenderIndependentGlyphPath(IFont font, GlyphId glyphId, Glyph glyph, 
    TransformationMatrix glyphPlacementMatrix)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| font | IFont | Die Schriftart, die die Glyphe enthält. |
| glyphId | GlyphId | Physischer Glyphenindex innerhalb der Schriftart. Beachten Sie, dass dies kein Unicode ist. |
| glyph | Glyph | Zu rendernde Glyphe. |
| glyphPlacementMatrix | TransformationMatrix | Matrix, die auf Glyphenkoordinaten angewendet wird. |
