## TtfEncoding.DecodeToGid method

Die DecodeToGlyphId-Implementierung von TTF Font findet eine Unicode-Tabelle und gibt die Glyph-ID für ein Unicode-Zeichen zurück. Die Glyph-ID ist eine eindeutige Nummer für ein Glyph, die vom Schriftarttyp abhängig ist. Zum Beispiel: Die ID von Typ1 ist ein Glyphname, Instanz von ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. TTF-ID ist ein int-Index, Instanz von ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) Klasse.

public GlyphId DecodeToGid(uint unicode)
