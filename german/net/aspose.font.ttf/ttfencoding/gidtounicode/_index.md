## TtfEncoding.GidToUnicode method

Dekodiert die Glyphen-ID in Unicode. Die Glyphen-ID ist eine eindeutige Nummer für eine Glyphe, die vom Schriftarttyp abhängig ist. Zum Beispiel: Die ID von Typ1 ist ein Glyphenname, Instanz von ([`GlyphStringId`](../../../aspose.font.glyphs/glyphstringid)) class. TTF-ID ist ein int-Index, Instanz von ([`GlyphUInt32Id`](../../../aspose.font.glyphs/glyphuint32id) Klasse.
