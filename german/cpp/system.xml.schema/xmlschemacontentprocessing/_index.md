## XmlSchemaContentProcessing enum


Stellt Informationen über den Validierungsmodus von **any**- und **anyAttribute**-Elementersetzungen bereit.

```cpp
enum class XmlSchemaContentProcessing
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Keine | 0 | Dokumentelemente werden nicht validiert. |
| Skip | 1 | Dokumentelemente müssen wohlgeformtes XML enthalten und werden vom Schema nicht validiert. |
| Lax | 2 | Wenn das zugehörige Schema gefunden wird, werden die Dokumentelemente validiert. Andernfalls werden keine Fehler ausgelöst. |
| Strict | 3 | Der Schema-Prozessor muss ein dem angegebenen Namensraum zugeordnetes Schema finden, um die Dokumentelemente zu validieren. |
