## LineCap enum


Gibt die verfügbaren Endstilarten an, mit denen ein [Pen](../../system.drawing/pen/) Objekt eine Linie beenden kann.

```cpp
enum class LineCap
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Flach | 0 | Flacher Linienabschluss. |
| Square | 1 | Quadratischer Linienabschluss. |
| Round | 2 | Runder Linienabschluss. |
| Dreieck | 3 | Dreieckiger Linienabschluss. |
| NoAnchor | 16 | Kein Anker. |
| SquareAnchor | 17 | Quadratischer Anker-Linienabschluss. |
| RoundAnchor | 18 | Runder Anker-Linienabschluss. |
| DiamondAnchor | 19 | Rautenförmiger Anker-Linienabschluss. |
| ArrowAnchor | 20 | Pfeil-Anker-Linienabschluss. |
| AnchorMask | 240 | Eine Maske, die verwendet wird, um zu prüfen, ob ein Linienabschluss ein Ankerabschluss ist. |
| Custom | 255 | Benutzerdefinierter Abschluss. |
