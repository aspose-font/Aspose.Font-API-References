## SmoothingMode enum

```cpp
enum class SmoothingMode
```

| Name | Wert | Beschreibung |
| --- | --- | --- |

| Standard | n/a | Standardmodus, kein Antialiasing. |

| HighQuality | n/a | Antialiasing-Rendering, niedrige Geschwindigkeit. |

| AntiAlias | n/a | Antialiasing-Rendering. |
