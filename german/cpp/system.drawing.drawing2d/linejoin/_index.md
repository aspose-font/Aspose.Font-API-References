## LineJoin enum


Gibt an, wie aufeinanderfolgende Linien oder Kurven verbunden werden.

```cpp
enum class LineJoin
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Miter | 0 | Miter-Verbindung, die je nach Länge des Miter das Miter-Limit überschreitet, entweder eine scharfe oder abgeschnittene Ecke erzeugt. |
| Bevel | 1 | Bevel-Verbindung, die eine diagonale Ecke erzeugt. |
| Round | 2 | Runde Verbindung, die einen glatten kreisförmigen Bogen zwischen den Linien erzeugt. |
| MiterClipped | 3 | Miter-Verbindung, die je nach Länge des Miter das Miter-Limit überschreitet, entweder eine scharfe oder abgeschrägte Ecke erzeugt. |
