## IGlyphAccessor::GetAllGlyphIds method


Gibt alle glyph ids zurück, die im [Font](../../../aspose.font/font/) verfügbar sind. [Glyph](../../glyph/) id ist eine eindeutige Nummer für ein Glyph, das vom Font‑Typ abhängig ist. Zum Beispiel: [Type1](../../../aspose.font.type1/)'s id ist ein Glyph‑Name, Instanz der ([GlyphStringId](../../glyphstringid/)) Klasse. TTF's id ist ein int‑Index, Instanz der ([GlyphUInt32Id](../../glyphuint32id/)) Klasse.

```cpp
virtual System::ArrayPtr<System::SharedPtr<GlyphId>> Aspose::Font::Glyphs::IGlyphAccessor::GetAllGlyphIds()=0
```


### ReturnValue

[Glyph](../../glyph/) identifiers.
