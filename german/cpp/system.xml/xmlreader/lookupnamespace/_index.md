## XmlReader::LookupNamespace method


Wird in einer abgeleiteten Klasse überschrieben, löst es ein Namensraum‑Präfix im Geltungsbereich des aktuellen Elements auf.

```cpp
virtual String System::Xml::XmlReader::LookupNamespace(const String &prefix)=0
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Präfix | const String\& | Das Präfix, dessen Namespace-URI Sie auflösen möchten. Um den Standardsnamespace zu verwenden, übergeben Sie eine leere Zeichenfolge. |

### ReturnValue

Der Namespace‑URI, auf den das Präfix abbildet, oder **nullptr**, wenn kein passendes Präfix gefunden wird.
