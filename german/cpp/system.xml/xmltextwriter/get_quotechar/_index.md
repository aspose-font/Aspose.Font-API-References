## XmlTextWriter::get_QuoteChar method


Gibt zurück, welches Zeichen zum Zitieren von Attributwerten verwendet werden soll.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### ReturnValue

Das Zeichen, das zum Anführen von Attributwerten verwendet wird. Es muss ein einfaches Anführungszeichen **'** oder ein doppeltes Anführungszeichen **\"** sein. Standardmäßig ist ein doppeltes Anführungszeichen eingestellt.
