## NamespaceHandling enum


Gibt an, ob doppelte Namespace‑Deklarationen im [XmlWriter](../xmlwriter/) entfernt werden sollen.

```cpp
enum class NamespaceHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Standard | 0 | Gibt an, dass doppelte Namespace‑Deklarationen nicht entfernt werden. |
| OmitDuplicates | 1 | Gibt an, dass doppelte Namespace‑Deklarationen entfernt werden. Damit ein doppelter Namespace entfernt wird, müssen Präfix und Namespace übereinstimmen. |
