## NewLineHandling enum


Gibt an, wie Zeilenumbrüche behandelt werden.

```cpp
enum class NewLineHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Replace | 0 | Zeilenumbruchzeichen werden ersetzt, um dem in [XmlWriterSettings::set_NewLineChars](../xmlwritersettings/set_newlinechars/) angegebenen Zeichen zu entsprechen. |
| Entitize | 1 | Zeilenumbruchzeichen werden entifiziert. Diese Einstellung bewahrt alle Zeichen, wenn die Ausgabe von einem normalisierenden [XmlReader](../xmlreader/) gelesen wird. |
| Keine | 2 | Die Zeilenumbruchzeichen bleiben unverändert. Die Ausgabe ist identisch mit der Eingabe. |
