## XmlWriter::get_XmlSpace method


Wenn in einer abgeleiteten Klasse überschrieben, erhält sie ein [XmlSpace](../../xmlspace/), das den aktuellen **xml:space**‑Geltungsbereich darstellt.

```cpp
virtual System::Xml::XmlSpace System::Xml::XmlWriter::get_XmlSpace()
```


### ReturnValue

Ein [XmlSpace](../../xmlspace/), das den aktuellen **xml:space**‑Geltungsbereich darstellt.

|||
|-|-|
| Wert | Bedeutung |
| Keine | Dies ist der Standard, wenn kein xml:space‑Bereich existiert. |
| Standard | Der aktuelle Bereich ist xml:space=\"default\". |
| Preserve | Der aktuelle Bereich ist xml:space=\"preserve\". |
