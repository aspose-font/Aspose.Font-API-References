## XmlWriter::WriteAttributes method


Wenn in einer abgeleiteten Klasse überschrieben, schreibt alle Attribute, die an der aktuellen Position im [XmlReader](../../xmlreader/) gefunden wurden, aus.

```cpp
virtual void System::Xml::XmlWriter::WriteAttributes(SharedPtr<XmlReader> reader, bool defattr)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| reader | SharedPtr\<XmlReader\> | Der [XmlReader](../../xmlreader/), von dem die Attribute kopiert werden sollen. |
| defattr | bool | **true**, um die Standardattribute aus dem [XmlReader](../../xmlreader/) zu kopieren; andernfalls **false**. |
