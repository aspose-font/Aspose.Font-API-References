

| Parameter | Typ | Beschreibung |

| ns | String | Die Namespace-URI, deren Präfix Sie finden möchten. |

### ReturnValue

Das passende Präfix oder **nullptr**, falls im aktuellen Gültigkeitsbereich kein passender Namespace‑URI gefunden wird.
