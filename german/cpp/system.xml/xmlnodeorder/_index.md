## XmlNodeOrder enum


Beschreibt die Dokumentreihenfolge eines Knotens im Vergleich zu einem zweiten Knoten.

```cpp
enum class XmlNodeOrder
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Vorher | 0 | Der aktuelle Knoten dieses Navigators befindet sich vor dem aktuellen Knoten des übergebenen Navigators. |
| Nachher | 1 | Der aktuelle Knoten dieses Navigators befindet sich nach dem aktuellen Knoten des übergebenen Navigators. |
| Gleich | 2 | Die beiden Navigatoren sind auf demselben Knoten positioniert. |
| Unbekannt | 3 | Die Knotenpositionen können in der Dokumentreihenfolge relativ zueinander nicht bestimmt werden. Dies kann auftreten, wenn die beiden Knoten in unterschiedlichen Bäumen liegen. |
