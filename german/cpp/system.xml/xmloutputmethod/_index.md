## XmlOutputMethod enum


Gibt die Methode an, die zum Serialisieren der [XmlWriter](../xmlwriter/)-Ausgabe verwendet wird.

```cpp
enum class XmlOutputMethod
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Xml | 0 | Serialisiert gemäß den XML‑1.0‑Regeln. |
| Html | 1 | Serialisiert gemäß den von XSLT festgelegten HTML‑Regeln. |
| Text | 2 | Serialisiert nur Textblöcke. |
| AutoDetect | 3 | Verwenden Sie die XSLT‑Regeln, um zur Laufzeit zwischen den Ausgabemethoden [XmlOutputMethod::Xml](./) und [XmlOutputMethod::Html](./) zu wählen. |
