## XmlDeclaration::get_Standalone method


Gibt den Wert des Attributs standalone zurück.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Gültige Werte sind **yes**, wenn alle für das XML‑Dokument erforderlichen Entity‑Deklarationen im Dokument enthalten sind, oder **no**, wenn eine externe Document Type Definition (DTD) erforderlich ist. Wenn im XML‑Deklaration kein standalone‑Attribut vorhanden ist, gibt diese Methode [String::Empty](../../../system/string/empty/) zurück.
