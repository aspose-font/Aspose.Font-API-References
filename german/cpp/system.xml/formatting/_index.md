## Formatting enum


Gibt Formatierungsoptionen für den [XmlTextWriter](../xmltextwriter/) an.

```cpp
enum class Formatting
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Keine | 0 | Es wird keine spezielle Formatierung angewendet. Dies ist die Vorgabe. |
| Indented | 1 | Führt dazu, dass Kind-Elemente gemäß den Einstellungen von [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) und [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/) eingerückt werden. |
