## XmlWriterSettings::get_IndentChars method


Gibt die Zeichenkette zurück, die beim Einrücken verwendet wird. Diese Einstellung wird verwendet, wenn der Wert [XmlWriterSettings::set_Indent](../set_indent/) auf **true** gesetzt ist.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Die Zeichenkette, die beim Einrücken verwendet wird. Sie kann auf jeden Zeichenfolgenwert gesetzt werden. Um jedoch gültiges XML sicherzustellen, sollten Sie nur zulässige Leerzeichen‑Zeichen angeben, wie Leerzeichen, Tabulatoren, Wagenrückläufe oder Zeilenumbrüche. Der Standardwert sind zwei Leerzeichen.
