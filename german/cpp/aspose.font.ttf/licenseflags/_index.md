## LicenseFlags class


Stellt einen Hilfs-Wrapper für Einbettungsflags aus der 'OS/2'-Tabelle (Feld fsType) dar.

```cpp
class LicenseFlags : public System::Object
```

## Methoden

| Methode | Beschreibung |
| --- | --- |
| [get_FSType](./get_fstype/)() const | Gibt den rohen fsType-Wert aus der OS/2-Tabelle zurück oder 0, falls dieses Feld in der Tabelle nicht existiert. |
| [get_FSTypeAbsent](./get_fstypeabsent/)() const | Gibt true zurück, wenn das fsType-Flag in der 'OS/2'-Tabelle fehlt. |
| [get_IsBitmapOnlyEmbedding](./get_isbitmaponlyembedding/)() | Erkennt, ob fsType ein reines Bitmap-Einbetten erlaubt. |
| [get_IsEditableEmbedding](./get_iseditableembedding/)() | Ermittelt, ob fsType ein editierbares Einbetten zulässt. |
| [get_IsInstallableEmbedding](./get_isinstallableembedding/)() | Ermittelt, ob fsType ein installierbares Einbetten zulässt. |
| [get_IsNoSubsettingEmbedding](./get_isnosubsettingembedding/)() | Ermittelt, ob fsType ein NoSubsetting‑Einbetten zulässt. |
| [get_IsPreviewAndPrintEmbedding](./get_ispreviewandprintembedding/)() | Ermittelt, ob fsType ein Vorschau‑ und Druck‑Einbetten zulässt. |
| [get_IsReservedType](./get_isreservedtype/)() | Ermittelt, ob das reservierte Bit (Bit 0) für fsType gesetzt ist. |
| [get_IsRestrictedLicenseEmbedding](./get_isrestrictedlicenseembedding/)() | Ermittelt, ob fsType ein eingeschränktes [License](../../aspose.font/license/)‑Einbetten ist. |
