## TtfFont::GetAllGlyphIds method


Gibt ein Array aller Glyph‑IDs zurück, die im [Font](../../../aspose.font/font/) verfügbar sind. Eine Glyph‑ID ist eine eindeutige Nummer für ein Glyph, die vom Font‑Typ abhängt. Eine TTF‑[Font](../../../aspose.font/font/) Glyph‑ID kann eine Instanz der ([GlyphStringId](../))‑Klasse oder der ([GlyphUInt32Id](../))‑Klasse sein. Die Glyph‑Adressierung nach Name (String) wird für TTF‑Fonts über die Post‑Tabellenzuordnung unterstützt. Im Falle eines CFF‑[Font](../../../aspose.font/font/) wird die CFF‑Struktur verwendet, um Glyphs per Name anzusprechen.
