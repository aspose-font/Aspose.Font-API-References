## XPathNavigator::get_LocalName method


Wenn sie in einer abgeleiteten Klasse überschrieben wird, ruft sie das [XPathNavigator::get_Name](../get_name/) des aktuellen Knotens ohne Namensraumpräfix ab.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_LocalName()=0
```


### ReturnValue

Ein [String](../../../system/string/), der den lokalen Namen des aktuellen Knotens enthält, oder [String::Empty](../../../system/string/empty/), falls der aktuelle Knoten keinen Namen hat (z. B. Text‑ oder Kommentarknoten).
