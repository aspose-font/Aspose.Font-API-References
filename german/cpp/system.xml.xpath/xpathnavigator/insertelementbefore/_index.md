## XPathNavigator::InsertElementBefore method


Erstellt ein neues Geschwisterelement vor dem aktuellen Knoten mithilfe des angegebenen Namensraumpräfixes, des lokalen Namens und der angegebenen Namensraum-URI, mit dem angegebenen Wert.

```cpp
virtual void System::Xml::XPath::XPathNavigator::InsertElementBefore(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Präfix | String | Das Namespace-Präfix des neuen untergeordneten Elements (falls vorhanden). |
| localName | String | Der lokale Name des neuen untergeordneten Elements (falls vorhanden). |
| namespaceURI | String | Die Namespace-URI des neuen untergeordneten Elements (falls vorhanden). [String::Empty](../../../system/string/empty/) und **nullptr** sind äquivalent. |
| value | String | Der Wert des neuen untergeordneten Elements. Wenn [String::Empty](../../../system/string/empty/) oder **nullptr** übergeben werden, wird ein leeres Element erstellt. |
