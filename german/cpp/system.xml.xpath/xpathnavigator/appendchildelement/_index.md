## XPathNavigator::AppendChildElement method


Erstellt einen neuen Kind-Elementknoten am Ende der Kindknotenliste des aktuellen Knotens unter Verwendung des angegebenen Namensraumpräfixes, des lokalen Namens und der angegebenen Namensraum‑URI mit dem angegebenen Wert.

```cpp
virtual void System::Xml::XPath::XPathNavigator::AppendChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Präfix | String | Das Namespace‑Präfix des neuen Kind‑Element‑Knotens (falls vorhanden). |
| localName | String | Der lokale Name des neuen Kind‑Element‑Knotens (falls vorhanden). |
| namespaceURI | String | Die Namespace‑URI des neuen Kind‑Element‑Knotens (falls vorhanden). [String::Empty](../../../system/string/empty/) und **nullptr** sind äquivalent. |
| value | String | Der Wert des neuen Kind‑Element‑Knotens. Wenn [String::Empty](../../../system/string/empty/) oder **nullptr** übergeben werden, wird ein leeres Element erstellt. |
