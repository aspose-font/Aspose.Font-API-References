## RegexOptions enum


[Regex](../regex/) options.

```cpp
enum class RegexOptions
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Keine | 0 | Standardverhalten. |
| Kompiliert | 1 | Regex für Leistung kompilieren. Wird standardmäßig immer durchgeführt. |
| CultureInvariant | 2 | Verwende kulturinvariante Übereinstimmung. Ignoriert. |
| ECMAScript | 4 | Verwenden Sie ECMAScript-Syntax. Ignoriert. |
| ExplicitCapture | 8 | Nur explizite Erfassung. Ignoriert. |
| IgnoreCase | 16 | Groß-/Kleinschreibung beim Abgleichen ignorieren. |
| IgnorePatternWhitespace | 32 | Leerzeichen im Muster ignorieren. Nicht unterstützt. |
| Multiline | 64 | Behandle '^' und '$' als Anfang bzw. Ende der Zeile, nicht des gesamten Strings. |
| RightToLeft | 128 | Rechts-nach-links-Abgleich. Nicht unterstützt. |
| Singleline | 256 | Lässt '.' jedes Zeichen ohne Ausnahmen matchen (normalerweise werden Zeilenumbrüche nicht gematcht). |
