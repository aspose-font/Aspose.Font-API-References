## CffFont::GetAllGlyphIds method


Gibt ein Array aller Glyph‑IDs zurück, die im [Font](../../../aspose.font/font/) verfügbar sind. Eine Glyph‑ID ist eine eindeutige Nummer für ein Glyph, die vom Schriftarttyp abhängt. Eine CFF‑[Font](../../../aspose.font/font/) Glyph‑ID kann eine Instanz der Klasse ([GlyphStringId](../)) oder der Klasse ([GlyphUInt32Id](../)) sein.

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Cff::CffFont::GetAllGlyphIds() override
```


### ReturnValue
