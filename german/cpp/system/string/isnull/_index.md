## String::IsNull method


Prüft, ob der String als null betrachtet wird. [String](../) ist null, und zwar nur, wenn er über den [String()](../string/) Konstruktor erstellt, verschoben, kopiert oder von einem null-String zugewiesen wurde oder die [reset()](../reset/) Methode aufgerufen wurde.

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true, wenn der String als null betrachtet wird, sonst false.
