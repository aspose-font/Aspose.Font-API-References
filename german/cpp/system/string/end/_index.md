## String::end method


Gibt einen Zeiger auf das Ende des tatsächlichen Zeichenkettenpuffers zurück. Allokiert nie neu. Garantiert nicht, dass der Puffer nullterminiert ist.

```cpp
const UChar * System::String::end() const
```

### ReturnValue

Zeiger auf das erste Element nach dem tatsächlichen Zeichenpuffer, nullterminiert oder nicht.
