## SmartPtr::RemoveAliasing method


Entfernt das Alias (erstellt durch einen Alias‑Konstruktor) vom Zeiger und stellt sicher, dass er (bei Shared) verwaltet oder (bei Weak) verfolgt dasselbe Objekt, auf das er zeigt.

```cpp
SmartPtr_ System::SmartPtr<T>::RemoveAliasing() const
```


### ReturnValue

Der Zeiger auf dasselbe Objekt, auf das dieser Zeiger zeigt und der die Lebensdauer desselben Objekts überwacht.
