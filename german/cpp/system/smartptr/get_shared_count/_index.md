## SmartPtr::get_shared_count method


Liefert die Anzahl der vorhandenen Shared-Zeiger auf das referenzierte Objekt, einschließlich des aktuellen. Prüft, dass der aktuelle Zeiger im Shared-Modus ist.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Anzahl der vorhandenen Shared-Pointer auf das referenzierte Objekt, falls vorhanden. Wenn der Zeiger null ist, wird 0 zurückgegeben.
