## DateTime::CompareTo method


Vergleicht zwei Datums- und Zeitwerte, die vom aktuellen Objekt und der angegebenen Instanz der [DateTime](../)-Klasse dargestellt werden, und gibt den Wert zurück, der die relative Position der Werte auf der Zeitlinie anzeigt.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | DateTime | Eine Instanz der [DateTime](../)-Klasse, mit der das aktuelle Objekt verglichen wird |

### ReturnValue

Ein Wert, der kleiner als 0 ist, wenn das aktuelle Objekt einen Wert darstellt, der früher liegt als der von **value** dargestellte; 0, wenn die von beiden Objekten dargestellten Werte gleich sind; ein Wert, der größer als 0 ist, wenn der vom aktuellen Objekt dargestellte Wert später liegt als der von **value** dargestellte.
