## DateTime::Compare method


Vergleicht zwei Werte, die durch die angegebenen Instanzen der [DateTime](../)‑Klasse dargestellt werden, und gibt den Wert zurück, der die relativen Positionen der Werte auf der Zeitleiste angibt.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| t1 | DateTime | Der erste Operand |
| t2 | DateTime | Der zweite Operand |

### ReturnValue

Ein Wert, der kleiner als 0 ist, wenn **t1** früher als **t2** ist; 0, wenn **t1** gleich **t2** ist; ein Wert, der größer als 0 ist, wenn **t1** später als **t2** ist.
