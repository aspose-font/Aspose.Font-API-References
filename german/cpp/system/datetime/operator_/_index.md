## DateTime::operator<(DateTime) const method


Bestimmt, ob das aktuelle Objekt den Datums- und Uhrzeitwert darstellt, der früher liegt als der von dem angegebenen [DateTime](../)-Objekt dargestellte Wert.

```cpp
bool System::DateTime::operator<(DateTime other) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | DateTime | Das [DateTime](../)-Objekt, mit dem das aktuelle Objekt verglichen werden soll |

### ReturnValue

Wahr, wenn der vom aktuellen Objekt dargestellte Datums- und Zeitwert früher liegt als der von **other** dargestellte Wert, sonst - falsch
