## DateTime::Equals(DateTime) const method


Bestimmt, ob die angegebene Instanz der [DateTime](../)-Klasse denselben Datums- und Zeitwert wie das aktuelle Objekt darstellt.

```cpp
bool System::DateTime::Equals(DateTime other) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | DateTime | Die Instanz der [DateTime](../)-Klasse, mit der das aktuelle Objekt verglichen wird |

### ReturnValue

Wahr, wenn sowohl **other** als auch das aktuelle Objekt denselben Wert darstellen, andernfalls – false
