## DateTime::operator<=(DateTime) const method


Bestimmt, ob das aktuelle Objekt den Datums- und Uhrzeitwert darstellt, der früher oder gleich dem von dem angegebenen [DateTime](../)-Objekt dargestellten Wert ist.

```cpp
bool System::DateTime::operator<=(DateTime other) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | DateTime | Das [DateTime](../)-Objekt, mit dem das aktuelle Objekt verglichen werden soll |

### ReturnValue

Wahr, wenn der vom aktuellen Objekt dargestellte Datums‑ und Uhrzeitwert früher liegt als oder dem gleichen Wert entspricht, der von **other** dargestellt wird, andernfalls – falsch
