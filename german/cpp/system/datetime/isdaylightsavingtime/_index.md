## DateTime::IsDaylightSavingTime method

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```


### ReturnValue

Wahr, wenn der vom aktuellen Objekt angegebene Datums‑ und Zeitwert in den Zeitraum der Sommerzeit fällt, andernfalls falsch
