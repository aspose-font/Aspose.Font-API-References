## Array::cend method


Gibt einen Iterator auf das Element zurück, das dem letzten Element des Containers folgt. Dieses Element dient als Platzhalter; ein Zugriff darauf führt zu undefiniertem Verhalten.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```


### ReturnValue

Ein Iterator, der auf das theoretische, const‑qualifizierte Element zeigt, das nach dem letzten Element des Containers platziert ist.
