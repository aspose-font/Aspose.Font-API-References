## ValueType class


Basisklasse für Werttypen mit [Object](../object/) Vererbung, die aus Leistungsgründen abgeschnitten wird. Dieser Typ sollte auf dem Stack alloziert und an Funktionen per Wert oder Referenz übergeben werden. Verwenden Sie niemals die [System::SmartPtr](../smartptr/) Klasse, um Objekte dieses Typs zu verwalten.

```cpp
class ValueType
```
