## Uri::GetComponents method


Gibt die angegebenen Komponenten der vom aktuellen Objekt dargestellten URI unter Verwendung der angegebenen Maskierung zurück.

```cpp
String System::Uri::GetComponents(UriComponents components, UriFormat format) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| components | UriComponents | Eine bitweise Kombination der Werte von [UriComponents](../../uricomponents/), die angibt, welche Teile des URIs zurückgegeben werden sollen |
| Format | UriFormat | Gibt an, wie die Sonderzeichen escaped werden |

### ReturnValue

Die angeforderten Komponenten
