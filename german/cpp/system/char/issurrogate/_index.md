## Char::IsSurrogate(char_t) method


Bestimmt, ob das angegebene Zeichen eine UTF-16 Surrogate-Codeeinheit ist.

```cpp
static bool System::Char::IsSurrogate(char_t c)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | char_t | Ein Zeichen |

### ReturnValue

Wahr, wenn das angegebene Zeichen eine UTF-16-Surrogat-Codeeinheit ist, sonst - falsch
