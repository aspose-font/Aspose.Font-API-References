## Char::IsNumber(char_t) method



| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | char_t | Das zu testende Zeichen |

### ReturnValue

Wahr, wenn das angegebene Zeichen eine Zahl ist, sonst - falsch
