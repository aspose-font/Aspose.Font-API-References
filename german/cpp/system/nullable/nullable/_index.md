## Nullable::Nullable(const Nullable\<T1\>\&) constructor


Konstruiert eine Instanz, die einen Wert darstellt, der durch das angegebene [Nullable](../)-Objekt repräsentiert wird. Das angegebene Nullable-Objekt kann einen Wert eines anderen Typs als den zugrunde liegenden Typ der konstruierten Instanz repräsentieren; in diesem Fall wird der dargestellte Wert in einen Wert des Typs T konvertiert.

```cpp
template<typename T1> System::Nullable<T>::Nullable(const Nullable<T1> &value)
```


| Parameter | Beschreibung |
| --- | --- |
| T1 | Der Typ des von dem angegebenen [Nullable](../)-Objekt dargestellten Wertes |
