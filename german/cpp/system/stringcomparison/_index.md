### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CurrentCulture | 0 | Verwenden Sie die aktuelle Kultur. |
| CurrentCultureIgnoreCase | 1 | Verwenden Sie die aktuelle Kultur, ignorieren Sie jedoch die Groß-/Kleinschreibung. |
| InvariantCulture | 2 | Verwenden Sie die Invariantkultur. |
| InvariantCultureIgnoreCase | 3 | Verwenden Sie die Invariantkultur, aber ignorieren Sie die Groß-/Kleinschreibung. |
| Ordinal | 4 | Verwenden Sie ordinale Sortierregeln. |
| OrdinalIgnoreCase | 5 | Verwenden Sie ordinale Sortierregeln, aber ignorieren Sie die Groß-/Kleinschreibung. |
