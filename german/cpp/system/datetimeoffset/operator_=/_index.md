## DateTimeOffset::operator<=(const DateTimeOffset\&) const method


Bestimmt, ob das aktuelle Objekt den Datums‑ und Uhrzeitwert darstellt, der früher liegt als oder dem gleichen Wert entspricht, der vom angegebenen [DateTimeOffset](../)-Objekt repräsentiert wird.

```cpp
bool System::DateTimeOffset::operator<=(const DateTimeOffset &other) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | const DateTimeOffset\& | Das [DateTimeOffset](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### ReturnValue

Wahr, wenn der vom aktuellen Objekt dargestellte Datums‑ und Uhrzeitwert früher liegt als oder dem gleichen Wert entspricht, der von **other** dargestellt wird, andernfalls – falsch
