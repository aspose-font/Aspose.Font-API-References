## Boolean::Parse method


Konvertiert die angegebene Zeichenkette in einen Wert des bool-Typs.

```cpp
static bool System::Boolean::Parse(const String &value)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Wert | const String\& | Die Zeichenkette zum Konvertieren. |

### ReturnValue

Die bool-Variable, in die das Ergebnis der Konvertierung geschrieben wird; das Ergebnis ist true, wenn die angegebene Zeichenkette gleich \"True\" ist, und false, wenn die angegebene Zeichenkette gleich \"False\" ist.
