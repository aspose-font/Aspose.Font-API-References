## StringFormatFlags enum


Gibt die Anzeige- und Layoutinformationen für Textstrings an.

```cpp
enum class StringFormatFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Richtung von rechts nach links. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) ist vertikal ausgerichtet. |
| FitBlackBox | 4 | Teile von Zeichen können über die Layout-Box hinausragen. |
| DisplayFormatControl | 32 | Steuerzeichen werden angezeigt. |
| NoFontFallback | 1024 | [Font](../font/) Fallback ist deaktiviert. |
| MeasureTrailingSpaces | 2048 | Fügt das nachfolgende Leerzeichen am Ende jeder Zeile ein. |
| NoWrap | 4096 | [Text](../../system.drawing.text/) Zeilenumbruch ist deaktiviert. |
| LineLimit | 8192 | Gesamte Zeilen werden im Formatierungsrechteck angeordnet. |
| NoClip | 16384 | Überstehende Teile von Glyphen dürfen angezeigt werden. |
