## StringDigitSubstitute enum


Gibt an, wie Ziffern in einem String gemäß der Gebietsschema- oder Spracheinstellung ersetzt werden.

```cpp
enum class StringDigitSubstitute
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| User | 0 | Benutzerdefinierte Substitution. |
| Keine | 1 | Subtitution ist deaktiviert. |
| National | 2 | Ersetze Ziffern, die der offiziellen Landessprache der Benutzereinstellung entsprechen. |
| Traditional | 3 | Ersetze Ziffern, die dem nativen Schriftsystem oder der Sprache des Benutzers entsprechen, die möglicherweise nicht mit der offiziellen Landessprache der Benutzereinstellung übereinstimmt. |
