## SystemFonts class


Stellt eine Menge vorgefertigter [Font](../font/)-Objekte bereit, die Schriftarten darstellen, die zur Anzeige von Text in [Windows](../../system.windows/)-Anzeigeelementen verwendet werden. Dies ist ein statischer Typ ohne Instanzdienste. Sie sollten niemals Instanzen davon auf irgendeine Weise erstellen.

```cpp
class SystemFonts : public System::Object
```
