## Color::get_IsNamedColor method


Gibt einen Wert zurück, der bestimmt, ob die [Color](../)-Struktur eine benannte Farbe oder ein Mitglied der [KnownColor](../../knowncolor/)-Aufzählung darstellt.

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```


### ReturnValue

Wahr, wenn die [Color](../) mithilfe der FromName(String)-Methode oder der [FromKnownColor(KnownColor)](../fromknowncolor/)-Methode erstellt wurde; andernfalls wird false zurückgegeben.
