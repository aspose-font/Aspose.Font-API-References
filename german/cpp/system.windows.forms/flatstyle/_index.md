## FlatStyle enum


Definiert, ob das Steuerelement flach oder dreidimensional gezeichnet wird.

```cpp
enum class FlatStyle
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) wird flach gezeichnet. |
| Popup | 1 | [Control](../control/) wird flach gezeichnet, es sei denn, die Maus bewegt sich darüber, wenn es 3D-ähnlich gezeichnet wird. |
| Standard | 2 | [Control](../control/) wird 3D-ähnlich gezeichnet. |
| System | 3 | Der Stil wird vom Betriebssystem gesteuert. |
