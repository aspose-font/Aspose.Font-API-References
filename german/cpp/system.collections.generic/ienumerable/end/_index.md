## IEnumerable::end() method


Gibt einen Iterator zurück, der direkt nach dem letzten Element (falls vorhanden) der Sammlung zeigt. Dieser Iterator kann nicht verwendet werden, um ein referenziertes Objekt zu ändern, weil [GetEnumerator()](../getenumerator/) ein Kopie‑Objekt von T zurückgibt.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```


### ReturnValue

Ein Iterator, der direkt nach dem letzten Element (falls vorhanden) der Sammlung zeigt
