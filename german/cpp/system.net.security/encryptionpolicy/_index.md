## EncryptionPolicy enum


Enumeriert die Verschlüsselungsrichtlinien.

```cpp
enum class EncryptionPolicy
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| RequireEncryption | 0 | Verschlüsselung erfordern und niemals einen 'Null'-Cipher zulassen. |
| AllowNoEncryption | 1 | Vollständige Verschlüsselung bevorzugen, aber ein 'Null'-Cipher kann verwendet werden, wenn der Server zustimmt. |
| NoEncryption | 2 | Erlaube keine Verschlüsselung und fordere an, dass ein 'Null'-Cipher verwendet wird, wenn das andere Endpunkt einen 'Null'-Cipher verarbeiten kann. |
