## X509KeyStorageFlags enum


Definiert, wie der Schlüssel gespeichert wird.

```cpp
enum class X509KeyStorageFlags : int32_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| DefaultKeySet | 0 | Verwende den Standardschlüsselsatz. |
| UserKeySet | 1 | Verwende den benutzerzugeordneten Speicher anstelle des maschinenlokalen. |
| MachineKeySet | 2 | Verwende den lokalen Maschinenspeicher anstelle des Benutzerspeichers. |
| Exportable | 4 | Markiert importierte Schlüssel als exportierbar. |
| UserProtected | 8 | Benachrichtige den Benutzer, dass der Schlüssel verwendet wird. |
| PersistKeySet | 16 | Der Schlüssel wird beim Importieren des Zertifikats gespeichert. |
