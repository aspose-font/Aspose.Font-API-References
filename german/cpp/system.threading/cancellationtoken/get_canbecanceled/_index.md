## CancellationToken::get_CanBeCanceled method


Ermittelt, ob dieses Token in den abgebrochenen Zustand versetzt werden kann.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true, wenn dieses Token in den abgebrochenen Zustand versetzt werden kann; andernfalls false.
## Hinweise



Tokens, die von [CancellationTokenSource](../../cancellationtokensource/) erstellt wurden, geben true zurück, während das None-Token immer false zurückgibt.
