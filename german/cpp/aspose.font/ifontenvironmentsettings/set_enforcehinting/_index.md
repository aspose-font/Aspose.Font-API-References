## IFontEnvironmentSettings::set_EnforceHinting method


Dieses Flag steuert die Logik der Verwendung von Bytecode-Hinting. Wenn es wahr ist, muss die Schriftart Bytecode-Hinting aktivieren; wenn es falsch ist, wird Bytecode-Hinting je nach Standardlogik ein- oder ausgeschaltet. Standardmäßig false.

```cpp
virtual void Aspose::Font::IFontEnvironmentSettings::set_EnforceHinting(bool value)=0
```
