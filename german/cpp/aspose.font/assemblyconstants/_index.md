## AssemblyConstants class


Definiert die Konstanten, die an der Lizenzprüfung für die Komponente teilnehmen. Diese wurden früher direkt als assembly attributes definiert, aber ich habe sie in eine separate Klasse verschoben, weil ich im .NET Compact Framework nicht auf assembly attributes zugreifen kann. Jetzt verwendet der Lizenzcode, wenn er für das .NET Compact Framework kompiliert wird, diese Konstanten anstelle der assembly attributes.

```cpp
class AssemblyConstants
```
