## Font::GetAllGlyphIds method


Gibt alle Glyph-IDs zurück, die im [Font](../) verfügbar sind. Eine Glyph-ID ist eine eindeutige Nummer für ein Glyph, die vom Schriftarttyp abhängt. Zum Beispiel: Die ID von [Type1](../../../aspose.font.type1/) ist ein Glyph-Name, eine Instanz der Klasse ([GlyphStringId](../)). Die ID von TTF ist ein Integer-Index, eine Instanz der Klasse ([GlyphUInt32Id](../)).

```cpp
System::ArrayPtr<System::SharedPtr<Glyphs::GlyphId>> Aspose::Font::Font::GetAllGlyphIds() override=0
```


### ReturnValue

Glyph‑Bezeichner.
