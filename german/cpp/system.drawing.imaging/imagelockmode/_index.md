## ImageLockMode enum


Gibt Eigenschaften des gesperrten Bildbereichs an.

```cpp
enum class ImageLockMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ReadOnly | 1 | Der Bereich ist nur zum Lesen gesperrt. |
| WriteOnly | 2 | Der Bereich ist nur zum Schreiben gesperrt. |
| ReadWrite | n/a | Der Bereich ist sowohl für das Lesen als auch für das Schreiben gesperrt. |
| UserInputBuffer | 4 | Das Bild befindet sich in einem Benutzer-Eingabepuffer, dessen Zugriff vom Benutzer gesteuert wird. |
