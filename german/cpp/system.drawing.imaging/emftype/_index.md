## EmfType enum


Gibt die Typen der Datensätze an, die in einer EMF-Datei abgelegt werden.

```cpp
enum class EmfType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) Enhanced Metafile. Enthält GDI‑Befehle. Metadateien dieses Typs werden als EMF‑Datei bezeichnet. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) Enhanced Metafile Plus. Enthält GDI+‑Befehle. Metadateien dieses Typs werden als EMF+‑Datei bezeichnet. |
| EmfPlusDual | n/a | Dualer [Windows](../../system.windows/) Enhanced Metafile. Enthält äquivalente GDI‑ und GDI+‑Befehle. Metadateien dieses Typs werden als EMF+‑Datei bezeichnet. |
