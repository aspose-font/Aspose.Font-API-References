## ColorMatrixFlag enum


Gibt die Typen von Bildern und Farben an, die von den Farb- und Graustufeneinstellungen eines [ImageAttributes](../imageattributes/) Objekts betroffen sind.

```cpp
enum class ColorMatrixFlag
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Standard | 0 | Sowohl Farben als auch Graustufenwerte werden farblich angepasst. |
| SkipGrays | 1 | Graustufenwerte werden nicht farblich angepasst. |
| AltGrays | 2 | Nur Graustufenwerte werden farblich angepasst. |
