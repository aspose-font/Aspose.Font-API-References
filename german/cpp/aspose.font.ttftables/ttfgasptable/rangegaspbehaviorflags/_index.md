## RangeGaspBehaviorFlags enum


Flags, die das gewünschte Verhalten des Rasterisierers beschreiben.

```cpp
enum class RangeGaspBehaviorFlags : uint16_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| keine | n/a | Optional für sehr große Größen, typischerweise ppem>2048. |
| GASP_GRIDFIT | n/a | Verwende Grid-Fitting. |
| GASP_DOGRAY | n/a | Verwende Graustufen-Rendering. |
| GASP_SYMMETRIC_GRIDFIT | n/a | Verwende Grid-Fitting mit ClearType symmetrischer Glättung. Nur unterstützt in Version 1 'gasp'. |
| GASP_SYMMETRIC_SMOOTHING | n/a | Verwende Glättung entlang mehrerer Achsen mit ClearType®. Nur unterstützt in Version 1 'gasp'. |
| Reserviert | n/a | Reservierte Flags — auf 0 gesetzt. |
