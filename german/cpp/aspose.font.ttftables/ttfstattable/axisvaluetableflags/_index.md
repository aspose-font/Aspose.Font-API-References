## AxisValueTableFlags enum


Gibt die Flags der Achsenwerttabelle an.

```cpp
enum class AxisValueTableFlags : uint16_t
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| OlderSiblingFontAttribute | n/a | Wenn gesetzt, liefert diese Achsenwerttabelle Achsenwertinformationen, die für andere Schriften derselben Schriftfamilie gelten. Dies wird verwendet, wenn die anderen Schriften früher veröffentlicht wurden und keine Informationen zu Werten für einige Achsen enthalten. Wenn neuere Versionen der anderen Schriften die Informationen selbst enthalten und vorhanden sind, wird diese Tabelle ignoriert. |
| ElidableAxisValueName | n/a | Wenn gesetzt, zeigt dies an, dass der Achsenwert den „normalen“ Wert für die Achse darstellt und beim Erstellen von Namenszeichenketten weggelassen werden kann. |
| Reserviert | n/a | Für zukünftige Verwendung reserviert. |
