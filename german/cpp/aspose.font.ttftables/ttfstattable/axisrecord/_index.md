## AxisRecord class


Stellt die Axis Record-Struktur dar. Spezifikation: Der Achsen-Datensatz liefert Informationen über eine einzelne Design-Achse.

```cpp
class AxisRecord : public System::Object
```

Spezifikation: Das Feld axisNameID liefert eine Namens-ID, die verwendet werden kann, um Zeichenketten aus der 'name'-Tabelle zu erhalten, die zur Bezeichnung der Achse in den Benutzeroberflächen der Anwendung verwendet werden können. |
