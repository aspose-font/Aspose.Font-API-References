## AxisValueTableBase class

Spezifikation: Achsenwerttabellen liefern Details zu einem bestimmten Stil‑Attributwert auf einer bestimmten Achse der Designvariation oder einer Kombination von Design‑Variations‑Achsenwerten sowie die Beziehung dieser Werte zu Bezeichnungen, die als Elemente in Unterschriftennamen verwendet werden.
