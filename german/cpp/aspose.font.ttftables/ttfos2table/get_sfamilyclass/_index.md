## TtfOs2Table::get_SFamilyClass method


Dieser Parameter ist eine Klassifizierung des Schriftfamilien-Designs. Die Schriftklasse und Unterklasse sind von IBM registrierte Werte, die jeder Schriftfamilie zugewiesen werden. Dieser Parameter ist für die Auswahl einer alternativen Schrift gedacht, wenn die gewünschte Schrift nicht verfügbar ist.
