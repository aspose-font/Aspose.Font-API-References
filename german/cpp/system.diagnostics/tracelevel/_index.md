## TraceLevel enum


Gibt an, welche Nachrichten für die Klassen [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) und System.Diagnostics.TraceSwitch ausgegeben werden sollen.

```cpp
enum class TraceLevel
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Off | 0 | Gibt keine Trace‑ und Debug‑Nachrichten aus. |
| Fehler | 1 | Gibt Fehlermeldungen aus. |
| Warning | 2 | Gibt Warnungen und Fehlermeldungen aus. |
| Info | 3 | Gibt Informationsnachrichten, Warnungen und Fehlermeldungen aus. |
| Verbose | 4 | Gibt alle Debug‑ und Trace‑Nachrichten aus. |
