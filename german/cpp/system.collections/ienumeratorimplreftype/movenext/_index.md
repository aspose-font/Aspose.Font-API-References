## IEnumeratorImplRefType::MoveNext method


Bewegt den Enumerator zum nächsten Element. Wenn zuvor kein Element referenziert wurde, wird die Referenz auf das erste verfügbare Element gesetzt. Wird das Ende des Containers erreicht, geschieht nichts.

```cpp
virtual bool System::Collections::IEnumeratorImplRefType<T>::MoveNext() override
```


### ReturnValue

Wahr, wenn nach dem Aufruf ein Element verfügbar ist, andernfalls falsch.
