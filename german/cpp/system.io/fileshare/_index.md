## FileShare enum


Gibt an, welche Art von Zugriff andere [FileStream](../filestream/)‑Objekte auf eine geöffnete Datei haben.

```cpp
enum class FileShare
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Keine | 0 | Kein Zugriff. |
| Read | 1 | Nur-Lesezugriff. |
| Write | 2 | Nur-Schreibzugriff. |
| ReadWrite | 3 | Lese‑ und Schreibzugriff. |
| Löschen | 4 | Die Datei kann gelöscht werden. |
| Vererbbar | 16 | Macht den Dateihandle für Kindprozesse vererbbar. |
