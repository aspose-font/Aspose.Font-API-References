## MemoryStream::TryGetBuffer method


Gibt das Array aus vorzeichenlosen Bytes zurück, aus dem dieser Stream erstellt wurde.

```cpp
bool System::IO::MemoryStream::TryGetBuffer(ArraySegment<uint8_t> &buffer)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| buffer | ArraySegment\<uint8_t\>\& | Byte-Array - Ausgabeparameter. Wenn diese Methode true zurückgibt, das Byte-Array-Segment, aus dem dieser Stream erstellt wurde; wenn sie false zurückgibt, wird dieser Parameter auf den Standardwert gesetzt. |

### ReturnValue

True, wenn die Konvertierung erfolgreich war.
