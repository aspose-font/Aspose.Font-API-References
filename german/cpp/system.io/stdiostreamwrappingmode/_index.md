## STDIOStreamWrappingMode enum


Gibt den Modus der I/O‑Operationen an, die Wrapper auf std::iostream‑ähnlichen Streams ausführen werden.

```cpp
enum class STDIOStreamWrappingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Binary | 0 | Ein Modus, der Eingabeoperationen ermöglicht, Stream-Daten des Typs char_type in Bytes zu dekodieren und Bytes in char_type-Daten für Ausgabeoperationen zu enkodieren. |
| Conversion | 1 | Ein Modus, der Eingabeoperationen ermöglicht, Stream-Daten vom Typ char_type in den Typ uint8_t und umgekehrt für Ausgabeoperationen zu konvertieren. |
