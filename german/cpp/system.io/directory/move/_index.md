## Directory::Move method


Verschiebt die angegebene Entität an den neuen Ort. Wenn die zu verschiebende Entität ein Verzeichnis ist, wird es mit seinem gesamten Inhalt verschoben.

```cpp
static void System::IO::Directory::Move(const String &sourceDirName, const String &destDirName)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| sourceDirName | const String\& | Ein Verzeichnis oder eine Datei zum Verschieben |
| destDirName | const String\& | Der neue Speicherort von **sourceDirName** |
