## SearchOption enum


Gibt an, dass die Suche nur im aktuellen Verzeichnis oder im aktuellen Verzeichnis und allen Unterverzeichnissen durchgeführt werden soll.

```cpp
enum class SearchOption
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| AllDirectories | 0 | Suche im aktuellen Verzeichnis und allen Unterverzeichnissen. |
| TopDirectoryOnly | 1 | Suche nur im aktuellen Verzeichnis. |
