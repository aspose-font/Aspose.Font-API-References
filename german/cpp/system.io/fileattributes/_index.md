## FileAttributes enum


Stellt Attribute eines Verzeichnisses oder einer Datei dar.

```cpp
enum class FileAttributes
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Archiv | 32 | Das Dateisystemobjekt ist ein Kandidat für Entfernung oder Sicherung. |
| Komprimiert | 2048 | Das Dateisystemobjekt ist komprimiert. |
| Gerät | 64 | Nicht verwendet. |
| Verzeichnis | 16 | Das Dateisystemobjekt ist ein Verzeichnis. |
| Verschlüsselt | 16384 | Das Dateisystemobjekt ist verschlüsselt. |
| Versteckt | 2 | Das Dateisystemobjekt ist verborgen. |
| Integritätsstrom | 32768 | Das Dateisystemobjekt enthält Unterstützung für Datenintegrität. |
| Normal | 128 | Das Dateisystemobjekt ist eine Datei ohne besondere Attribute. |
| NichtInhaltIndiziert | 8192 | Das Dateisystemobjekt wird vom Inhaltsindizierungsdienst nicht indiziert. |
| KeineBereinigungDaten | 131072 | Das Dateisystemobjekt ist nicht im Datenintegritätsscan enthalten. |
| Offline | 4096 | Das Dateisystemobjekt ist offline und seine Daten sind nicht sofort verfügbar. |
| ReadOnly | 1 | Das Dateisystemobjekt ist schreibgeschützt. |
| Neuverarbeitungspunkt | 1024 | Das Dateisystemobjekt ist eine Datei, die einen Neuverarbeitungspunkt enthält. |
| Sparse-Datei | 512 | Das Dateisystemobjekt ist eine Sparse-Datei. |
| System | 4 | Das Dateisystemobjekt ist eine Systemdatei. |
| Temporär | 256 | Das Dateisystemobjekt ist eine temporäre Datei. |
| Invalid | 0 | Ungültiger Wert. |
