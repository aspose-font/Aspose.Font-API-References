## StreamSource::MustCloseAfterUse method


Die Erben können das Schließen des Streams verhindern. Gibt true zurück, wenn die Stream‑Source den Stream nach der Verwendung schließen möchte. Andernfalls wird false zurückgegeben.

```cpp
virtual bool Aspose::Font::Sources::StreamSource::MustCloseAfterUse()
```


### ReturnValue

True, wenn die Stream-Quelle den Stream nach der Verwendung schließen möchte, andernfalls false.
