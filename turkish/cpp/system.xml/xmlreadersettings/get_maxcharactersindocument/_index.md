## XmlReaderSettings::get_MaxCharactersInDocument method


XML belgesindeki izin verilen maksimum karakter sayısını gösteren bir değer döndürür. Sıfır (0) değeri, XML belgesinin boyutu için sınırlama olmadığını ifade eder. Sıfır olmayan bir değer, karakter cinsinden maksimum boyutu belirtir.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### ReturnValue

Bir XML belgesinde izin verilen maksimum karakter sayısı. Varsayılan değer 0'dır.
