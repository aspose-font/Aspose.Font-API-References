## XmlWriterSettings::get_IndentChars method


Girintileme sırasında kullanılacak karakter dizisini döndürür. Bu ayar, [XmlWriterSettings::set_Indent](../set_indent/) değeri **true** olarak ayarlandığında kullanılır.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Girintileme sırasında kullanılacak karakter dizisi. Bu, herhangi bir dize değerine ayarlanabilir. Ancak, geçerli bir XML sağlamak için yalnızca boşluk karakterleri, sekmeler, satır başı karakterleri veya satır sonu karakterleri gibi geçerli boşluk karakterlerini belirtmelisiniz. Varsayılan iki boşluktur.
