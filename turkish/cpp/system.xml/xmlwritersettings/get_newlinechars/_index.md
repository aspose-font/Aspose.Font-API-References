## XmlWriterSettings::get_NewLineChars method



### ReturnValue

Satır sonları için kullanılacak karakter dizisi. Bu, herhangi bir dize değeri olarak ayarlanabilir. Ancak, geçerli XML sağlamak için yalnızca boşluk karakterleri, sekmeler, satır başı veya satır beslemeleri gibi geçerli boşluk karakterlerini belirtmelisiniz. Varsayılan değer **\\r\\n** (satır başı, yeni satır) dir.
