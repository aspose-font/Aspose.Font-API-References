## Point::Subtract method

```cpp
static Point System::Drawing::Point::Subtract(const Point &point, const Size &size)
```


| Parametre | Tür | Açıklama |
| --- | --- | --- |
| nokta | const Point\& | Çevrilecek nokta |
| size | const Size\& | **point**'in koordinat değerlerinden çıkarılacak değerleri belirten [Size](../../size/) nesnesi |

### ReturnValue

**size**'ın genişlik değerinin **point**'ın X koordinat değerinden çıkarılması sonucu elde edilen X koordinat değerine ve **size**'ın yükseklik değerinin **point**'ın Y koordinat değerinden çıkarılması sonucu elde edilen Y koordinat değerine eşit olan yeni bir [Point](../) nesnesi
