## File::Copy method


Belirtilen dosyayı belirtilen konuma kopyalar. Hedef dosya zaten varsa, bir parametre onun üzerine yazılıp yazılmayacağını belirler.

```cpp
static void System::IO::File::Copy(const String &sourceFileName, const String &destFileName, bool overwrite=false)
```


| Parametre | Tür | Açıklama |
| --- | --- | --- |
| sourceFileName | const String\& | Kopyalanacak dosyanın yolu |
| destFileName | const String\& | Kopyalanacak dosyanın yeni konumunun yolu |
| üzerine yaz | bool | Hedef dosya zaten mevcutsa, mevcut hedef dosyanın üzerine yazılması gerekiyorsa doğru, kopyalamanın başarısız olması gerekiyorsa yanlış |
