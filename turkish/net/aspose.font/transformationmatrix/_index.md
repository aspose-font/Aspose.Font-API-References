## TransformationMatrix class

3x3 dönüşüm matrisini temsil eder &#x7C; AB 0 &#x7C; &#x7C; CD 0 &#x7C; &#x7C; TX TY 1 &#x7C;. Koordinatları aşağıdaki şekilde dönüştürür: x1 = A*x + C*y + TX; y1 = B*x + D*y + TY.
