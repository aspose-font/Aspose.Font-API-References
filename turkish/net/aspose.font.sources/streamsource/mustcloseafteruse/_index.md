## StreamSource.MustCloseAfterUse method

Mirasçılar akışın kapanmasını engelleyebilir. Akış kaynağı kullanımdan sonra akışın kapatılmasını isterse true döndürür. Aksi takdirde false döndürür.

```csharp
public virtual bool MustCloseAfterUse()
```

### Geri dönüş değeri
